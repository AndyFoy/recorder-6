Finding Related Taxon Occurrences

This dialog allows you to search for a taxon occurrence by looking for matches to typed text.

In the 'Related Occ.' page of a taxon occurrence record, you can type some text at the 'Related occurrence' prompt and then press <Enter>. The list shows the taxon occurrences in the current Survey which match the typed text.

Taxon occurrences are labelled using a string composed of:

  • Sample date.

  • Location of the sample if one is given, otherwise the spatial reference.

  • Sampling method.

  • Preferred name of the taxon.

To find an occurrence:

  • Type some text likely to appear in the label of the occurrence that you want to find. The list will show labels starting with the string you type.

  • If you prefix the text you type with an asterisk, "*", then the list will show names containing the text.

  • Select an item in the list by clicking on it with your mouse, or press <Tab> to position the cursor in the list and then use the <up arrow>, <down arrow>, <PgUp>, <PgDn>, <Home> and <End> keys to position the highlight over the correct category.

    • Choose [OK] or press <Enter> to select the highlighted category.

    • Double-click on an item in the list to return it.

    • Choose [Cancel] or press <Esc> to close the Find dialog without making a selection.
