Drawing polygons

To draw a polygon or line:

  • Select the drawing tool using the icon or through the map menu.

  • When the drawing tool is selected, the map cursor changes to a cross hair

  • Click with the mouse on the point at which you want to start drawing. The point selected is at the dot in the centre of the cross hairs.

  • Left click on additional points (which will be joined by a red line) until the transect line is completed or the polygon is nearly complete.

  • Right click and a menu of options appears at the position of the mouse. Select the most appropriate:

    • Finish polygon & add to <name of current polygon layer>.

    • Finish polygon & add to layer
    • Finish line & add to <name of current polygon layer>.
    • Finish line & add to layer
    • Cancel.

NOTE: No means of editing lines or polygons is available.

  • If you make a mistake while drawing:

    • Right click.

    • Select Cancel from the menu and start again.

  • If you find the completed line or polygon is incorrect, then you must delete it and redraw from scratch.

  • The new polygon will have the properties (shading, selected and unselected colours) of the layer to which you chose to add it. You can change these settings, but any change affects all polygons on that layer.

Holes in Polygons

It is possible to add a hole to a polygon. Any reports based on that polygon will not include records that appear in the "hole". To add a hole to a polygon;

  • Draw both the polygon and the "hole" that you want to take out of the polygon using the above method

  • Choose the pointer tool

  • Select the polygon from which you want to remove the hole

  • Choose the tool

  • As instructed, select the "hole" polygon and press the F8 function key

  • The polygon should now appear hatched and the hole unhatched.
