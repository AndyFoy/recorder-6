Delete

To delete the currently selected item in the active window:

    • Choose [Delete].

    • Select Edit – Delete from the Main menu.

  • You will be asked to confirm that you really want to delete the item. If you respond positively, the item will be permanently deleted from the database.

  • You may be informed that the delete cannot be carried out because other data depend upon this item. Delete these dependent items first.

NOTE: Recorder 2002 does not allow cascading deletes. You must explicitly remove dependent records before you can remove the record they depend on.
