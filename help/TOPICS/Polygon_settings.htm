Polygon map layer settings/TD

This dialog is used to enter or change the settings for a polygon map layer.

Name:

Type a name for the polygon layer.

Pattern:

Choose one of the patterns from the list. This is the shading that will be applied to the polygon.

Selected Colour:

Choose the colour in which the polygon will be shown when it is selected.

Unselected Colour:

Choose the colour in which the polygon will normally be shown, when it is not selected.

When the necessary details have been filled in, choose [OK] to save or update the details. [Cancel] abandons the process of adding a map without making any changes to the map layer.

NOTE: If you change the colour settings for a polygon layer, the polygon layer has to be redrawn using the new colours. This may take some time if the layer contains a lot of polygons, or complex polygons with a lot of points. A progress bar is displayed.
