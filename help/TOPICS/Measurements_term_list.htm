Measurement term-list

The Measurements term list has some special features:

  • The context in which a particular type of measurement should be used needs to be specified.

  • It is linked to two other term lists (for qualifiers and units)

Measurements are available for Locations, Samples, Taxon occurrences and Biotope occurrences to enter all types of measurements and counts. However, a particular type of measurement is not necessarily appropriate in all of these contexts. This is specified using the "Available on:" list. In the example shown above, "Abundance" will only be offered in the list of available measurement types in the Taxon occurrences measurement page.

To specify the context in which a measurement type should be used

  • Tick the appropriate items in the list labelled "Available on:"

  • If none of the items in this list is ticked then this type of measurement will be available everywhere.

Units and Qualifiers

Measurement types are linked to two other term lists:

  • Measurement qualifier allows the user to record to what a measurement applies. In the example shown, Abundance could be specified for "male".

  • Measurement unit allows the user to specify the units for a particular measurement.

To add measurement qualifiers or units:

  • Select either "Measurement Qualifier" or "Measurement Unit" and then proceed as for any other linked list.

Data type for Measurement Units:

This specifies the type of information that will go into the Data field of the measurement window when this Unit is used. This then affects how Data is reported. The possibilities are:

  • Boolean (Yes/No)

  • Memo
  • Number
  • Spatial reference
  • Text
  • Vague date
