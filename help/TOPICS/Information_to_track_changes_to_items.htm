Information to track changes to items

When Recorder saves an item, it automatically keeps track of:

  • The time and date on which the item was originally created.

  • The name of the user who created it.

  • The time and date on which the item was last changed.

  • The name of the user who made the last change.

The 16 character key, or unique identifier, of an item provides additional information. The first 8 characters are the Site (or Copy) Id of the copy of Recorder to which the item was originally input. This copy of Recorder will be its owner or custodian unless custody has been reassigned to another copy.

You can view this information using the Show Metadata tool.
