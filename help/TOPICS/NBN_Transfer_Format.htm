NBN transfer format

A format for exchanging data between users of the National Biodiversity Network (NBN) data model, or systems that are compatible with it, has been devised. This uses XML version 1. XML is a method for putting structured data into text files. The structure is formally described in a Document Type Definition (DTD). The DTD for Recorder can be found in the Recorder 6\DTD directory of a standard installation, and is itself a text file written in XML syntax.

Recorder 6 has facilities to read and write XML files using NBN transfer format.
