Surveys General Tab

This page contains fields describing the survey.

 

Name

Required.

Free text. A descriptive title for the survey.

Run by

Required.

The main organisation or individual responsible for organising and running the survey. This consists of a reference to an item in the Name and Addresses window.

You can:

  • Drag & drop, or copy & paste a name from the Names and Addresses window or from a rucksack.

  • Use the button to display the Names and Addresses window. Locate the name you wish to insert, and use Return data to send it back.

  • Type a value in the box and press <Enter> to open the Find Individuals/Organisations window to search for matching names.

Type

Required.

A value chosen from the survey type term list. Users can add new terms if necessary.

Records allowed From / To

'From' required.

A range of Vague dates over which records will be accepted by this Survey. Although the 'From:' date is required, 'Unknown' is an acceptable value.

Validation checks also require that a 'To:' value is given if the survey is flagged as 'Completed', and also that the 'From' is before the 'To' date.

Operating From / To

The date range over which the Survey has operated. This may be the same period as that given in 'Records Allowed From / To', but records may be allowed from a much earlier date if the Survey accepts historical data.

The the 'From' must be before the 'To' date.

Status

A value chosen from the survey status term list. Users can add new terms if necessary. Indicates whether the survey is ongoing, completed, etc.

Media

A value chosen from the survey media term list. Users can add new terms if necessary. Indicates the way in which survey results are stored (e.g. on cards, on computer).

Periodicity

Free text. If the survey involves repeated observations, this field can be used to indicate how often, and the next time they are due to be carried out.

Description

RTF Memo. Descriptive text which should include an indication of the methods used, and how results are checked and validated.

NOTE: The Records allowed From / To dates (if not 'Unknown') are used to validate dates given for survey events and samples attached to the survey.
