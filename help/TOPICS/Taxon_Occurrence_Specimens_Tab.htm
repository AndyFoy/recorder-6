Taxon Occurrence Specimens Tab

If physical evidence for a particular observation such as specimens or signs have been retained, this page can be used to record the details.

 

Specimen number

Required.

Free text. An identifier for the material that was retained, so that it can be located in future.

Accession date

The date at which this material was added to a collection.

Type

Required.

Item selected from the specimen type term list. Users can add terms to this list if necessary.

Location

Free text. An indication of where the material is kept.

Comments

RTF Memo. Any text describing the material and its condition.

To add a specimen:

  • Click on the button, and the detail panel will go into Edit mode with all the fields empty.

  • Type in the required information.

    • Click the button to save your changes.

    • Click the button to abandon your changes.

To delete a specimen:

  • Highlight the specimen that you want to delete.

  • Click the button.

  • You will be asked to confirm that you really want to delete this record.

To edit a specimen:

  • Highlight the specimen that you want to edit.

  • Click the button to put the detail panel into Edit mode.

  • Make the changes you require.

    • Click the button to save your changes.

    • Click the button to abandon your changes.
