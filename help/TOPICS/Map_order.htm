Map layer order

Overview

If you have several map layers for a map that overlap in a given geographical area, the order in which they are drawn can be an issue. Think of them like a stack of sheets of paper. Whilst vector maps such as your polygon layers are transparent and allow other layers to show through, this is not the case with raster maps or backgrounds. Raster maps cover up and hide anything that is "underneath" them in the stack.

For example, suppose you have an 1:50,000 map tile (a raster layer) showing a site in which you are interested and you also have a more detailed, 1:10,000 map tile (another raster layer) covering just that site. You would probably like to arrange things so that, as you zoom in on the area, the 1:50,000 map is displayed with the 1:10,000 map on top of it. This is a considerable help in selecting the right area to enlarge. You would set the "Cut-in" and "Cut-out" scales so that the 1:50,000 map "cuts-out" when you have zoomed in closer leaving just the more detailed map visible. However, if the order of display is such that the 1:50,000 map is "on top" of the 1:10,000 map, the latter will be hidden "behind" the former until it reaches its cut-out scale. Consequently you would not see the 1:10,000 map until you had zoomed right in. To fix the problem, move the 1:10,000 map down of the list of map layers until it is below the 1:50,000 map.

Note: In Recorder, map layers are displayed "on top" of layers further up the list. However, the "Base Map Sheet" is a special case and will always be appear "on top" of all other layers.

To change the order of display of map layers:

Use the Map Layers box in the bottom left-hand corner of the open Map window. To open the map you want to work on select Map - View Map from the Main menu then, in the sub-menu, select the required map, or use the Display Map Window icon on the toolbar. If the map window is already open, use the Active Map box (top left) to select the required map.

  • Click on a layer's name in the Map Layers list at the bottom left of the window to select it.

  • Drag it to a new position in the list, or use the up or down arrows at the top of the list to move it up or down.

Or, select Map - Map Options from the main menu, then select the Map Layers tab. Select the map you want to work on using the drop down arrow beside the Map box. Note that this box is only displayed if you have more than one map loaded.

  • Click the layer you want to move to select it.

  • Click the "Move Up" or "Move Down" buttons. The selected layer will move up or down one position in the list for each click.
