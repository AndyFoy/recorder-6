Standard reports

Two built-in standard reports are provided:

  • Locations for occurrences

  • Occurrences for locations

The standard reports are available via the Report Wizard if you select 'Use existing layout' in the Layout Selection screen. They are shown at the top of the list of available reports in the Layout Selection dialog.

Locations for occurrences

This standard report can be used to show the results of a query produced by the Report Wizard when you choose 'Taxon records' or 'biotope records' in the first page of the Wizard (under Select report style).

The report has a section for each taxon or biotope. Each section has a header consisting of the name of the taxon or biotope, and any 'Facts' available about it from the dictionaries. Observations are then grouped by location, and the date and determiner are shown for each observation together with any comment.

Occurrences for locations

This standard report can be used to show the results of a query produced by the Report Wizard when you choose 'Report about a place' in the first page of the Wizard (under Select report style).

The report has a header section which shows details about the location, and then sections for biotope and taxon observations from that location. Observations are grouped by biotope or taxon, and the date and determiner are shown for each observation together with any comment.

The standard report can be shown in the Print Preview screen, from which it can be printed or saved.
