Getting grid references or locations from maps

This facility is accessed by pressing the button next to a prompt for a spatial reference.

  • On pressing this button, the Map Browser window will be displayed.

  • You can use the Zoom and Pan tools on the map to locate the point for which you want a grid reference.

  • Select the Pointer tool and click on the map at the desired point:

    • The co-ordinate of the point at which you clicked will be shown in the Spatial Reference box, formatted according to the currently selected spatial reference system.

    • If you clicked within a site-polygon which is associated with a location, the name of the location will appear in the Location box.

  • You can now return the data by:

    • Selecting Edit - Return data from the Main menu.

    • Clicking on the button in the active toolbar.
    • Pressing the short-cut key <F9>.

Alternatively, with both the Map Browser window open and the window in which you are adding or editing data:

  • Select the Pointer tool and click on a point on the map in the Map Browser window, and the Spatial Reference and Locations boxes will be filled as above.

  • Drag and drop, or copy and paste the spatial reference from the Spatial Reference box and/or the name from the Location box to the appropriate prompt in another window.

TIP: The precision of the spatial reference you obtain when you click on the map is determined by the scale of the map. See Grid references and spurious accuracy.
