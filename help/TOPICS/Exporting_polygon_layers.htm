Exporting polygon layers

Polygon layers can be exported from Recorder as ESRI shape files (.shp) for use in other systems such as GIS like MapInfo and AcrView.

Open the map which contains the polygon layer(s) to be exported by selecting Map - View Map from the Main menu then, in the sub-menu, select the required map, or by using the Display Map Window icon on the toolbar. If the map window is already open, use the Active Map box (top left) to select the required map.

To export a single polygon layer, either:

  • Select and right-click the polygon layer to be exported in the list of map layers to the left-hand side.

  • Select Export Polygon Layer from the resulting menu.

or

  • Select the polygon layer to be exported using the down arrow in the Current Polygon Layer box (bottom left corner of the map window).

  • Select Map - Export Current Polygon Layer from the Main menu.

To export all polygon layers, either:

  • Right-click in the map layers box to the left-hand side.

  • Select Export All Polygon Layers from the resulting menu.

or

  • Select Map - Export All Polygon Layers from the Main menu.

Selecting one of the above options opens a standard Windows Save dialog to enable you to specify the file name for the export and where you want it saved. The exported polygon layer(s) will be saved as ESRI projection and shape files etc. (.dbf, .prj, .shp and .shx) containing the Location Keys, Location Names and Field Codes for locations linked to polygons.

For adding, deleting and changing polygon layers see Map Options - Map layers

Note: Polygons can be linked to locations in the location hierarchy when importing them or by associating them with locations.

Note: Polygon layers can't be exported from the Map Layers tab in Map - Map Options, unlike adding map layers etc.
