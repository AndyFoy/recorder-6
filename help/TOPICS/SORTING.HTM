Sorting

To sort the list or hierarchy on the left-hand side of any of the main windows:

  • Select Edit – Sort by from the Main menu.

  • Press the button in the active toolbar.

  • Click with the secondary mouse button (i.e. the one you use least) in the list or hierarchy.

You will see a sub-menu with a list of items by which the currently active list of items can be sorted. For example, if you are dealing with a list of individuals, the options will be 'Surname' or 'Forename'.

Choose one of the available items and the list will be redisplayed, sorted in order by the selected item. The items in the list or hierarchy will continue to be displayed in this order whenever you open this window until you choose to change the order again.

 

Note: in the report wizard; choose "sort order" as an attribute so that you can then sort your items into the appropriate taxonomic order.
