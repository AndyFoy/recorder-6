Maps overview

Maps are used for four purposes in Recorder:

  • To show the distribution of items such as species, biotopes and sampling stations.

  • To allow the user to get a spatial reference and Location by clicking on a map:

    • This allows an accurate spatial reference to be obtained (spatial references are one of the most error-prone aspects of biological records).

    • If the indicated point is within a digitised site boundary, it allows an unambiguous link to be made with a location associated with that digitised boundary.

  • To specify bounding boxes. These are used to validate spatial references and also for selecting locations or observations for reports.

  • To specify polygons, e.g. digitised site boundaries and administrative area boundaries. These are used to select observations for reports.

The mapping facilities in Recorder are provided by a third party component library called MapServer.

You can set up multiple maps in Recorder. Each map is made up of different types of layers:

  • The base map - the outline map on display when you open the map window with the entire map on view.

  • Background layers - raster tiles you can load as a background to your mapping tasks.

  • Polygon layers - map layers on which you can draw polygons or import them from vector maps.

For information about the format of background and polygon map files that can be used in Recorder 6, see Map formats.

To set up and manage the maps you want to use in Recorder, see Map - Map Options. You will need to set up at least one map using the Enabled Maps tab on Map - Map Options before you can start using Recorder's mapping facilities.

See The Map Window or the Map menu for information about using Recorder's maps.

The Map Navigator Tool enables you to navigate to a specific polygon in the Map Window. If you have drawn or imported polygons, and associated them with locations in your location hierarchy (optional) you will be able to use this tool to view a particular polygon.
