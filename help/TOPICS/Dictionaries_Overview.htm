Dictionaries overview

Recorder contains a number of Dictionaries which are standardised lists of terms in wide use in the biological recording community. These terms are supplied by the National Biodiversity Network (NBN), the Natural History Museum (NHM) and the Joint Nature Conservation Committee (JNCC), and help improve standardised recording. The terms that are supplied with Recorder, i.e. when it is installed or via upgrades, are classed as system-supplied and cannot be changed. However, users can add new terms and add information to existing terms. This is classed as user added data and can be edited or deleted but it is specific to your system and cannot be exported to another copy of Recorder. The ability to add new taxa and biotopes enables you to enter observations for these terms, and include them in reports, while you wait for them to be added as system-supplied terms. The ability to add information to existing taxa and biotopes enables you to hold local information specific to your biological recording, e.g. local common names, status information and facts. See system supplied vs. user added and how to add taxa and biotopes for details.

Both the Taxon and Biotope Dictionaries provide a window for viewing the data – the Taxon Browser and Biotope Browser – and for adding, deleting or editing the data – the Taxon Dictionary Details and Biotope Dictionary Details windows. To use these facilities, select Dictionaries - Taxon, Biotope, Edit Taxon Details or Edit Biotope Details respectively from the main menu.

Note: Changes to the system-supplied entries in the dictionaries in your Recorder database will be incorporated via Dictionary Upgrades.

Taxon Dictionary

One of the biggest problems with biological recording in the past has been that different individuals and organisations use different names for the same species or other taxon. Terminology also changes over time. Usually, an organisation maintains a list of species that it records in a 'checklist'. This will contain a list of the scientific and common names of the organisms with which they are concerned, and may also be organised into a taxonomic hierarchy. Such 'checklists' form the basis for the organisation's biological recording effort.

In Recorder, a number of these checklists are held together in the Taxon Dictionary. The Taxon Dictionary enables the system to maintain any number of checklists of species that can be used throughout Recorder. Each checklist can contain different species, synonyms, common names and hierarchies.

The system-supplied information in the Taxon Dictionary is the NBN's Species Dictionary adapted for use in Recorder. It aims to provide a standard reference for names of organisms found in the United Kingdom. It is a project of the National Biodiversity Network and is hosted and managed by the Natural History Museum, London. The NHM obtain the individual lists from the most appropriate source for that taxonomic group, usually a national scheme or society. When changes become available you will be able to incorporate them into your system by running a dictionary upgrade. Requests for corrections and changes to the species dictionary should be posted on the NBN Recorder forum under the species dictionary section, or alternatively contact JNCC (e-mail: Recorder@JNCC.gov.uk) or the NHM - see how to add taxa or biotopes to dictionaries.

Biotope Dictionary

The Biotope Dictionary provides a number of standard biotope classifications that can be used throughout Recorder. It is used in the same way as the Taxon Dictionary. Users can use the system-supplied data or they can add their own entries to lists (see new taxa/biotopes).

Requests for corrections and changes to the system-supplied information in the Biotope Dictionary should be sent to JNCC (e-mail: Recorder@JNCC.gov.uk).

  • For information on viewing checklists see Biotope Dictionary Browser.

  • For information on adding facts to a biotope see Adding Biotope Facts.

  • To add a new biotope see Adding Biotope General.

Administrative Area Dictionary

The Administrative Area Dictionary provides a number of standard lists of administrative areas that can be used in Recorder. It is used in the same way as the Taxon Dictionary. For the current local government administrative areas select the Type 'Country'.

Requests for corrections and changes should be sent to JNCC (e-mail: Recorder@JNCC.gov.uk).

  • For information on viewing lists of administrative areas see Administrative Area Dictionary Browser.

Note: Unlike the taxon and biotope dictionaries, users cannot add to the Administrative Area Dictionary.
