Getting Help

The simplest way to obtain help is to press the <F1> short-cut key or to click the button in the active toolbar. This will take you straight to the help topic relevant to the control where the cursor is currently located.

Alternatively, the Help menu item gives access to the following facilities:

Help

Opens the Help window at the Table of Contents.

Index

Opens the Help window at the Index.

Context sensitive

The same as pressing <F1>. Opens the Help window at the topic appropriate to the control in which the cursor is currently located.

Help is provided via Microsoft HTML Help.

When you press <F1>, the Help window is initially shown with just the topic window displayed:

The text may contain hyperlinks which take you to related topics. Many topics include a [Related topics] button at the bottom of the page, which provides a list of related items.

If you need to explore the help more widely, clicking the [Show] button will expand the window to show the controls. Selecting Help - Help or Help - Index from the Main menu will open the Help window, with the controls already visible.

The three pages on the left give access to:

  • The Table of Contents. Navigate around the tree by double-clicking on the book icons. Click on a topic, and it will be displayed in the right hand panel.

  • The Index. Enter a keyword in the box at the top, and the list will be positioned at that item. Double-click on a topic in the list, and it will be displayed in the right hand panel.

  • A word Search facility that will search all topics to find a word or phrase you type. Enter the word for which you want to search. Click the [List topics] button, then double-click on a topic in the resulting list, and it will be displayed in the right hand panel.
