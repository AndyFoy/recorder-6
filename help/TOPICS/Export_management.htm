Export Management

This facility allows you to export data from one or more surveys or taxonomic groups more easily than by using the other Export methods. It is particularly useful for users who are contributing data to others on a regular basis, e.g. for users who are regularly contributing data from several surveys to their local record centre, or data from one or more taxonomic groups to a national recording scheme. To use it you create a filter which you then use to do exports. If the export is always to be e-mailed to the same person or organisation you can set up a recording scheme to automatically e-mail the exported data to a specified e-mail address.

Export filters

The Export filter specifies what is to be exported, e.g. all observations in a selection of surveys, or all herptile observations.

  • Export filters are saved so that they can be used to do an export many times.

  • Export filters can be created and then sent to another user. This means that someone can work out what data they need, and send a filter to the data supplier who can then use it to export the required data.

  • Export filters can also be used as the basis for a Recording Scheme.

Filters are created by selecting Tools - Export Management - Manage Export Filters from the Main menu - see Export filters for more details, including how to use a filter to export data.

Recording Schemes

Quite often, an export of observations needs to be sent to the same person on a regular basis, e.g. new and changed bird observations needing to be sent to the county bird recorder once a month. To facilitate this process a Recording Scheme can be set up which specifies the Export filter to be used, the export type to be used and the e-mail address that the data is to be sent to. The monthly submission can then be made more easily. Recorder uses the export filter to locate the required observations then it exports only those which have been entered or changed since your last submission to a file of the specified export type. The resulting file is then zipped and e-mailed to the specified e-mail address. The time and date is automatically recorded, so that only new and changed information is sent next time.

Recording schemes are set up by selecting Tools - Export Management - Manage Recording Schemes from the Main menu - see Submitting observations to a scheme or organisation for more details, including how to use a recording scheme to export data.

Note: If an export filter is not specified, the export works on the whole database.

Note: Exports done using Tools - Export Management will include:

  • Confidential observations provided a systems manager has ticked the Enable export of confidential occurrences option on the General tab in Tools - Options. This is true when using Export Using Filter but not when using Contribute to Scheme. In this case confidential observations are not exported regardless.

  • Unchecked observations
  • Observations flagged as Failed/Pending Verification

Exports done by other means will exclude these observations although confidential observations will be included if Enable export of confidential occurrences is ticked - see Exports.
