Names and Addresses Associations Tab

This page can be used to record the association between two organisations and/or individuals,e.g. membership of a society.

 

With

Required.

A reference to another individual or organisation. The reference can be obtained by:

  • Drag and drop. Find the name of the individual or organisation from whom the communication came in the list in the left panel of the window, and drag it into this field.

  • Cut and paste. Find the name of the individual or organisation from whom the communication came in the list in the left panel of the window and select it.

  • Copy it to the clipboard by selecting Edit – Copy from the Main menu, pressing the button in the active toolbar or pressing the short-cut key <Ctrl-C>.

  • Return to the 'With' field, and paste by by selecting Edit – Paste from the Main menu, pressing the button in the active toolbar or pressing the short-cut key <Ctrl-V>.

  • Find the name by typing some text and then pressing <Enter> to open the Find Individual/Organisation dialog.

Role

Required.

The role of this individual or organisation (e.g. member, employee, volunteer). Free text with suggested values available from list. Common roles can be picked from the list.

Name code

Free text. Can be used to record membership number or similar details.

Comment

RTF memo. Can be used to amplify the nature of the association.

From

Required vague date.

The date at which this association was commenced. Although this is required, 'Unknown' is an acceptable value.

To

Vague date. The date at which this association ended.

NOTE: Associations are NOT automatically reciprocated. For example, if you make an entry recording that a person is an member of an organisation, an entry for the organisation recording that individual as a member WILL NOT be set up for you.

To add an association:

  • Click on the button, and the detail panel will go into Edit mode with all the fields empty.

  • Type in the required information.

  • Click the button to save your changes.

  • Click the button to abandon your changes.

To delete an association:

  • Highlight the association that you want to delete.

  • Click the button.

  • You will be asked to confirm that you really want to delete this record.

To edit an association:

  • Highlight the association that you want to edit.

  • Click the button to put the detail panel into Edit mode.

  • Make the changes you require.

  • Click the button to save your changes.

  • Click the button to abandon your changes.
