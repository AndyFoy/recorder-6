Location Relations Tab

The Location Relations tab holds information on any relationships between the location and other sites or subsites. This tab allows the user to add or delete any number of relationships.

Related With

The Location with which you want to create a relationship.

Relation

The nature of the relationship. The possibilities are:

  • Within.

  • Contiguous.
  • Adjoining.
  • Overlapping.
  • Adjacent.

See related site types for definitions.

It is not necessary to enter any data, but if you make an entry in a row, values are required in both columns.

To add a relation:

  • Select the location with which you want to create a relationship

  • Click on the [Add] button.

  • The Find Location dialog box is displayed.

  • Select an item and click [OK].

OR

  • Drag a location into the grid.

  • Select a value for the relation.

  • Click the button to save your changes.

  • Click the button to abandon your changes.

To delete a relation:

  • Highlight the item you want to delete.

  • Click on the [Delete] button.

  • You will be asked to confirm the deletion.
