Filter by Source

You will see this screen if you selected the option to 'Restrict report to one or more sources' when selecting the style of report.

 

Select the type of source to which you want to restrict the report by selecting one of:

Documents

Restrict the report to observations related to one or more records in the Documents window.

Individuals / Organisations

Restrict the report to observations related to one or more organisations and/or individuals.

Surveys

Restrict the report to observations related to one or more surveys.

Survey tags

Restrict the report to observations related to one or more survey tags.
