Sharing Recording Cards with others

Recording cards are a useful method of rapid data entry for a particular recording scheme or national survey. Because the cards are simple Windows text files, they can be exchanged easily between users of Recorder. Therefore, for a particular co-ordinated survey or scheme, a standard recording card could be created (for example, by the scheme organiser) and circulated to all contributors to promote a common method of recording.

To transfer a recording card

Recording cards are stored in the recording cards directory as simple text files with a ".crd" extension. This directory is '\Recorder 6\User Files\Recording Cards' in a 'standard installation', but can be changed using the Options - File Locations dialog box. For example, if you have created a Recording card called 'Common spring flowers', you will find a file called 'Common spring flowers.crd' in your recording cards directory. To transfer this card to other users, simply copy the file and distribute it on a floppy disk or as an email attachment.

To use another individual's recording card

If you have received a recording card which was created by another user, then you can easily use it in your copy of Recorder.

  • Identify where recording cards are stored by looking at the recording card directory in Options - File Locations dialog box.

  • Copy the ".crd" file you have been sent to this directory.

  • When you next start Recorder and go to the list of cards (via Data Entry – Enter Species for a place), you will see the additional card that you have copied.

Note: You need to close Recorder down and start it again before a new recording card will be detected and made available in the Data Entry - Enter species for a place sub-menu.
