Sample type (term list)

Entries in the Sample Type term list require some additional pieces of information compared with other term lists.

See Adding a term to a list for information about the standard items.

Additional information

Icon

A 16x16 pixel bitmap which will be used to represent samples of this type in the Observation hierarchy.

  • The [Open] button gives access to a standard Windows open file dialog to assist in finding a bitmap file (.bmp).

  • The [Default] button will assign the default icon (Default.bmp located in the \Recorder 200x directory).

If you don't supply an icon for a sample type, Default.bmp will be used to represent samples of this type.

Associated record card

A recording card can optionally be associated with a sample type. If such an association is made, then the facility to enter records by pointing at the map will be available.
