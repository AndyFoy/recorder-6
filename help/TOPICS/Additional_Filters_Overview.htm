Additional Filters overview

This dialog is opened by the [Additional Filters] button on the Layout selection screen of the Report Wizard. It allows the user to specify additional criteria to select observations for a report e.g. limit observations to those made since 1970.

The page has a tab control at the top which switches between two pages.

Conditions

Specify additional criteria.

Constraints

Specify whether or not to include observations, depending upon their status with respect to validation, confidentiality, etc.
