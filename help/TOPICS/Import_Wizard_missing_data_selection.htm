Import Wizard - missing data selection

The Import Wizard needs to know which survey to add the observations to, but you cannot specify this in the import file, so you are asked to select a survey on this page. Use the drop-down arrow to make your selection.

Tip: If the survey to which you want to add the imported data does not exist in Recorder yet, just open the Observation window and add it. When you return to this page of the Wizard, using Window - Import Wizard, you should find that your new survey is included in the drop-down list.

When importing observations via the Wizard, the import file doesn't need to contain all the data required to add them to the Recorder database. Any data that is required but not selected for import on the column selection page of the Wizard, is requested on this page. Hence, if there is no observer column you will be asked to specify an observer here, if there is no sampling method column you will be asked to specify a sampling method here. Similarly for the date, record type and location data. This data is entered in much the same way as when using Data Entry – Enter a species record or Data Entry – Enter Species for a place – [recording card], e.g. using the Get button or the drop-down arrows, as appropriate. The data entered here applies to all observations in the import.

Tip: You may find that the entry you need is not in Recorder yet, e.g. a determiner's name is not in the list of individuals in the Names and Addresses list, or a sampling method is not in the sample type term list. If this is the case, use the Get button to open the appropriate window in Recorder, add the entry and Return it. If the Get button isn't available, open the appropriate window, e.g. Tools - Term Lists - Sample Type, add the entry and return to the Import Wizard, using Window - Import Wizard. Clicking on the [Previous] button then the [Next] button will effectively refresh the drop-down list, so that the newly added item will be available for selection.

For help with this stage of an import, see the text on the left of the window. The [Next] button won't become active until all the issues are resolved.

Once all the missing data has been selected, the [Next] button will become active. Click on the [Next] button to move to the next screen.
