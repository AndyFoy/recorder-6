Recording Card Columns

Standard columns

Biotope

Look-up a biotope code. Note that a biotope occurrence attached to the sample will be created for each unique value in this column and a comment will be added stating that the species was associated with the particular biotope.

Code number

Information only. You cannot enter data into this column. Shows the checklist code number of the species, if there is one.

Comment

A comment (free text) which will be stored in the taxon occurrence comment field.

Common name

Information only. You cannot enter data into this column. Shows the common name of the species, if there is one.

Count

Adds generic Count and Count Of columns to the recording card. These can be used to enter a count which will be stored in the taxon occurrence data. The Measurement type will be "Abundance" and the measurement qualifier will come from the Count Of column.

Date of determination

Enter a date. Stored in the taxon determination page of the taxon occurrence.

Determiner

Look-up the person who determined the species. Stored in the taxon determination page of the taxon occurrence.

Provenance

Choose the Provenance of the species record. Stored in the Provence field of the taxon occurrence.

Record type

Choose the Type of Record of the species record. Stored in the Record Type field of the taxon occurrence.

Spatial ref

Enter a spatial reference for this particular observation. Note that, a separate sample will be created for each unique value you enter into this column.

Substrate

Choose the Substrate on which the species was record. Stored in the Substrate field of the taxon occurrence.

Measurements

This list includes all combinations of Measurement types, Measurement qualifiers and Measurement units applicable to taxon observations.

The entry in Data is taken from you entry in the Recording Card columns, whilst the Measurement, Qualifier and Unit are as specified by the column headers. The Accuracy will be "Exact" unless you include non-numeric characters, in which case it will be "Estimate" (e.g. >5, 6-8, 10?, many).

This makes it possible to enter more than one measurement for a single taxon observation entered via the Recording Card.

Note: The measurement column names can be changed to something shorter and more appropriate. E.g. change "abundance of male (count)" to "male"See "New Recording card" page for details.
