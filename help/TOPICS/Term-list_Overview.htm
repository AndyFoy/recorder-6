A somewhat arbitrary distinction is made between dictionaries (which are large and have a complex structure spread over a number of tables), and term-lists which are mostly simple flat files. Both are used in a similar way to provide a set of terms which are common to many users, and therefore facilitate the exchange and understanding of information.

However, it is very difficult, if not impossible, to provide comprehensive lists given the variety of activities encompassed by the biological recording community, and the fact that new fields of study and survey methods are constantly emerging. Users can therefore add new terms when necessary.

When considering adding a new term, you are encouraged to decide:

  • Is it really necessary, or is an existing term sufficiently similar?

  • If you do need to add a new terms, try to consult with others involved in similar areas of biological recording, so that duplication is avoided. Ideally one individual or organisation should be designated to make additions. These are then distributed to relevant recorders via an XML transfer file, so that all those concerned are using the same set of terms.

  • Define the terms you add, so that others can understand exactly how you intend them to be used.

To add, edit or delete terms select Tools - Term Lists from the main menu.
