Taxa Selection

You will see this screen if you have chosen options to report on one or more taxa.

Check list

Select a checklist from those available in the Taxon dictionary. If a rucksack is open when the Report Wizard is started there will be two additional options at the end of the drop down list: "Current Rucksack" and "Current Rucksack (Expanded)". Selecting one of these will allow you to select items from the taxon list in the open rucksack.

Find taxon

This works like the Find dialog. Type a name, or part of a name here, and all matching names are shown in the "Available" list. If you type part of a name (e.g. "hel"), then the "Available" list will contain all names in the selected checklist or rucksack starting with what you typed. If you prefix it with "*" (e.g. "*hel"), then all names containing this string will be included. You can use 5-letter abbreviations for species names but they need to be preceeded by "*".

Tip: type "*" at the Find Taxa prompt to add all the items from the rucksack to the Available list.

Include All Taxa

If this option is selected then all taxa are included in the final report, in which case, the other controls for selecting particular taxa are disabled. It is for with other selection criteria, e.g. when restricting a report to observations by one or more recorders, or to a bounding box.

All Taxa from list

If this option is selected then all taxa from the selected list will be included in the report, in which case, the other controls for selecting particular taxa are disabled.

Expand Taxonomic Group

If you include a higher-taxon in the 'Selected List', e.g. an order, family, genus, etc., this gives you the option of whether or not to include all the taxa within that group in the query. For example, if you include the orders Diptera and Coleoptera in the 'Selected list', then if this option is ticked, all observations of beetles and flies will be included in the report. If this option is not ticked, the report will only include observations for which "Diptera" or "Coleoptera" was specifically given as the preferred determination.

Move the taxa whose associated observations you need in the report into the "Selected" list. See Add and Remove items from selected list.

You can add one or more names to the 'Selected' list, and then make changes in the 'check list' and/or 'Find' part of the screen. This will result in a different 'Available' list from which you can make additional selections.

Note: Users working on taxa for which lots of name changes have occurred may like to know more about the dictionary translates between different names on different checklists.
