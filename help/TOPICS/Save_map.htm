Save a map

A map, as it is currently displayed on screen, can be saved to a file or copied to the clipboard so that it can be pasted into another program.

To save the map to a file:

  • Select File - Save As... from the Main menu.

Two formats are offered:

  • Bitmap (.bmp)

  • JPEG (.jpg)

To copy the map to the clipboard:

  • Select Edit - Copy Map to Clipboard from the Main menu.

  • Press the [Ctrl-M] key.

  • Click the button on the toolbar.

You can now switch to another program and paste the image, for example, into a word processor document.
