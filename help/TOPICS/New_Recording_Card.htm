New Recording Card

  • Select Data Entry – Enter species for a place – New Recording Card.

The New Recording Card dialog box appears.

To create a new recording card:

There are 4 steps to creating a new recording card.

1. Enter a Name

Type in a name for the new recording card in the edit box, or select an existing card from the drop-down box.

2. Choose the species

When you create a new recording card, it must have a list of species to display on the card. This species list can come from a taxon dictionary list (see Dictionaries overview) or a list of taxa in a Rucksack. The options are:

  • Taxon Dictionary. Choosing this option enables the user to select a checklist from the drop-down box. This checklist will form the species list for the recording card.

  • Rucksack. Choosing this option enables the user to select a rucksack from the drop-down box. The list of species in the selected rucksack will form the species list in the recording card.

Tip: If an item in the Taxon list in the Rucksack has further items nested beneath it, you are prompted whether or not to expand the list in the Recording Card to include these items. So, for example, you could put 'Carabidae' in the Rucksack list and accept the option to expand the list, resulting in a list of all ground beetle species in the Recording Card. This is a great deal quicker than building a list containing all the individual species names!

3. Choose any additional columns

At it's most basic, a recording card will consist of just two columns; a list of species names and a column of tick boxes. You may, at this point choose any additional columns that you want to include to allow you to record additional information. The additional columns are of two types;

  • standard - a fixed list of 11 columns

  • measurements - includes all available combinations of measurement type and qualifier.

These two types are displayed on two tabs, which can be switched between.

See Recording card columns for details of columns available

  • The list box on the left displays the columns of either type that are available.

  • The list box on the right displays the columns that will be shown on the card.

  • Use the arrows to select items from the list on the left and add them to the list on the right.

4. Changing how the columns are displayed.

Order of columns

The order in which the columns are displayed on the recording card will follow the order in which they are listed in the right hand list box. For example, if Count is displayed at the top of the list box, then it will be the leftmost column in the recording card after the taxon name. To change the order in which the columns are displayed:

  • Select the item in the right hand list box whose display position you want to change.

    • Click on the [up arrow] to display the column further to the left on the recording card.

    • Click on the [down arrow] to display the column further to the right on the recording card.

Names of columns

The default titles of some of the columns, especially some of the measurement columns are very long and cumbersome. You may wish to change the titles to something more useful.

  • Select the item in the right hand list box whose title you want to change.

  • Right-click and choose 'Rename'

  • Over-write the item with the new title

 When all of the options are set for the new recording card, click [OK], and the new card will be saved and the dialog box will close.

TIP: Recording cards are saved as .crd files in the \Recorder 200x\Recording Card directory in a standard installation. It will probably be located on the server in a networked installation. This directory is specified in the Options - File Locations dialog box. Because the cards are saved as normal text files, they can be exchanged between users. See Sharing Recording Cards with others.

To remove a recording card:

  • Select the Recording card you wish to remove from the list in the 'Name' prompt.

  • Click the [Remove] button.
  • You will be prompted to confirm the deletion.

To edit an existing recording card:

  • Select the card you wish to edit from the 'Name' list.

  • Make the desired changes.
  • Click the [OK] button to save your changes.

Troubleshooting

NOTE: If you base a Recording card on a species list from a Rucksack, then when you save the Recording card, the list of species is read from the Rucksack and saved as part of the Recording card file. However, if you subsequently change the species list in the Rucksack, the changes WILL NOT appear when you use the Recording card.

If you want the changes to the list in the Rucksack to be reflected in the Recording card, do the following:

  • Open the 'New Recording Card' window.

  • Select the card in question from the list in the 'Name' prompt.

  • Choose the [OK] button.

  • You will see a message reporting that a Recording card of this name already exists and asking you to confirm that you want to over-write it.

  • Choose the [Yes] button, and the Recording card file will be updated with the modified species list from the Rucksack.
