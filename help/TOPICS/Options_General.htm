Options General

The General tab of the Options window contains various options which can be selected to alter the way Recorder operates. Select Tools - Options from the main menu to access it:

Open with windows from last Session

If this option is selected and you close the application with one or more windows still open, then when you next start the application, those same windows will be opened.

If this option is not selected, or you close without any windows open, then when you next start the application, the 'Welcome' summary window will be displayed.

Show Welcome Window at Startup

If selected, the Welcome window will be shown when Recorder 6 starts. To see it at any other time select Help - Summary Info... from the Main menu.

Show Menu icons

Controls the appearance of menus. If selected, the icons which appear on toolbar buttons will also be shown next to the corresponding items in the menus.

Graduated menus

Also controls the appearance of menus in combination with the previous option. If selected, the menus use the Windows XP style with graduated shading.

Use Common Names

If selected, the preferred common name is shown on windows such as the Observation hierarchy, the Taxon Dictionary and Rucksacks, if one is available, e.g. "Comma (Polygonia c-album)".

Use Full Translation to Preferred Term

If selected, this makes use of the full translation across all lists provided by the nameserver. If not selected, the translation is limited to synonyms that are actually on the list used for the report.

Restrict Taxon Searches

This allows you to define what lists in the taxon dictionary or rucksacks will be searched to find the names of taxa. The options are:

  • Unrestricted: searches across all the lists that are allowed to be used for data entry. Note that certain lists cannot be used for data entry, e.g. status lists such as the Red Data Book of Birds.

  • Preferred Checklists: this limits searches to entries on the Preferred lists.

  • Current Checklist: this limits searches to entries on the checklist currently selected in the taxon dictionary browser, i.e. the checklist on view when the taxon dictionary browser is open.

  • Contents of Rucksack: the available rucksacks will be listed under this heading and when one is selected searches will be limited to the taxa on the Taxa tab in the selected rucksack.

Enable partial name search for taxa

If selected, you will be able to search for taxa using an abbreviation consisting of variable number of letters from the start of the first word or genus and from the second word or species name, separated by a space. See abbreviated species names for details.

Auto-complete search in Import Wizard

If selected, then when you use the Get button to find matches for species in the import wizard, the Find Taxon dialogue will open with the Search Text box populated with the species name from the Imported Data column. You can then edit the Search Text to help find a match. If you have Enable partial name search for taxa selected you will have more options for editing the Search Text, e.g. you will be able to remove characters from the end of the genus. This will often speed up the matching process.

Enable export of confidential occurrences

If selected, then confidential observations will be included in exports when they meet the selection criteria for an export, otherwise they will be excluded.

Minimum access level

Users with access levels of this and higher can view confidential observations.

Autocreate e-mails for Recording Schemes

Option to automatically create e-mails when doing an "export to scheme". If not selected, the appropriate file is created saved and a message appears telling the user where the file should be e-mailed to.

Remember applied filters

If selected, specifies that filters applied using the filter icon on the active window toolbar - see Simple filter - are remembered and reapplied the next time the screen containing the filtered data is opened.

Organise Surveys by Tag

If selected, Surveys in the Observation hierarchy can be organised into folders by assigning one or more tags to them. The folders appear as the top level in the hierarchy and the folder names are the tags. The tags can be created and assigned to surveys in the Observation hierarchy. They can also be created via Tools - Enhanced Termlists. Tags enable groups of surveys to be exported and included in reports more easily.

Use the v6.13 Import Wizard

If selected, the system will activate the v6.13 version of the Import Wizard instead of the version in v6.14 which has been extensively modified to speed up large imports.

Ignore Previous Import Wizard Matches

If selected, the matches remembered from previous imports done using the Import Wizard will be ignored.

Century cut-off year

This is the number which is used used to interpret dates if you type a year number with only one or two digits. It can be an integer number between 0 and 99. See two digit years.

Rapid Entry Delimiter

Defines the character used to separate data for each attribute when doing rapid data entry using the Add Species box on a Recording card. It must be a single non-alphanumeric character, excluding ~, >, <, - & /.
