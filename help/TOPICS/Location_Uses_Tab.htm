Location Uses Tab

The Location Uses tab holds information on any uses associated with the location. This tab allows the user to add or delete any number of uses.

At the top of the tab is a box which shows all of the uses associated with the location and allows the user to add, delete or edit uses.

Use

Required.

Enter a value for the usage of the location.

From

Required.

Enter a valid vague date when the use was started.

To

Optional. Enter a valid vague date when the use ceased (must be later than the 'From' date).

Potential

Optional. Enter a description of the potential for the use.

Comments

Optional. Any additional comments.

To add a use:

  • Click the [Add] button.

  • Enter the values for the use (see the table below).

  • Click the [Save] sub-record button.

To delete a use:

  • Highlight the item you want to delete.

  • Click the [Delete] button.

  • You will be asked to confirm the deletion.

To edit a use:

  • Select the item you want to edit.

  • Click the [Edit] button.

  • Make the changes to the data.

  • Click the [Save] sub-record button.
