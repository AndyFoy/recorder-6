Biotope Dictionary Browser

The Biotope Dictionary browser displays information from Biotope Classification schemes in a screen with two panels. These data are Read-only and cannot be edited – the browser is simply for displaying the data, and returning or dragging items to other screens. To add data to the Biotope Dictionary, see Add New Biotope.

The biotope browser consists of two panels and a drop-down box for the classification scheme name:

  • To display a classification scheme, select a name from the drop-down box.

    • The left hand panel displays the items on the classification scheme in a hierarchy.

    • The right hand panel shows the detail of the item which is currently selected in the left hand panel. You can choose whether or not to display this panel.

To change the sort order of the classification scheme:

  • Right click on the hierarchy, and select the preferred sort order from the menu that appears.

  • Click on the [Sort Order] button on the tool bar, and select the preferred sort order.
