Adding, deleting and sorting items

To add items to the current rucksack:

  • Go to the appropriate list by clicking on the 'Taxon', 'Biotope', 'Locations', 'People' or 'References' tab.

  • Add an item by dragging it from the appropriate window, or using copy and paste.

Taxon and biotope names can be added by double clicking items in the appropriate dictionary browser. In this case, the Rucksack does not have to be open at the Taxon or Biotope pages.

When entering taxa you can also add a free text Search Code. This code can be used in a Taxon/Search Text box when finding taxa and in the Add Species box when entering data via a recording card.

To remove items from a rucksack list:

  • Select an item by clicking on it with your mouse.

  • Remove it by:
    • Pressing the <Del> key.
    • Selecting Edit – Remove from the Main menu.
    • Clicking the button in the active toolbar.
    • Right-click and select Remove from the menu.

You can select multiple items in a list using the standard Windows techniques (<Shift-click> or <Ctrl-click>), in which case all the selected items will be removed.

To change the order of items in a rucksack list

  • Select an item by clicking on it with your mouse

  • Drag it to the position in the list where you want it. If you release it on top of another item, it will be placed immediately before that item in the list.
