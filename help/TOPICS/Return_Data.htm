Return data

  • Prompts which refer to items elsewhere in the system have a Get button

  • When you choose this button, the window from which the required information can be obtained will open, or if it is open already, it will come to the front. For example, at the location prompt in a record entry screen, choosing this button opens the Locations Window.

  • Select the item you require. For example, highlight the name of a location in the hierarchy in the left-hand panel of the Locations window.

  • Return the selected item to the original window by:

    • Selecting Edit – Return data from the Main menu.

    • Click the Return data button in the toolbar.
    • Press the short-cut key <F9> on the keyboard.

  • If you close either the 'calling' or 'returning' window in the middle of this process, the connection which is established between them will be terminated, and no data will be returned.

TIP: You can enter one or more new items in the window that is opened. Once you have saved a new item, it can be selected and returned just like any other item.
