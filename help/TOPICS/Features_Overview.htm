Features overview

Features are significant aspects of locations which are managed or monitored. They could, for example, be the reasons for which a designated site was notified, or the items for which specific provisions are made in a management plan. Features are most likely to be used by organisations responsible for the management of sites.

In Recorder, features are linked to a location, which can be at any level of the location hierarchy. A given location can have any number of features associated with it.

Apart from their general description, features can have management aims and threats. It is also possible to record occurrences of damage to the feature.

 

To add a feature:

  • In the Locations window, select the location to which you want the new feature to be attached.

    • Select Edit - Add from the Main menu.

    • Click [Add].

  • In either case, you will see a sub-menu from which you select 'Feature'.

  • The detail panel on the right will be shown in Edit mode with blank prompts ready for you to type your data.

  • Fill in the prompts with the required data.

    • Choose [Save] to save the record.

    • Choose [Cancel] to abort the addition. The Locations window will then return to its state before you started to add the feature, and no changes will be made to your database

To edit a feature:

  • Select the feature you wish to edit in the hierarchy on the left.

    • Select Edit - Edit from the Main menu.

    • Click [Edit].

  • Make any changes to the information.

    • Choose [Save] to save your changes.

    • Choose [Cancel] to abandon changes without changing the database.

To delete a feature:

  • Select the feature you wish to delete in the hierarchy on the left.

    • Select Edit - Delete from the Main menu.

    • Click [Delete].

  • You will be prompted to confirm that you really want to delete this record.
