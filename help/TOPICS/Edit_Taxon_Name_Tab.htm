Edit Taxon details - Names

The Taxon Names page of the Edit Taxon Details window allows additional names to be entered for existing species. For example, a local common name could be entered here.

Names entered here are not exported, so they are for use only within this copy of Recorder.

Use standard common name

The item that is ticked, either "Use standard common name", or in one of the rows of the table, will be the name that is displayed on screen.

Taxon name

Type a name for the taxon.

Language

The language for the name (English, Welsh, etc.). Select from the list.

To add a new name to a species:

  • Click in the next empty row in the table.

  • Type a name and select a language from the drop-down list.

  • Click the [Save] button to save your changes.

To select the name to show on screen:

  • If the "Use common names" option is selected, then the name that is ticked on this page will be used. This can be the "Use standard common name" item at the top of the page, or one of the names in the list.
