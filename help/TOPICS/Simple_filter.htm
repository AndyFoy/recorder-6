Simple filter

This tool is available for all of the main windows, and allows the number of items displayed in the hierarchy, or list, in the left-hand panel, to be reduced by specifying a single search criterion. When the filter is applied, only those items which match the criterion are displayed. This is helpful when you are trying to find a particular item amongst a large number. For example, you might want to see only the observations of a particular species in the Observation window, or only locations whose name contains the word 'wood' in the Locations window.

To apply a filter:

  • Select the type of item you wish to filter. For example, in the Observation window, select a taxon occurrence if you require all the observations for a particular species.

    • Select Edit - Filter from the Main menu.

    • Click the button in the active toolbar.

The Filter dialog will be displayed (to filter surveys in this example).

  • Select one of the 'Available Fields' from the list on the left by clicking one with your mouse or using the <up arrow>, <down arrow>, <PgUp>, <PgDn>, <Home> and <End> keys to move the highlight up and down the list. The content of the list depends upon the type of item you are filtering.

  • Select the Condition you want to use by clicking on one of the options listed to the right - only one can be selected. The options available depend upon the type of data (number, text, True/False). See the table below for details.

  • Type in the value you wish to search for in the 'Criteria' box.

  • Choose [OK] to apply the filter.

  • You can press [Cancel] at any time to close the dialog without applying a filter.

Applying a filter is very like using the [Related Data] button to view items related to the one you have selected. Related Data applies a filter for you, the criterion being 'items Equal To the selected item in the original window'.

NOTE: You can only apply one filter at a time.

Note: selecting an item and clicking on the * on the numeric keypad on the keyboard will expand that item in full.

To remove a filter:

When a filter is applied to a window (either by using the Simple filter tool, or as a result of using the [Related data] button from another window, or by loading a filter), a [Show All] button usually appears towards the bottom right-hand corner of the window.

  • Click [Show All] and the filter will be removed.

  • All data in the window will be accessible again via the hierarchy, or list, in the left-hand panel.

  • The item that was selected when the [Show All] button was clicked will be highlighted.

  •  The [Show All] button will disappear again.

Note: when using the Simple filter tool in the dictionaries, the [Show All] button appears to the right of the selection box at the top.

Note: filters can be remembered if Remember applied filters is ticked on the General tab in Tools - Options, in which case they will be reapplied the next time the screen containing the filtered data is opened.

Note: filters of different types can be saved and loaded at a later date - see Loading filters.
