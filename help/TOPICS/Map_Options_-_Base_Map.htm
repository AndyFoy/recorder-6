Map Options - Creating and deleting a Map

This facility is only available to System Managers. At least one map must be created and enabled before the mapping area of the programme can be used.

You may create and use a number of different maps within Recorder 6- each with its own basemap, layers and polygons.

  • Select Map-Map Options from the main toolbar

  • Ensure that you have the "enabled maps" page in view

You have three options from this page: you can create,delete or reset each of your maps. You can also choose which one of your maps is the default.

 To create a map and choose a base map for it

  • Click on the [Add] button

  • Change the display name to something appropriate

  • Choose the base map you wish to use by clicking on one of the options in the drop-down list

For information about MapServer and Base maps see Base map and Selecting a base map.

  • Click on the [close] button

To delete a map

Warning:Deleting a map will remove all of the polygons and background layers that are associated with that map from the programme. If you do not want to lose these objects you should transfer these objects to another map before you go ahead with the deletion. See map menu for details

In a network installation, the site boundary polygons and transect lines are a shared resource, so this will remove the polygons from the map for all users of the networked copy.

  • Select the map that you want to delete

  • Click on the [delete] button

Resetting a map

Warning:Resetting a map will remove all of the polygons and background layers that are associated with that map. If you do not want to lose these objects you should transfer them to another map before you go ahead with the deletion. See map menu for details

In a network installation, the site boundary polygons and transect lines are a shared resource, so this will remove the polygons from the map for all users of the networked copy.

  • Select the map that you want to delete

  • Click on the [delete] button

The default map

Exactly one map is marked as the default using the tick boxes to the left of the screen. If you use the button to go to a map to pick up a geographical information for a record or "send details to map" from the report wizard, it is the default map that will be used.
