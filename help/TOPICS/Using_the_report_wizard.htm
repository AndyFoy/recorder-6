Report Wizard

The Report Wizard is started by:

  • Selecting Reports – Wizard from the main menu.

  • Pressing the short-cut key <Ctrl-W>.
  • Choosing the [Reports] button in the Welcome screen.

The Report Wizard consists of a series of screens which ask you to make choices about the type of report you require or to select the items (places, taxa, biotopes, etc.) on which to report.

When you have selected options, or entered values in a screen, press [Next] to proceed to the next screen. You can use the [Back] button at any time to go back and change your selections. The [Cancel] button can also be used at any time to abort the process.

When you start the Report Wizard the first window will ask you to select the report style. The windows that follow will depend on your selection, however, they will include:

  • Taxa Selection, Biotiopes Selection or Places selection

  • Layout Selection
  • Select Attributes if you selected 'New Layout'
  • Sort Attributes if you selected 'New Layout'
  • Filter Result

The Filter Result is an intermediate stage. It lists the observations that meet your selection criteria. From this point, you can print the results, present them on a map, export them in a variety of formats or select/design a report template (layout). The export formats include Excel, the format required to enable the observations to be imported into another copy of Recorder and the option to create a snapshot. For more details, see Filter Result, which includes details of how to save and re-run reports.

Note: Although the Wizard is designed to answer the questions about the observations for taxa, biotopes and places, it contains many ways of varying the observations selected. The [Additional Filters...] button on the Layout Selection window allows you to set Conditions and change the Constraints, e.g. you could restrict your report to observations made between specified dates or to observations with a particular record type. Another example is that you could use the 'Restrict to Bounding Box' or 'Restrict to one or more Polygons' options on the first window combined with the 'Include All Taxa' box on the Taxa Selection window to report on all observations in a bounding box or in several polygons.

Note: The particular screens of the Wizard you see, and the order in which you see them, depends on the options you select.

Tip: For more detailed help use the wizard to prepare a report and use the context sensitive help, <F1>, for each window that appears.
