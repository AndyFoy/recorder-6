File Menu

 

Save

Save a file. Only active when there is a file to be saved, for example in Rucksack, or when a report template is being designed.

Save as

Save a file in a new location or with a new name. Active as above.

Print preview

View the layout of a report on screen before printing or saving to a file. Only active during reporting.

Print

Only active when there is something to print: i.e. in detail panel of Taxon and Biotope browser, from a map or during reporting.

Exit

Close the application.
