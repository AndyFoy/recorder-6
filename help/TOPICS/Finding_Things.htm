Finding things

Many prompts require data stored elsewhere in the system, and it is necessary for the user to find the correct item. There are a number of ways to do this:

  • Copy the required item to the Windows Clipboard and paste it into the prompt.

  • Drag the required item from another window, and drop it into the prompt.

  • Click the [Get] button located next to such prompts. This displays the relevant window in which the item you require can be located using the [Find] button, then returned using the [Return data] button or one of the other options. In some windows the [Add] button provides this function, e.g. Geo Info - Administrative Areas on locations.

  • Search for the required item by typing some text and pressing <Enter>. This opens a Find dialog, allowing items to be found by searching for matching text. If you prefix the text you type with an asterisk, *, then the Matches box will show names containing the text.

  • When you are in a list such as the hierarchy in the left hand panel of the main windows or dictionary browsers, click the [Find] button in the active toolbar, or press the short-cut key <Ctrl-F> to open the Find dialog.

Find Dialogs can be used to find:

  • locations

  • taxa

  • biotopes

  • names and addresses

  • documents

  • administrative areas

  • surveys

  • survey events

  • samples

  • taxon occurrences

  • biotope occurrences
