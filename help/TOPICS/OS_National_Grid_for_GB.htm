OS National Grid for GB

Grid references for Great Britain are given by specifying the two letter code for the 100km square, and then giving an even number of digits representing the eastings and northings of a point within the 100km square. For example: NZ123456.

When entering a grid reference:

  • A valid two letter code is required - the letters are automatically forced into upper-case.

  • The number of digits must be even, and may range from 2 (10km accuracy) to 10 (1m accuracy).

  • Spaces can be used to improve readability if desired (e.g. NZ 123 456).
