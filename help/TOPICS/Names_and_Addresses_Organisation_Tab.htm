Names and Addresses Organisation Tab

This page allows details about an organisation to be entered.

 

Acronym

Free text. Enter an abbreviated name for the organisation.

Full name

Required.

Free text. Enter the organisation's name in full.

Type

Pick a value from list. This is a term list to which the user can add entries.

Founded

Date on which the organisation was founded. Accepts a vague date.

Ended

Date on which the organisation was wound up. Accepts a vague date.

Comment

RTF memo in which any comment can be typed.

The Address panel is Read-only, and is automatically generated from the preferred address and contact numbers. It is shown for quick reference to the organisation's name and address.
