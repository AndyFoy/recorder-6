Mapping- pan

This command allows you to move the map display, without changing its scale, to view different parts of the map.

To pan:

  • Select the pan tool:

    • Select Map – Pan from the Main menu.

    • Click the button on the active toolbar.

    • Press the short-cut key <F6>.

  • The map cursor changes to .

  • Click on the point on the map you wish to move, and hold down the primary mouse button.

  • Drag the mouse to the spot where you would like that place to appear on the screen.

  • Release the mouse button, and the map will be redrawn so that the point at which you clicked is now at the place where you released the mouse button.

You can move the map by larger amounts using the Mapping scrollbars.
