Linking polygons to locations

Polygons and lines can be associated with Locations in the location hierarchy or Administrative Areas in the Admin Area Dictionary. The entry in the location hierarchy provides information about the polygon, such as the name of the site and its designation status. When a polygon has be associated with a Location the Boundaries tab under the Geo Info tab will provides information about the polygon, such as the map and polygon layer it is on.

Note: The instructions below are for associating one polygon with one location. When importing polygons multiple polygons can be automatically linked to locations at the same time. You can match polygons to locations manually, one by one, or by matching a polygon attribute to Location Keys, Location Names or Field Codes for locations in the location hierarchy.

To associate a line or polygon with a location:

  • Select the pointer tool so that the map-cursor is .

  • Click on the polygon or line you wish to associate. It will change colour (from the unselected colour to its selected colour for that polygon layer) to indicate that it is selected.

    • Select Map – Associate Boundary from the Main menu, or

    • Click the button on the active toolbar, or
    • Right-click and select Associate Boundary from the resulting menu.

  • If the line or polygon is already associated with a location, the Location or Admin Area Dictionary window will open with the details of that particular location displayed.

  • If the line or polygon is not yet associated with a location, you will be see a dialog:

    • Create a new location. Select this option and choose [OK]. The Location window will open in Edit mode. The site centroid grid reference is already filled in, and ready for the location name to be typed. Enter at least the required information for this location, and save the record to complete the link.

    • Link the boundary to an existing location. Select this option and choose [OK]. The Location window will open with the Find location dialog displayed to locate the particular site. Select a location. It will be shown in Edit mode, with the polygon ID filled in appropriately. Save the record to complete the link.

    • Link the boundary to an Admin Area. If this option is selected, the Admin Area Type box will become active. You can select the type of Admin Area (e.g. county, vice-county, National Park, etc.) with which you wish to associate it. Choose [OK] and the Link Boundary to Admin Area dialog will open:

Type the name (or parts of it) of the Admin Area you want into the Search Text box and choose the required Admin Area from the list of Matches. Press [OK] to save the link. Pressing [Cancel] will abort the process - no link will be made. This dialog works the same as the one to find Admin Areas.

  • You can also associate a location with a polygon starting from the Location window.
