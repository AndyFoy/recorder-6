Raster map registration

The Map Registration dialog is used to enter or change the details for a raster background map.

Name

Free text. Type a name which will help you recognise this map in future.

File Name

The name of the file, in one of the formats supported by MapServer, from which to read the map.

Choose the button for a standard Windows file dialog which will allow you to locate and open the file.

South-West

If the selected file is a raster tile, enter the spatial co-ordinate for its south-west corner. This information is not needed for vector map files, so the prompt will be greyed out.

North-East

If the selected file is a raster tile, enter the spatial co-ordinate for its north-east corner. This information is not needed for vector map files, so the prompt will be greyed out.

Cut In

The scale at which this background map should become visible. See Cut-in and cut out scales.

Cut Out

The scale at which to hide this background map. See Cut-in and cut out scales.

When the necessary details have been filled in, choose [OK] to save or update the details. [Cancel] abandons the process of adding a map without making any changes to the map layer.

NOTE: It is very important that the south-west and north-east spatial references for raster tiles are correct and as accurate as possible. The spatial references obtained by clicking on a map are entirely dependent on these values. If you make a mistake here, then any spatial reference obtained for observations or locations by clicking on the map will be incorrect. Once you have loaded a map, it is worth checking the spatial references reported when you click on some known points, to ensure that it has been configured correctly.
