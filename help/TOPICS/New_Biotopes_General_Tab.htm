Edit Biotope details - General

The Edit Biotope details -General page holds information on a user-added biotope item. This tab allows the user to add, edit and delete information for a user-added biotope only. It does not allow the user to edit system-supplied data.

Short term

Optional. The short term for the biotope. This value is displayed in the biotope tree view.

Full term

Required.

The full term for the biotope.

Original code

Optional. The original code for the biotope, if one is used.

To enter a new biotope:

  • Click on the [Add] button .

  • Select either a sibling or a child to add.

  • The hierarchy changes to grey to indicate that an item is being added.

  • Enter the values for the new biotope.

  • Click [Save].

To edit a biotope:

  • Select the item that you want to edit.

  • Click on the [Edit] button .

NOTE: With system-supplied data (see System Supplied vs User Added. and new taxa/biotopes), no information can be edited on this tab. However, with user-entered data, all items can be edited.

  • Make the changes to the values.

  • Click [Save].

To delete a biotope:

  • Select the item that you want to delete.

  • Click [Delete].
  • You will be prompted to confirm the deletion.

NOTE: System-supplied data (see System Supplied vs User Added. and new taxa/biotopes) cannot be deleted, but user-entered data can be.

TIP: A biotope cannot be deleted until all of it's children have been deleted.
