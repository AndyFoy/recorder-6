Saving and loading rucksacks

To load a rucksack:

  • Select Tools - Rucksack from the main menu.

  • The drop-down list at the top of the rucksack window shows the names of files in the default directory.

  • Select one of the names from this list, and the rucksack will be loaded.

  • The button opens a standard Windows file dialog which allows you to open rucksack files (.Ruk) in any other directory to which you have access.

To save a rucksack:

  • When you close the rucksack window, or change to a different rucksack, and you made changes to the contents of the current rucksack, you will be prompted for whether you want to save these changes.

  • You can save a rucksack at any time by selecting File – Save from the Main menu. If the rucksack has not yet been saved to a file, you will be prompted for the name and location in which to save it (with the directory shown in Tools – Options – File Locations – Rucksack Files as the default).

  • You can save a rucksack with a new name, or in a new location, by selecting Files – Save as from the Main menu.

TIP: The file name you give appears in the list at the top of the rucksack window – so this is the name by which you will recognise a rucksack in future. Recorder supports long file names, so give your rucksack files meaningful names. It will make it much easier to remember for which purpose a particular rucksack was set up.

To create a new, empty rucksack:

  • The first time you open the rucksack window in a particular session, it will be empty until you load an existing rucksack file.

  • Otherwise, select the '<Blank rucksack>' item from the list at the top of the Rucksack window.

  • You can now start adding items to the lists in the new Rucksack.

By default, rucksacks are saved in the directory shown in Tools – Options – File Locations – Rucksack Files, but they can be saved in any location to which the application has access.
