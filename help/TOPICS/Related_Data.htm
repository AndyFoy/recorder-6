Related data

This facility allows you to move from a selected data item in one window to any of the other items of data in other windows to which it is related. For example, you could go from a recorder's name to the observations recorded by that person, or from a sample to its location. The selected item filters the destination window, so that it only displays the data related to that item.

All of the main windows have a [Related Data] button at their bottom-right corner. To use this button:

  • Select an item in the window.

  • Choose the [Related Data] button at the bottom-right corner of the window.

  • A list will be displayed of the types of data that can be related to the selected item.

  • Select one of the types from the list, and the appropriate window will be displayed with the list or hierarchy on the left containing the relevant, related items only.

Note: selecting an item and clicking on the * on the numeric keypad on the keyboard will expand that item in full.

To return to viewing all data in the window click the [Show All] button beside the [Related Data] button. This removes the filter and makes all data in the window accessible again via the hierarchy, or list, in the left-hand panel. It also highlights the item that was selected when the [Show All] button was clicked.

Example 1:

  • Select a recorder's name in the Names and Addresses window.

  • Choose the [Related Data] button, and select "Occurrences" from the list.

  • The Observations window opens showing the surveys in the observation hierarchy that contain observations recorded and/or determined by the selected recorder. Expand the surveys to display all the taxon and biotope observations recorded and/or determined by this person.

Example 2:

  • Select a sample in the Observation Window.

  • Choose the [Related Data] button and select "Locations" from the list.

  • The Locations window will open with a filter on it so that the location from that sample is on view. If that particular location is part of a sub-site hierarchy, the site and all its parents will be displayed, with the relevant sub-site selected.
