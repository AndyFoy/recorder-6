Map Menu

Map window not open

 

View Map

Opens the mapping screen to view the selected map

Map Options

Open the Map Options dialog manage distribution data sets, map layers and the base map.

Map Navigator Tool

Open the Map Navigator Tool window.

Map window open

 

Map

Opens the selected map or brings it to the front if it is already open.

Pointer

Obtain a spatial reference by pointing at a place on the map. Select an item on the map (e.g. a site boundary).

Zoom

Redraw a selected area of the map in greater detail.

UnZoom

Expand the area of the map you are viewing so that a larger area is shown in less detail.

Zoom to extents

Redraw the map display so that it shows the whole area of the base map.

Pan

View a different part of the map without changing its scale.

Find Source Data

Go to the observation(s) which generated a selected distribution map symbol.

Add boundary

Add a site boundary or transect line which can be associated with a location.

  • Draw

Draw the site boundary or transect on the map using your mouse.

  • Import

Import the site boundary or transect from a vector file generated by some other mapping software.

Delete polygon

Remove a site boundary or transect from the map.

Create hole in polygon

Used to create polygons with holes in them. Both the polygon itself and the "hole" must be created before this button can be used.

Associate boundary

Attaches a polygon boundary to a location or administrative area so that the location or administrative area is represented by the boundary or boundaries. Polygon must be selected

Move polygon to layer

Move polygon to a different polygon layer.-polygon must be selected

Extract Grid Squares

See Location Grid Squares

Export Distribution Points

Exports data to SHP files

Add Polygon Layer

Add a new polygon layer.

Add Background Layer

Add a new background layer.

Export Current Polygon Layer

Exports the current polygon layer as ESRI shape files (.shp).

Export All Polygon Layers

Exports all polygon layers as ESRI shape files (.shp).

Distribution points

Show a distribution map for a taxon, biotope, survey events or samples for a particular survey, or the results of the Report Wizard.

  • 10km

Show distribution represented as presence in 10km squares.

  • 5km

Show distribution represented as presence in 5km squares.

  • 2km

Show distribution represented as presence in 2km squares.

  • 1km

Show distribution represented as presence in 1km squares.

  • 100m

Show distribution represented as presence in 100m squares.

  • Default

Show distribution represented as points centred on the spatial reference given in the original observation.

Grid lines

Show the spatial reference system’s grid overlaid on the map.

  • 100km

Show grid lines at 100km interval.

  • 50km

Show grid lines at 50km interval.

  • 10km

Show grid lines at 10km interval.

  • 1km

Show grid lines at 1km interval.

  • None

Don’t show the grid.

Add sample

Enter taxon observations from a particular sample by positioning an icon representing the sampling method on the map. This will then open a recording card with the spatial reference and location (if the icon was positioned within a site boundary associated with a location) via which the taxon observations can be entered.

Clear

Remove all distribution symbols from the map.
