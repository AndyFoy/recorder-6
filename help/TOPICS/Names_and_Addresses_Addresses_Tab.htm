Names and Addresses Addresses Tab

This page allows addresses for an organisation or individual to be managed. One address is marked as 'preferred' – this is the address which appears in the summary on the first page.

 

Address

Required.

Type the address of the individual or organisation.

Post code

Type the post code. Letters are forced into upper case.

Country

Free text. Type the name of the country for the address location.

From

Required.

The vague date for which the person or organisation was at this address. Although this is required, 'Unknown' is an acceptable value.

To

The vague date at which the person or organisation left this address.

Comments

RTF memo. Any comment.

Home/Work address

For individuals only, select one of these two options. 'Home address' is the default.

Preferred

If you only enter one address, this option will be greyed out, and the address is automatically 'preferred'. If more than one address is entered, only one can be 'preferred'. Click in the preferred box to mark this address. The tick will automatically be cleared from the address that was previously marked as preferred. The summary on the first page shows the preferred address.

To add an address:

  • Click on the button, and the detail panel will go into Edit mode with all the fields empty.

  • Type in the required information.

  • Click the button to save your changes.

  • Click the button to abandon your changes.

To delete an address:

  • Highlight the address that you want to delete.

  • Click the button.

  • You will be asked to confirm that you really want to delete this record

To edit an address:

  • Highlight the address that you want to edit.

  • Click the button to put the detail panel into Edit mode.

  • Make the changes you require.

  • Click the button to save your changes.

  • Click the button to abandon your changes.
