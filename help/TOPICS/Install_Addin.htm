Installing Add-Ins

For details of the add-ins that can be installed in Recorder, see the Recorder web site, www.jncc.gov.uk/recorder.

This facility is accessed by selecting Tools – Install – Add-In module from the Main menu.

The list shows modules that are already installed. If a module is selected in the list, its description (Read-only) will appear in the 'Description' box.

To install an Add-In:

  • Click [Install].

  • This will open a standard Windows file location dialog.

  • Locate the Add-In module’s file which will have a .ocx or .dll extension.

  • Choose [Open] to install the selected file and close the file dialog.

  • The module will be installed.

  • If this module has previously been installed on this copy (an earlier version, for example) - even if it has been removed - you will see the following message. Click [Yes] and installation will proceed normally.

NOTE:

  • An add-in may be supplied with several files, apart from the .dll or .ocx itself. These typically include one or more files with ".bmp" extensions and sometimes an ".mdb" file and perhaps others. All these files MUST be in the same directory as the .dll or .ocx file when you do the installation. If the files that the add-in requires cannot be found, the installation will fail.

  • You cannot install an add-in from the \Recorder\Addins directory. You must install the add-in and its related files from a floppy-disk, CD or some other location on your computer's hard disk. Recorder will copy the files to the Addins directory as part of the installation process. If you attempt to install an add-in from the same directory that Recorder is trying to copy it to, you will see the message:

The error occurs, in this case, because Recorder itself is the 'other process' that is already using the file. What you have asked it to do is to simultaneously read and write the same file!

To remove an Add-In module:

  • Highlight the module you wish to remove in the list.

  • Click [Remove].

  • You will be asked to confirm that you really want to remove this module.

NOTE: If the Add-In module is implemented as a DLL, you will be unable to remove it and then install it again within the same session (if you attempt this, you will see the "Error 32:" message above). You must close the application and restart before the add-in can be re-installed. This is a limitation imposed by Windows, because DLLs are not removed from memory until the programme that loaded them terminates.
