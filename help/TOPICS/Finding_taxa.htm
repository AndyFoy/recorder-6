Finding Taxa

This dialog allows you to search for a taxon name by looking for matches to text you type.

When you are prompted for a taxon name, this dialog can be accessed by typing some text and pressing <Enter>, although it will only appear if there is more than one match. The text you typed will appear in the 'Search Text' box. You can add to or change the text in this box at any time to refine your search.

When you are in the Taxon Dictionary Browser window, you can also access it by:

  • Selecting Edit - Find from the Main menu.

  • Pressing the short-cut key <Ctrl-F>.

  • Clicking the button in the active toolbar.

To find a taxon name:

  • Type some text likely to appear in the name you want to find in the Search Text box. The Matches box will show names starting with the string you type, if the Search By option is set to Name.

  • If you prefix the text you type with an asterisk, *, then the Matches box will show names containing the text, if the Search By option is set to Name, e.g. type "*warbler" to find all warblers (Aquatic Warbler, Dartford Warbler, Eurasian Reed Warbler, etc.).

  • See abbreviated species names for information about the use of 5-letter abbreviations and partial name searches. The Search By option must be set to Abbreviation when using 5-letter abbreviations.

  • If Search Codes have been added to a rucksack and Restrict Searches to has been set to the rucksack concerned then search codes can be used instead of matching text.

  • Select an item in the Matches box by clicking on it with your mouse, or press <Tab> to position the cursor in the list and then use the <up arrow>, <down arrow>, <PgUp>, <PgDn>, <Home> and <End> keys to position the highlight over the required taxon.

    • Choose [OK] or press <Enter> to select the highlighted taxon.

    • Double-click on an item in the list to return it.

    • Choose [Cancel] or press <Esc> to close the Find dialog without making a selection.

  • Restrict Searches to allows you to define what lists in the taxon dictionary or rucksacks will be searched to find the names of taxa. The options are:

    • Unrestricted: searches across all the lists that are allowed to be used for data entry. Note that certain lists cannot be used for data entry, e.g. status lists such as the Red Data Book of Birds.

    • Preferred Checklists: this limits searches to entries on the Preferred lists.

    • Current Checklist: this limits searches to entries on the checklist currently selected in the taxon dictionary browser, i.e. the checklist on view when the taxon dictionary browser is open.

    • Contents of Rucksack: the available rucksacks will be listed under this heading and when one is selected searches will be limited to the taxa on the Taxa tab in the selected rucksack.

    You can change your selection using the drop down arrow or by changing 'Restrict Taxon Searches to' on the General tab in Tools - Options.

Note: Restrict Searches to defaults to the last selection used if Find Taxon is being used by typing some text in the Taxon box and pressing <Enter> on a data entry screen like Data Entry - Enter a species record... If Find Taxon is being used in the taxon dictionary browser it defaults to Current Checklist but if the required entry is not found Restrict Searches to can be changed. If it is changed and an entry is selected from another list, that list opens with the selected entry highlighted.

Note: the searches work on both scientific and common names.
