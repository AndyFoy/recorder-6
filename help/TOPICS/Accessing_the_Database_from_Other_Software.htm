Accessing the database from other software

The database used to store all the data entered into the Recorder application is a SQL Server database. It can be accessed directly if you have the appropriate tools, e.g. SQL Server Management Studio or Microsoft Access.

The database can also be accessed through an ODBC connection, if required. This allows the use of other software tools for reporting and analysis.

In a standard Recorder installation the database will be a file called NBNData_Data.MDF. Associated with it will be a log file called NBNData_log.LDF. In a standard standalone SQL Server Express installation with Windows XP, these will be stored in the Data folder in a Microsoft SQL Server folder, e.g. C:\Program Files\Microsoft SQL Server\MSSQL.1\MSSQL\Data for SQL Server 2005 Express.

Note: The Setting table in NBNData_Data.MDF holds the version of the database in the DATA field where NAME = "Version". It also holds the version of the dictionary in the DATA field where NAME = "Dict Seq".

Warning: Please be aware that changing records directly can seriously affect the integrity of your database. IT SHOULD NOT BE DONE UNLESS YOU ARE VERY SURE THAT YOU KNOW WHAT YOU ARE DOING.

When Recorder is installed, a linked Access database is also created so that users with Access expertise can query the SQL Server database directly. This database is a file called NBNData.mdb. In a standard installation, it is located in the Recorder 6 Database folder on the drive where Recorder is installed. Note that it contains links to the tables in the SQL Server database, it does not contain the data itself.

Finally, you may choose to use the Snapshot Tool to provide you with the data that you need in a simplified format.

The advantages are:

    • The data is in a simpler format that will be easier to manipulate and use in another application

    • The snapshot is not the database itself. Therefore, there is no chance of accidentally altering your Recorder database.

    • Once a snapshot is designed and made, it can be refreshed at the touch of a button.
