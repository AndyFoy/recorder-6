Map Navigator Tool

The Map Navigator Tool enables you to navigate to a specific polygon in the Map Window. If you have drawn or imported polygons, and associated them with locations in your location hierarchy (optional) you will be able to use this tool to view a particular polygon.

To use this tool, select Map - Map Navigator Tool from the main menu, then:

  • Select the required base map in the Map box.

  • Select the polygon layer that contains the required polygon in the Layer box.

  • Select the required polygon in the next box, then

    • Click [Centre] to centre the map on the required polygon. Note that this does not change the scale of the map.

    • Click [Zoom] to centre the map on the required polygon and zoom to an appropriate scale.

    • Click [Select] to centre the map on the required polygon and select it ready for use. Note that this does not change the scale of the map.

Note: Polygons that have been associated with locations in your location hierarchy will be listed with the location name. Those that haven't will be listed as 'Unlinked Polygon'.
