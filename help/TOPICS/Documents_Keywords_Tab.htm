Documents Keywords Tab

The Documents Keywords tab lists the keywords assigned to a document. This tab allows the user to add or delete any number of keywords for a document. To add or delete keywords you must be in Edit mode.

Keywords can be used when finding documents.

Keyword

The keyword assigned to the selected document.

The keywords must be selected from the Enhanced Term Lists but if the required keyword is not present it can be added to this list.

To add a keyword:

  • Click the Add button

  • Use one of the standard methods for finding things to select the required keyword from the 'Available Terms' on the Enhanced Term Lists window. Repeat for each keyword required.

  • When you have added all the required keywords and completed any changes on the other tabs, choose [Save] in the right-hand panel of the window to update the database with the changes for the selected document.

  • You can choose [Cancel] in the right-hand panel to abandon any changes. No changes will be made to the database, and the window will be restored to its original state.

To remove a keyword:

  • Ensure that the keyword you wish to remove is selected.

  • Click the Delete button

  • You will be prompted to confirm that you really want to delete this item.

To edit a keyword:

  • Remove it and add the required keyword.
