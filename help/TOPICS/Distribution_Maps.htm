Distribution maps

Recorder provides two ways of producing distribution maps.

  • Using the built in mapping facilities described in this topic.

  • Exporting data to DMAP. See DMAP export.

Using the built-in mapping facilities, it is possible to map the distribution of observations of taxa, biotopes, survey-events, and samples. Distribution maps can be generated by:

  • dragging a name from the Taxon Dictionary browser on to a map;

  • dragging a name from the Biotope Dictionary browser on to a map;

  • dragging a survey or survey event from the Observations window on to a map;

  • sending the results of the Report Wizard to the map from the Filter Result screen.

By default, distribution is initially indicated by a small coloured circle plotted at the spatial reference of the observation, sample or survey. The symbol size tool can be used to show presence in 10km, 5km, 2km or 1km grid squares. When presence in a grid square is selected, symbols are shown either filled in or hollow, depending upon whether they are before or from a given year onwards. The year is set in the Map Options - Distribution symbols screen. Distribution maps can be printed.

If Display grid references as squares on map on the Spatial References tab on Tools - Options from the main menu is ticked, the distribution points will appear as squares covering the area represented by the grid reference, e.g. an observation with a grid reference of SU1234 will display as a square covering an area 1 km x 1 km. The squares will be partially transparent so that you can see through to the underlying map and also to any other, more precise grid references. This only applies to the OS - GB and OS - Ireland spatial reference systems.

It is possible to show the distribution of up to 10 items on a single map. For example:

  • Drag a survey onto the map to show the distribution of samples or survey-events.

  • Now drag a taxon name onto the same map.

The result shows the survey events or samples for which the selected species was recorded. The colour and legend of symbol-sets can be changed and symbol sets can be turned on and off independently. See Map Options - Distribution symbols. You can also find the observations(s) which generated a particular distribution map symbol by clicking on a symbol.
