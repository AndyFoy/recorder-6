Location Designations Tab

The Location Designations tab holds information on any local or national designation(s) associated with the location. This tab allows the user to add or delete any number of designations for a location.

The box at the top of the tab shows all of the designations for the location, and allows the user to add, edit or delete designations.

Designation

Required.

The type of the designation. To enter a designation, select one of the values from the drop-down box.

The designation type is a ‘Term-list’, and items in this list can be added or deleted by clicking Options - Term-lists.

Ref code

Optional. Any user-specific file code can be entered here.

Authority

Required.

The authority responsible for the designation.

To enter an authority, use one of the methods described in Find overview.

From

Optional. Enter a standard date for when the designation was applied.

To

Optional. Enter a standard date for when the designation was removed. This date must be after the 'From' date.

Comments

Optional. Enter comments about the designation.

To add a designation:

  • Click on the [Add] button.

  • Type in the required information (see the table below).

  • Click the [Save] sub-record button.

To delete a designation:

  • Highlight the designation that you want to delete.

  • Click the [Delete] button.

  • A confirmation box is displayed.

To edit a designation:

  • Highlight the designation that you want to edit.

  • Click the [Edit] button.

  • Edit the record, and click the [Save] sub-record button.
