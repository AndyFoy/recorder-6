User Access Levels

Recorder supports the following levels of user access:

System Manager

Full access to all facilities and information. Can create other users and has access to database management tools in the Tools menu.

Full Edit

Can add, edit or delete all types of data, but cannot create users and does not have access to database management tools in the Tools menu. Has access to observations marked as 'confidential'.

Full Edit (Own data only)

As for Full Edit, but can only edit or delete data input by that user name.

Add Only

Can add data, including items to data input by other users, e.g. can add determinations to observations input by other users, but cannot edit or delete any data. Does not have access to data marked as 'confidential'.

Record cards only

Can use record cards, or single record entry, to add observations, but cannot edit or delete anything.

Read Only

Can browse data except for observations which are marked as 'confidential', but cannot add, edit or delete anything.
