Short-cut Keys

The following short-cut keys can be used in Recorder:

F1 - Context sensitive Help - see Getting Help.

F2 - Zoom in - Map window - see Map Browser Zoom In.

F2 - Entering data using a Recording Card:

  • When the cursor is in a cell in a Spatial Ref column, <F2> acts as a [Get] button to enable you to Get a spatial reference from the Map.

  • When the cursor is in a cell in a Biotope column, <F2> acts as a [Get] button to enable you to Get a biotope from the Biotope dictionary.

F3 - Zoom out - see Map Browser Un-Zoom.

F5 - Map pointer tool - see Map Browser Pointer.

F6 - Map panning tool - see Map Browser Pan.

F8 - Create hole in polygon - see Holes in Polygons.

F9 - Return data - see Return Data.

F11 - Auto-populates fields with current date or current user:

  • When the cursor is in an editable field which is either a standard date or vague date entry field, pressing <F11> puts the current date in.

  • When the cursor is in an editable field which requires an entry from the Names and Addresses window, pressing <F11> puts the name of the user currently logged on, as displayed on the Welcome window, in the field.

Esc (escape) - Cancel
