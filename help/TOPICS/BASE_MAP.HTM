Base Map

MapServer maintains its own information, quite separately from the database maintained by Recorder in which it stores map objects. The mapping co-ordinate system used to store these objects is established when the first map is imported. It cannot subsequently be changed, unless all maps are deleted and you start again from scratch. Consequently, it is necessary to decide which co-ordinate system you wish to use for maps. See Selecting a base map.

NOTE: It is possible to change the base map (see Reset map), but it is a profound step, which involves removing all site-polygons and their links to locations.

Base map settings you can change:

The Base map is treated like a background map layer for most purposes and you can:

  • Rename it.

  • Change its cut-in and cut-out scales.
  • Turn it on and off.

See Map settings.

However, you cannot (except by Resetting the map):

  • Remove it.

  • Change the outline that is used.
