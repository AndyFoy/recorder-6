Select Report Style

The first screen selects the type of report you want to produce.

Select report style: selects what type of question you want to ask. Do you want to select observations about taxa, biotopes or a place? You must select one of the following:

Taxon records

Select observations relating to one or more taxa.

Biotope records

Select observations relating to one or more biotopes.

Report about a place

Select observations of taxa and/or biotopes relating to a place. The place can be defined by selecting one or more locations, or by selecting a geographical area defined by a rectangular area on the map. See Report about a place.

In addition you can optionally:

Restrict report to one or more sources

Select only those observations relating to one or more surveys, survey tags, individuals, organisations or documents.

Restrict report to one or more admin areas

Select only those observations associated with locations attached to one or more administrative areas.

Restrict to bounding box

Select only those observations which fall within a rectangular area of the map which you specify.

Restrict to one or more Polygons

Select only those observations that are within one or more polygons on the map.

Restrict report to one or more Designations

Select only those observations that have specified designations.
