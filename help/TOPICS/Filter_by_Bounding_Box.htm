Filter by Bounding Box

You will see this screen if you selected the option to 'Restrict to bounding box' when selecting the style of report. This screen allows you to specify the bounding box you wish to use for this particular report.

 

To specify a bounding box:

  • Type the south-west and north-east spatial co-ordinates you require.

  • Click the button to use the Map Browser to draw a bounding box.

See Bounding boxes.
