Drag and Drop

When you need to add or edit a reference to an item elsewhere in the system, you can usually do so by dragging the required information and dropping it into the control. For example, you could drag a location name from the Location window into the location prompt of a record card.

  • Use your mouse to select the information you want to use.

  • Drag it to the place where you want to use it by holding down the primary mouse button while you move the mouse. The cursor will change to , indicating that you are dragging something

  • When the cursor is over the place where you want to use the information, release the mouse button to 'drop' the item.

Troubleshooting

  • If you are over a place where you cannot drop the item, the cursor changes to . This could be because:

    • The information you are dragging is not appropriate at that point (for example, you cannot drag a species name into a prompt for a location).

    • The window into which you are trying to drag the information is not in Edit mode.

    • You do not have the necessary access rights to add or change the data at the place to which you are trying to drag the item.

Coloured borders can be used to indicate controls that can be the source and destination of items to drag and drop. See Options - Appearance
