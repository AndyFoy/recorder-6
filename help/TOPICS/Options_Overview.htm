Options overview

This window allows you to customise various aspects of the behaviour and appearance of Recorder. Your selections are saved in the system registry.

  • [OK] applies the currently selected options.

  • [Cancel] abandons any changes that were made. The selected options will be unchanged.

  • [Default settings] returns the settings IN THE TAB WHICH IS CURRENTLY DISPLAYED ONLY to its original default settings.

If the computer is used as a workstation on a network, or it has been set up for use by two or more users under Windows, then each user will have their own set of registry settings which can include different options for Recorder. Note that this applies to users of the operating system, NOT Recorder 2002 user names.
