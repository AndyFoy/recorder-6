Moving polygons between layers

To move a polygon from one polygon layer to another:

  • In the Map window, select the polygon you want to move by clicking on it with the pointer tool.

  • Click on the button in the active toolbar or select Map - Move Polygon to Layer from the menu.

  • Select the name of the layer to which you wish to move the polygon from the drop-down list in the toolbar, or from the sub-menu.
