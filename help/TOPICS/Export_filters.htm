Export filters

Export filters enable you to export data from one or more surveys or taxonomic groups, e.g. all amphibian and reptile observations, in one operation. The filters are stored in the database and can be used again and again as new data is added or existing data changed. When doing an export using a filter you have the option of exporting all the data for the filter or only data that has been entered or changed since a specified date. Filters can also be saved externally and given to other users. If the export is always to be e-mailed to the same person or organisation you can set up a Recording scheme to automatically e-mail the exported data to a specified e-mail address.

To create an export filter:

  • Select Tools - Export Management - Manage Export Filters from the Main menu.

  • Choose [Add] at the bottom of the left-hand list.

  • Enter a filter name of your choice in the mandatory Filter Name box.

  • To add survey(s) to your filter:

    • Click the [Add] button beside the Filter on Surveys box and the Select Survey dialog box is displayed. See Select Survey.

  • To add survey tag(s) to your filter:

    • Click the [Add] button beside the Filter on Survey Tags box and the Find Survey Tag dialog box is displayed. Select survey tags in the same way as you select surveys.

  • To add taxonomic group(s) to your filter:

    • Click the [Add] button beside the Filter on Taxonomic Groups box and the Select Taxa dialog box is displayed. See Select taxa.

Note: When an export is done using a filter that contains taxonomic group(s), it will include observations for the taxa specified and for all taxa below them in the taxon dictionary hierarchy, e.g. if Amphibia from the AMPHIBIA list is included the export will include all observations for amphibia.

  • Repeat the process above for each survey/survey tag/taxonomic group you want to add to the filter.

Note: Drag and drop can be used to add items to a filter when the observation hierarchy or taxon dictionary browser is open.

  • To remove a selection from your filter:

    • Click on the item to be removed to highlight it.

    • Click the [Delete] button

  • Select the Samples tab and enter a Bounding box and/or date range for the filter, if required.

  • Select the Options tab and tick or untick the available options, if required:

    • Include all observations that match filter - this option is ticked by default.

    • Include all locations that match the filter's bounding box and survey criteria.

    • Include all individuals and organisations.

  • Choose [OK] in the right-hand panel of the window to update the database with the new filter.

  • You can choose [Cancel] in the right-hand panel to abandon any changes. No changes will be made to the database, and the window will be restored to its original state.

  • Choose [Close] to close the dialogue box.

Note: if you want to include observations for all surveys, survey tags, taxa, areas or dates, simply leave the appropriate box blank. The default is to include all and, apart from the name of the filter, no other fields are mandatory.
If, for example, you want to export names and addresses to other Recorder users, without the associated observations, you could create a filter with Include all observations that match filter unticked and with Include all individuals and organisations ticked, and use it to do an export.

Having created a filter, you could create a Recording scheme that uses it. When you run this Recorder will do an export in accordance with your filter, zip it and e-mail it to the specified e-mail address.

To export observations using an export filter:

  • Select Tools - Export Management - Export Using Filter - <filter name> from the Main menu

  • Select an Export Type from the drop-down box. See Export for more information.

Note: NBN Access Database (zipped) is the recommended Export Type for exchanging data between copies of Recorder, in most circumstances.

  • Specify a file name for the export and where it is to be stored in the Destination box. See Export for more information.

  • Select a date for Export records changed since or select Export all records for a filter.

  • Ticking Reassign Data Custody enables you to transfer ownership of your data to another Recorder system.

NOTE: You will not be able to edit data if you transfer ownership/custody of it to another Recorder system. See Transfer of ownership for details.

  • Choose [OK] and wait for the export to complete.

  • The resulting export file can then be sent to the data recipient for Import into their system.

Sharing Export Filters

To make an export filter available for someone else to use:

  • Select Tools - Export Management - Manage Export Filters from the Main menu.

  • Select the required filter.

  • Choose [Save].

  • Specify a file name for the export and where it is to be stored.

  • Choose [Save].

The export filter file, which will have the file extension .exf, is very small and can easily be e-mailed to the recipient.

To use an export filter provided by someone else:

  • Save the export filter file to somewhere on your hard drive.

  • Select Tools - Export Management - Manage Export Filters from the Main menu.

  • Choose [Open] and browse to the export filter file.

  • Highlight the file and press [Open].

  • Choose [Save].

The export filter will now be part of your system and can be used in the same way as any you have created yourself.
