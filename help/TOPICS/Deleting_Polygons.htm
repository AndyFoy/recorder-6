Deleting polygons

  • Select the pointer tool and the cursor will change to .

  • Click on the polygon or line you wish to delete. It will change colour (to the Selected Colour) to indicate that it is selected.

    • Select Map – Delete polygon from the Main menu, or

    • Click the button on the active toolbar, or
    • Right-click and select Delete Polygon from the resulting menu.

  • You will be prompted to confirm that you really want to delete the selected polygon.

NOTE: If the line or polygon is associated with a location, the linking information will also be deleted from the database (including the version and date of this boundary or transect).
