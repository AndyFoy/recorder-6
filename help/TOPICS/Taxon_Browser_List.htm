Taxon Browser List

An example of the list view is shown below:

In this example the current checklist is ‘Recorder 1998’.

The list view shows the items from the tree view in a two-column list. The left-hand list shows the Latin names, and the right hand list shows the Common Names (if there are any on the checklist).

TIP: To show all the names in a particular group, go to the hierarchy view, highlight the name of the group (e.g. the family name) and press the asterisk key on the numeric key pad.

To sort the items in either column:

  • Click on the column header once to sort alphabetically (i.e. A..Z).

  • Click again to reverse the order (i.e. Z..A).

NOTE: The items shown in the list view are only those items that have been displayed in the hierarchy view. For example, in the above picture, we can see that there are a number of species of the genus Emberiza displayed. These have only been displayed because the user opened up the genus node in the hierarchy to display all of the species.

TIP: Selecting an item in the list view will move the selection in the hierarchy to the same item.
