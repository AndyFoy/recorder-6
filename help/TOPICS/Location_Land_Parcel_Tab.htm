Location Land Parcel Tab

The Location Land Parcels tab holds information on any land parcels associated with the location. This tab allows the user to add or delete any number of land parcels.

Land parcels are numbers shown on the largest scale Ordnance Survey maps which identify each piece of land. An area of land (such as a Nature Reserve) can, therefore, be described in detail by listing the parcel numbers (and the map sheet from which they were taken) of the plots of land comprising the reserve. This has now largely been replaced by the use of GIS to directly store the boundaries of an area of land, but is retained in Recorder to deal with older data which still uses the land parcel system.

Number

A list of parcel numbers

Map system

The identity of the map sheet showing the parcel number.

It is not necessary to enter any data, but if you make an entry in a row, values are required in both columns.

To add a land parcel

  • Click on the [Add] button.

  • Enter a valid number value and map system.

To delete a land parcel

  • Highlight the item you want to delete.

  • Click on the [Delete] button.

  • You will be asked for confirmation.
