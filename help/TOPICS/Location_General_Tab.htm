Location General Tab

The General Tab of the Locations screen holds general information about the location.

This tab allows the user to add and delete names for the location. The only required information for a location is on this tab.

Name(s)

Required.

A location can have any number of names, and all of the names are displayed in this box. On the left of each name is a check box which is ticked for the preferred name. The preferred name is the actual name that is used throughout Recorder , for example in reporting.

To add a location:

  • Click on the [Add] button.

  • The Location Names dialog displays.

  • Type in a name for the location, and choose [OK].

  • The location name is added to the Name(s) box.

To delete a name for a location:

  • Highlight the name you want to delete by clicking on the name in the Name(s) box.

  • Click on the [Delete] button.

  • The name is deleted from the box.

To edit a name for a location:

  • Highlight the name you want to edit.

  • Click on the [Edit] button.

  • The Location names dialog displays.

  • Edit the name for the location, and choose [OK].

File Code

Optional. Any user-specific file code can be entered here.

Type

Required.

The type of location. Select one of the values from the drop-down box. The location type is a ‘Term-list’, and items in this list can be added or deleted by clicking Options - Term-lists.

Central Spatial Ref

Required.

The spatial reference of the site centroid.

To enter a spatial reference:

  • Type in a valid spatial reference, and press <Enter>.

  • Obtain a spatial reference from the map - see Getting grid references or locations from maps.

Spatial qualifier

Required.

See Spatial reference qualifiers.

Select a value from the drop-down box.

If the user has entered the spatial reference by using the Map browser, the value will be ‘Internal Map’ by default.

Description

Optional. Enter a description for the location. The format of the description box is rich text.
