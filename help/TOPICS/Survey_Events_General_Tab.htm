Survey Events General Tab

This page records the locality, date and surveyors who took part in the survey event.

 

Location

Optional. This is a reference to an item in the Locations window. To enter a location, use one of the methods described in Finding things.

If a location is entered, the spatial reference value is completed with the location's central spatial reference by default. However, this default value can be edited if an exact spatial reference is available.

Entering an exact spatial reference in the Spatial Ref field will not effect the location.

An entry in at least one of the Location, Location name or Spatial Reference fields is REQUIRED, i.e. some form of locality must be given.

See Location and Location Name for more information on the distinction between the two.

Location Name

Optional. A name for the survey event location which is stored as text in the Survey event record rather than a reference to an item in the Locations window. If you enter a Location, this can be used to provide additional details - for example, describing exactly where the event took place within a large site.

Spatial Ref

Required, unless Location Name is the only location information.

Enter a spatial reference. If this box was empty and you selected a location, then it will default to the centroid for that location. You can edit this reference manually, or you can obtain a spatial reference from the map. To do this:

  • Click the [Get] button to display the Map browser window.

  • Select the pointer tool and click on the desired position on the map.
  • Return the grid reference by:

    • Selecting Edit – Return data from the Main menu.

    • Clicking the button on the active toolbar.
    • Pressing the short-cut key <F9>.

  • If the point at which you clicked was within a boundary which is linked to a location, the name associated with that boundary will be returned in the location box (see Linking polygons to locations).

Spatial Ref qualifier

Required, if Spatial Ref is entered.

See Spatial references.

Describes how the spatial reference was obtained. Select a value from the drop-down box.

The value of the spatial reference qualifier is entered automatically in the following situations:

  • If you enter the spatial reference by using the Map browser, the value will be ‘Internal Map’.

  • If you select a location from the location hierarchy, the spatial reference will default to the centroid for that location and the qualifier will be ‘Site Centroid’.

Date

Required. Enter a valid vague date for the survey event.

The date, or date range, during which the survey event took place. 'Unknown' is an acceptable value. The date given here must be on or after the start date of the survey to which this event is attached, and on or before the end date (if one was given).

Weather

Free text. Describe the conditions during the survey event.

Comment

RTF memo. Any comment. This can be used to describe the locality in greater detail.

Recorder(s)

A list of names, from the Names and Addresses window, of those who took part in the survey event with an indication of their role. At least one entry is REQUIRED.

Note: when editing a date, location, location name or spatial reference in a survey event the change can be cascaded through the samples, as described in Validating data, Editing dates and locations.
