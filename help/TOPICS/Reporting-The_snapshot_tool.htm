Reporting-The snapshot tool

The snapshot tool exports data from the complex Recorder database in a simplified database structure. This simple database can be used for bespoke reporting or analysis or used with GIS packages or other software. Once you have created the snapshot, it can be refreshed by running the report again.

  • The first step is to create a report with the data that you want through the report wizard.

Note: Be sure to select "New layout" from the layout selection page.

Note: Make sure that you select all of the attributes that you want to see in your snapshot

  • Click on ' New Snapshot' from the button

  • You will be presented with the first snapshot screen.

  • This screen will help you to;

    • Select the place where the snapshot will be created

    • Provides SQL Server with the permissions by which you can create or alter that database

    • Allows you to define the rules that govern how the snapshot is created

Snapshots are mdf files that can be associated with a piece of software such as ArcGIS or Mapinfo using a standard ODBC connection.
