Export a Rucksack

When the Rucksack window is active, you can select Tools - Export data... from the menu to produce an export file in NBN transfer format (XML) or NBN Access Database format (ZIP) containing all the items in the Rucksack lists other than the Taxon and Biotope lists (these contain "system supplied" items which are not normally exported because the recipient should already have them).

This is particularly useful facility for a Recording Scheme Organised, or a Local Record Centre, who works with volunteers. They can build a rucksack for a particular Survey containing the Locations, Names and Addresses and Documents relating to that Survey and then produce an export file which they can send to participants. The volunteer can import the file into their copy of Recorder and they then have all the reference information they need for the Survey.

Tip: In the Export dialog box, you will probably want to ensure that the "Include observations" tick-box is not ticked. This will result in a file that just contains the Location, Names & Address and Document records, but not the actual observations you hold which are associated with them. You will probably want to leave the "Include subsites for exported Locations" ticked so that the file does include any subsites of the Locations you have chosen.
