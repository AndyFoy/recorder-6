My Rucksack

The Rucksack is a handy container in which you can keep frequently used items. It consists of a series of lists in which you can keep Taxon names, Biotope classes, Locations, Names & Addresses and References. You can save a rucksack to disk, and quickly switch from one to another. When you are entering data, or selecting items on which to report, the rucksack provides a handy source from which frequently used items can be obtained.

For example, suppose you were entering data for a particular survey. You could set up a rucksack containing the names of the most frequently recorded species, the locations which were surveyed and the people who did the sampling. If you have this rucksack open while you fill in record cards, most of the data items you need can be dragged from the rucksack into recording cards without having to search through all the items in the database. See adding, deleting and ordering items.

Rucksacks are stored in plain text files, consisting of lists of record keys, so they can easily be shared between people who are working on the same type of data. See saving & loading rucksacks;

The Taxon lists in rucksack files can be used as the source of species lists upon which to base recording cards. See setting up a new recording card.

Taxon and biotope lists from a rucksack file can also be used to select the items on which to report. See Report Wizard biotope and taxon selection.

You can export a Rucksack to produce a file in NBN transfer format (XML) containing a collection of Locations, Names and addresses and Documents.
