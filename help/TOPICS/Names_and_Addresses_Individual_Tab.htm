Names and Addresses Individual Tab

This page allows the name of an individual to be entered, and summarises their preferred address and contact number.

 

Title

Free text with suggested values available from list. Commonly used titles can be picked from the list.

Forenames

Free text. Type one or more forenames.

Initials

Free text. Type the person's initials. It is suggested that these are separated by '.'

Surname

Required.

Free text. Type the person's surname.

The Address panel is Read-only, and is automatically generated from the preferred address and contact numbers. It is shown for quick reference to the person's name and address.
