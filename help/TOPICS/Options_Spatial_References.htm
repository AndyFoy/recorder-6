Options Spatial References

This page of the Options window allows you to choose which spatial reference system you will work in. The standard options are:

  • OS National Grid for Great Britain.

  • OS National Grid for Ireland.

  • Latitude and Longitude co-ordinates.

  • Universal Transverse Mercator co-ordinates.

Other spatial references systems can potentially be added, and will appear in this list (see Add-Ons).

  • Only one system can be selected.

  • The selection affects the format in which spatial references are shown on screen.

  • If Display grid references as squares on map is not ticked distribution points on maps will appear as dots. If it is ticked they will appear as squares covering the area represented by the grid reference, e.g. an observation with a grid reference of SU1234 will display as a square covering an area 1 km x 1 km. The squares will be partially transparent so that you can see through to the underlying map and also to any other, more precise grid references. This only applies to the OS - GB and OS - Ireland spatial reference systems.

See also: spatial references, spatial reference validation.

NOTE: The settings made here do not affect the co-ordinate system used to display maps.
