Import Wizard

Increasingly, biodiversity observations are stored directly in electronic form by the original recorder. In particular, spreadsheets are very popular amongst volunteers for storing their own results. Recorder has good facilities for importing this sort of data providing it is formatted in a way Recorder can deal with. To import data of this type you use the Import Wizard which has the following features:

  • Loads simple files of data in various formats where each row represents one observation to be loaded into Recorder.

  • If all the data comes from one location, or was recorded and/or determined by the same person, it does not require this fixed data to be present in the data file. It can be specified via the Wizard. The simplest data file is, therefore, a list of species names.

  • Handles all the necessary types of data required for entering observations:

    • Dates, including all the vague date formats supported by Recorder

    • Spatial references including all formats supported by Recorder (e.g. GB grid refs, Irish grid refs)
    • Matching the names of sites to the locations in the Locations hierarchy in Recorder
    • Matching the names of species to the Taxon Dictionary in Recorder
    • Matching the names of habitats to the Biotope Dictionary in Recorder
    • Matching people's names to entries in the Name and Address list in Recorder
    • Matching references to entries in the Document list in Recorder
    • Matching abundance qualifiers (e.g. the 'male' in '3 males') to the Measurement Qualifiers in the Measurement Type term list in Recorder
    • Matching record types (e.g. voucher specimen) to the Record Type term list in Recorder
    • Matching other data to the appropriate term list in Recorder

  • Makes new entries in your copy of Recorder, if required, e.g. if an observer is not in your Name and Address list you will be given an opportunity to make a new entry for them.

  • Performs all validation checks required by Recorder and reports errors. Items like grid references and dates can be edited directly from the Wizard to correct such errors.

  • Remembers matches for use in subsequent imports.

  • The settings can be saved as a template that can be reused the next time you load data from a similar source.

To start the Import Wizard, select Tools – Import Data from the Main menu.

You will be presented with the standard windows Data Source selection box from which you:

  • Locate the folder containing the file to be imported using the 'Look in' box.

  • Select the file type you require from the list. Choose one of the file types available other than NBN Data (*.xml) or NBN Access Database (*.zip) as these are used for importing data from another copy of Recorder.

  • Select the file that you want to import.

  • Click the [Open] button.

You will then be guided through the stages of the wizard:

  • Template selection

  • Specifying the types of data in each column

  • Entering data not in the import file

  • Matching data in the import file to entries in Recorder

  • Analysis and Import
