Installing on a single machine

Installing Recorder on a single, standalone computer (i.e. one not attached to a network).

Recorder 6 is a Microsoft SQL Server system so users need a version of SQL Server to run it. It is supported when used with:

  • SQL Server 2000, or its free run-time version, MSDE, Microsoft SQL Server 2000 Desktop Engine.

  • SQL Server 2005, or its free run-time version, SQL Server 2005 Express.
  • SQL Server 2008, or its free run-time version, SQL Server 2008 Express.

If you already have SQL Server installed, you will be able to install Recorder 6 using your existing SQL Server instance. If not, you will be able to download and install SQL Server Express from a Microsoft web-site then install Recorder 6. If you are in any doubt about which version to use, please request advice from the supplier of your copy of Recorder, or via the forum, forums.nbn.org.uk, as to which version is most appropriate for you. Prior to version 6.14 the installation CDs had MSDE on them. The version 6.14 and 6.15 disks had SQL Server 2005 Express on them but later versions do not contain SQL Server because different versions are available for 32-bit and 64-bit computers and because Microsoft upgrades may necessitate using the latest version of SQL Server Express for installing Recorder 6.

There are standalone and network installation guides in the Documentation folder on all the Recorder 6 installation DVDs and on the Recorder web site, www.jncc.gov.uk/recorder. They have not been updated for some time but they still contain much useful information. Please refer to these for information about the system requirements and installing Recorder 6, or request updated information via the forum, forums.nbn.org.uk. Once SQL Server Express is installed, a standalone installation basically involves inserting the Recorder DVD in your DVD drive and following the on-screen instructions.

The installation programme should start automatically. If it doesn't:

  • Use Windows explorer to locate the Setup.exe file on the installation DVD (it is in the root folder)

  • Either double click on Setup.exe or right click and choose "Run as".

If you are installing Recorder 6 on the same computer as your Recorder 2002 system, the installation process will pick up your Site ID from Recorder 2002. Otherwise, you will be asked to enter your Site ID and verification key. You will find these on the licence agreement you received with the installation DVD.

The standalone installation guide also contains information on transferring data from Recorder 2002 to Recorder 6, although more detailed information is available on the Recorder web site, www.jncc.gov.uk/recorder. It also contains information on SQL Server permissions, backing up the database, uninstalling Recorder 6, the installation folders and the registry settings.

The network installation guide contains much information on SQL Server, including:

  • Comparisons of the different versions of SQL Server

  • Migrating from MSDE to SQL Server
  • System requirements for network installations
  • SQL Server permissions, security and login options

Note: The standalone installation DVD will not install Recorder 6 on a network. There is a separate DVD for networked installations.

Note: Recorder 6 version 6.17 is officially supported for use with Microsoft Vista and Windows 7 but the installation guides have not yet been updated. Until they are, for more information, please consult the forum, forums.nbn.org.uk, or contact the supplier of your copy of Recorder, or any of the Recorder suppliers who are listed on the Recorder web site, www.jncc.gov.uk/recorder.
