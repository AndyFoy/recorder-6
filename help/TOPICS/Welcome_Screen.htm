Welcome / Summary screen

The Welcome dialog is displayed in two circumstances.

  • You open it by selecting Help – Summary from the Main menu.

  • Application start-up, unless the option to 'Open with windows from the last session' is selected (see Option - Window), and you closed the application with some windows still open.

The screen presents some information about your copy of Recorder. A typical example is shown below.

 

User name

The user who is currently logged on.

Copy identifier

The registered number for this copy of Recorder.

Access level

The current user's access rights.

Summary statistics

A summary of the number of records in each of the main tables in the database.

Tip

Tip of the day. Click on the 'Tip' hyperlink to view another tip.

Show Welcome Window at Startup

This tick-box controls whether or not this window will appear when Recorder 2002 starts. Should you change your mind you can turn it back on again later via Tool - Options.

The dialog also includes [Quick start] buttons by which some of the most frequently used facilities can be immediately accessed:

Species list

Rapidly enter a list of taxa and/or a biotope observed in a particular sample.

Location

View, add and edit details about locations.

Report

Start the Report Wizard to select observations which can be printed, mapped or exported in various formats.

Choose the [OK] button, or press <Enter> to close this dialog.
