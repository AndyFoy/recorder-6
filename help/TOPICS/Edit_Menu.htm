Edit Menu

Add

Add a new item to the database. There may be a submenu, if there is a choice of item types which could be added.

Edit

Edit the currently selected item.

Delete

Delete the currently selected item. You will be prompted for confirmation before the item is actually deleted.

Cut

Only active during editing when text is highlighted within an prompt or RTF memo. Removes the highlighted text to the Windows clipboard.

Copy

Copy the currently selected item or text to the Windows clipboard. The selected text is not removed.

Paste

Paste material in Windows clipboard to current cursor position. Only active if there are relevant data in the Windows clipboard.

Return data

When the button is used to open another window, and an item has been selected, then this command returns the selected item to the original, calling window.

Find

Opens the Find dialog to search for an item of data by name.

Sort by

Sorts the items in the currently active list or hierarchy. There may be a sub-menu if there is a choice of ways by which items could be sorted. e.g. in the Taxon browser, items could be sorted by common name, scientific name, etc.

Simple filter

Applies a single selection criterion to the items listed in the current display. e.g. in the Locations window, you could select locations which were designated as 'SSSI' or whose names contained 'wood'.

Find On Map

Available in the Location and Observation windows. This command opens the Map Browser window and centers the map on the Location or spatial reference of the currently selected item.

Show Metadata

Displays a window showing meta-data about the currently selected item such as the persona who created it and the date on which it was last changed.

Bold

Only available while editing a memo field. Turns emboldening on and off.

Italic

Only available while editing a memo field. Turns italicising on and off.

Underline

Only available while editing a memo field. Turns underlining on and off.

This menu changes when the Report Designer is active. See View Menu.
