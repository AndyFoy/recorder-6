Selection by Name

You will see this screen if you have selected the option to select a place by name.

 

Find place

This works like the Find dialog. Type a location name or part of a name, and all matching names are shown in the 'Available' list. If you type part of a name (e.g. 'wood') then the 'Available' list will contain all names in the selected checklist or rucksack starting with what you typed. If you prefix it with "*" (e.g. '*wood'), then all names containing this string will be included.

Include Subsites

This tick-box specifies whether or not to automatically include in the report any sub-sites of the Locations in the 'Available' list. The option is on (i.e. the box is ticked) by default.

Move the locations whose associated observations you want in the report into the 'Selected' list. See Add and Remove items from selected list.

You can add one or more names to the 'Selected' list and then make changes in the 'Find' part of the screen. This will result in a different 'Available' list from which you can make additional selections.
