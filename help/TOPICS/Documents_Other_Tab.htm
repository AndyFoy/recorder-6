Number

Free text

ISBN, ISSN or other number identifying a publication.

Type

Free text with suggested values available from list

Type of number (e.g. ISBN, ISSN). You can select the common values from the list, but any text can be typed here.

These fields are NOT required, but if you choose to make an entry here, entries must be made in both fields.

To enter a value:

  • Press the button to add a new, blank line to the grid.

  • Edit the values directly in the grid.

To remove a value:

  • Highlight the row you wish to remove and choose the button.

  • You will be prompted to confirm that you really want to remove the selected number.

Original file

A file name including the directory

If the document is present on your computer, e.g. in a word-processor file, you can put a reference to the file here.

To add a file name:

  • Press the button, and a standard Windows file location dialog will appear. Select the relevant file and choose [OK] to close the dialog. The selected file name will appear in the box.

  • You can also type the path and name of the file directly into the box.

  • Press the button to view the file (see Sources).
