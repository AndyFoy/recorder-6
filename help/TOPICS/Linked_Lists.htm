Linked lists

There are two pairs of term lists which are linked:

  • Measurement types with measurement units.

  • Feature types with feature gradings.

In these two cases, you will see a hierarchy showing measurement or feature types as the top level, and units or gradings nested within the types.

In these two cases, when you 'Add' a term, you will see a submenu from which you can select.

Sibling

Add an item at the same level in the hierarchy:

  • If a measurement is selected, this will add a new measurement.

  • If a unit is selected, this will add another unit to the same measurement type.

Child

Add an item at the lower level of the hierarchy:

  • If a measurement is selected, this will add a unit for that measurement type.

  • If a unit is selected, this option is not available (it will be greyed).

Otherwise, terms are added in exactly the same way as for all other term lists.
