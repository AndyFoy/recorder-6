Help Menu

Help

Opens on-line help at the contents page.

Context sensitive

Opens on-line help at the page relevant to the current cursor position.

Summary

Opens the summary 'Welcome' screen. This displays information about the copy number of the application and the user who is currently logged on. It also shows summary statistics for the contents of the database.

About

Displays the About dialog which shows the version number of Recorder 6 itself, the dictionary version and system information. If you need to contact support services, they will need this type of version information. The dictionary version is the file name of the last script used to update the dictionary.

The More tab displays information about other components of the system, including the version numbers of the add-ins currently installed.
