View Menu

This menu is only available when the Report Designer is in use. It controls the appearance of the Report Designer window.

Toolbar

If this is ticked, the Report Designer's main toolbar is shown.

Font toolbar

If this is ticked, a second toolbar is displayed which includes buttons to set the alignment, colour and borders of the selected item.

Element toolbar

If this is ticked, a vertical toolbar is shown down the left-hand side of the Report Designer window containing buttons to add new elements to the report.

Status line

If this is ticked, the Report Designer has a status line.

Data fields

If this is ticked, the items you selected in the Select Attributes page of the Report Wizard are shown in a list which floats over the Report Designer window..

Zoom

Default size returns you to the default, Increase and Decrease size allow you to zoom in and out.

Rotate bands

This arranges the bands in the order you added them to the report.

Hide bands

Hides the bands (allowing you to access elements on the report background).

Reset bands

This arranges the bands in the order they will appear on the paper.

Options

Gives access to some options which control the overall functions of the report designer.

While the Report Designer is active, there are changes to some other menus as follows:

Edit

New

Add new elements to the report.

Undo

Undo the last action performed in the Report Designer.

Align

Select two or more elements. This command allows the second and subsequent ones to be aligned relative to the first one which was selected.

Centre elements

Select two or more elements. This command centres the second and subsequent ones relative to the first one which was selected.

Centre on band

Centres the selected element either vertically or horizontally within its band.

Send to back

Sends the selected element to the back, so that other elements will hide it if they overlap.

Send to front

Send the selected element to the front, so that it hides any other elements it overlaps.

Report

Options

Sets overall report options such as the page size and orientation, column layout and default font for the whole report.
