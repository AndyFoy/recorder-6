Taxon Occurrence Related Occurrences

This page is used to create relationships between occurrences in order to record interactions between taxa, e.g. predator-prey, parasite-host, etc.

NOTE: In order to relate two taxon occurrences, it is necessary to create both occurrences first, and then to create a relationship between them.

Thus, if you wanted to record a caterpillar feeding on a particular food-plant, you must create an occurrence record of both the caterpillar and the plant, and then relate them.

 

Related occurrence

Required.

This is a reference to another taxon occurrence record. It can be filled in by:

  • dragging and dropping another taxon occurrence from the observation tree to the left into this box

  • typing a part of a taxon occurrence and pressing <Enter>. The Find taxon occurrence dialog will open, allowing you showing matching occurrences. Select the desired occurrence and click [OK], or press <Enter> to fill in the box.

Relationship type

Required.

Item selected from the relationship type term list. Users can add terms to this list if necessary.

Comments

RTF Memo. Can be used to provide more details about the relationship.

To add a related occurrence:

  • Click on the button, and the detail panel will go into Edit mode with all the fields empty.

  • Type in the required information.

    • Click the button to save your changes.

    • Click the button to abandon your changes.

To delete a related occurrence:

  • Highlight the related occurrence that you want to delete.

  • Click the button.

  • You will be asked to confirm that you really want to delete this record.

To edit a related occurrence:

  • Highlight the related occurrence that you want to edit.

  • Click the button to put the detail panel into Edit mode.

  • Make the changes you require.

    • Click the button to save your changes.

    • Click the button to abandon your changes.
