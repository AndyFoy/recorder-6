Options Main Toolbar

If you choose to display the main toolbar (see: Options – General option), then this screen allows you to configure which commands it contains.

Show main toolbar

If selected, the main toolbar will appear. You can choose which commands you would like on this toolbar using Options – Main toolbar. All commands are available via the Main menu.

Show active window toolbar

If selected, a toolbar will appear containing commands relevant to the currently active window. All such commands are also available via the menu system.

  • The list in the left panel shows all the commands which are available to place on the toolbar.

  • The list in the right panel shows the currently selected commands which appear on the toolbar.

TIP: Installing Add-Ons may add new commands and icons to the 'Available action' list.

To add commands to the toolbar:

  • Select an item in the left-hand list by clicking on it with your mouse.

  • Click the button.

OR

  • Double click an item in the left-hand list to add it.

To remove commands from the toolbar:

  • Select an item in the right-hand list by clicking on it with your mouse.

  • Click the button.

OR

  • Double-click an item in the right-hand list to remove it.

OR

  • Click the button and all items will be removed from the right hand list.

To change the order of buttons on the toolbar:

  • Select an item in the right-hand list by clicking on it with your mouse.

  • Click the or buttons to move the selected item up or down the list. Repeated clicks on one of these buttons will keep moving the selected item until it reaches the top or bottom of the list.

If you choose [OK] to close the Options screen, the toolbar will be updated immediately.
