Map Options

This facility is used to set up and manage the maps you want to use in Recorder. It is accessed from Map - Map Options on the Main menu. You will need to set up at least one map using the Enabled Maps tab before you can start using Recorder's mapping facilities.

The Enabled Maps tab is used to:

  • Set up (Add) your first map so that you can start using the mapping facilities.

  • Set up (Add) additional maps and choose which map should be the default. Within Recorder it is possible to create and use entirely separate maps, with their own base map, background layers, polygon layers, display options etc. This is useful for people who want to use the maps in different ways, although for many users a single map will suffice.

  • Reset a map.

  • Delete a map.

The Map Layers tab is used to manage the map layers on a selected map.

For adding, deleting and changing map layers see Map Options - Map layers

Note: These functions can also be performed by right clicking in the Map Layers box in the open Map Window, as can exporting polygon layer(s).

The Distribution Symbols tab is used to:

  • Rename or delete distribution symbols by right clicking a selection.

  • Change the colour of distribution symbols by right clicking a selection and selecting Properties.

  • Turn the distribution symbols on and off by ticking and unticking them.

Note: These functions can also be performed by right clicking in the Distribution Points box in the open Map Window.

  • Change the cut off year for hollow/solid symbols.

This facility can be used whether or not the map browser is open. If the map browser is open, then changes you make will be shown on the map immediately.
