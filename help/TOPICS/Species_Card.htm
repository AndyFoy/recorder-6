Species Card

The species card screen allows the user to enter a single observation or record for a location. The same data can be entered using the Observations window, but when using the species record card, the survey event and sample are created for you with the observations; you don't need to enter each item separately. In addition, when an observation is entered, the data for the sample are preserved on the card, so that another observation with some data in common can be entered quickly. If you need to enter more than one observation for a location, use a Recording Card instead.

To enter a new species record:

  • Select Data Entry – Enter a species record.

  • The species record card is displayed.

  • Enter the values for the new species record (see the table below).

  • Click the [Save] button.

The information on a species card is stored as a new sample for the selected survey. Where the sample is stored depends upon the location, date and recorder for the species card. If a survey event already exists with the same location, date and recorder as the species card, then the card is added as a new sample for that event. However, if no matching event exists, a new survey event is created, and the card added as a new sample for this event.

Survey

Required.

Select a survey from the drop-down box.

Taxon

Required.

To enter a taxon, use one of the methods described in Find overview.

Recorder(s)

Required.

To enter a recorder(s) either:

  • Use one of the methods described in Find overview.

  • Click the [Add] button and the Find Individual dialog box is displayed. See Find dialog.

To delete a recorder:

  • Highlight the individual that you want to delete by clicking on the name.

  • Click the [Delete] button.

Document

Optional. To enter a document, use one of the methods described in Find overview.

Location

Optional. This is a reference to an item in the Locations window. To enter a location, use one of the methods described in Finding things.

If a location is entered, the spatial reference value is completed with the location's central spatial reference by default. However, this default value can be edited if an exact spatial reference is available.

Entering an exact spatial reference in the Spatial Ref field will not effect the location.

An entry in at least one of the Location, Location name or Spatial Reference fields is REQUIRED, i.e. some form of locality must be given.

See Location and Location Name for more information on the distinction between the two.

Location Name

Optional. A name for the sampling location which is stored as text in the Survey event and Sample records rather than a reference to an item in the Locations window. If you enter a Location, this field can be used to provide additional details - for example, describing exactly where the observation was made within a large site. It is intended to enable a more detailed, textual description of the location of the observation to be entered.

Date

Required. Enter a valid vague date for the observation.

The date, or date range, during which the sampling took place. 'Unknown' is an acceptable value. The date given here must be on or after the start date of the survey to which this observation is attached, and on or before the end date (if one was given).

Spatial Ref

Required, unless Location Name is the only location information.

Enter a spatial reference. If this box was empty and you selected a location, then it will default to the centroid for that location. You can edit this reference manually, or you can obtain a spatial reference from the map - see Getting grid references or locations from maps. If the point at which you clicked was within a boundary which is linked to a location, the name associated with that boundary will be returned in the location box - see Linking polygons to locations.

Spatial Ref qualifier

Required, if Spatial Ref is entered.

See Spatial references.

Describes how the spatial reference was obtained. Select a value from the drop-down box.

The value of the spatial reference qualifier is entered automatically in the following situations:

  • If you enter the spatial reference by using the Map browser, the value will be ‘Internal Map’.

  • If you select a location from the location hierarchy, the spatial reference will default to the centroid for that location and the qualifier will be ‘Site Centroid’.

Admin Area(s)

Read-only. The information for admin area is displayed if there are associated admin areas for the location. To associate admin areas with a location, see Location Admin Areas.

Sample Type

Required.

Select a sample type from the drop-down box. The sample type is a ‘term-list’, and items in this list can be added or deleted by clicking Tools - Term-lists. The icon for the sample type is used for the sample in the Observations screen.

Biotope

Optional. To enter a biotope, use one of the methods described in Find overview.

Count

Optional. The count list box allows the user to enter count information for the observations.

To enter count information:

  • Click on the [Add] button .

  • Type in the Count value (i.e. a number).

  • Click in the Count Of box and use the drop-down arrow to select what has been counted (i.e. a qualifier such as Juvenile).

  • Select a value for the accuracy using the drop-down arrow (or type a user-defined accuracy).

To delete count information:

  • Select the count information that you want to delete.

  • Click the [Delete] button .

Provenance

Optional. Select a value for the provenance for the species from the drop-down box.

Record Type

Optional. Select a value for the record type from the drop-down box.

Substrate

Optional. Select a value for the substrate from the drop-down box.

Determiner

Optional. To enter a determiner use one of the methods described in Find overview.

Date of Determination

Optional. If a determiner is entered, a valid determination date must be entered. This must be on or after the observation date.

Specimen Number

Optional, unless any other specimen details are entered.
Free text. An identifier for the material that was retained, so that it can be located in future.

Specimen Type

Optional, unless any other specimen details are entered.
Item selected from the specimen type term list. Users can add terms to this list if necessary.

Specimen Location

Optional. Free text. An indication of where the material is kept.

Specimen Comment

Optional. RTF Memo. Any text describing the material and its condition.

Comments

Optional. Free text field for a comments about the sample.

To Reset:

The [Reset] button is used to clear some of the controls in the species card. It allows the user to abandon entering information about a particular observation while preserving values from the previous sample. The following controls are reset: count list box, provenance, record type, substrate, determiner, date of determination, specimen number, type and comment, and comments.
