Source Selection

You will see this screen if you selected the option to 'Restrict report to one or more sources' when selecting the style of report. This screen allows you to select the particular source(s). It will differ in detail, depending upon whether you chose to restrict observations to those from documents, individuals / organisations, surveys or survey tags.

To select documents, individuals / organisations or survey tags:

  • The Find box at the top allows you to type part of the name you are looking for (author’s name or title for a document, the name of an individual / organisation or letters from a survey tag). It works like the Find dialog. If you type something, e.g. 'ron', then any item starting with 'ron' will appear in the list. If you prefix it with "*" (e.g. '*ron') then any item containing the letters 'ron' (Ronald, Baron) will appear in the list.

  • Searching starts as soon as you have typed 2 chatacters and the matches in the appropriate table are listed in the Available list.

  • Move the items whose associated observations you want in the report into the Selected list - see Add and Remove items.

To select surveys:

  • Select a survey type from the drop-down box at the top, and all surveys of that type will be listed in the Available list.

  • Move the items whose associated observations you want in the report into the Selected list - see Add and Remove items.

Note: you can add one or more items to the Selected list, and then make changes in the Find/Survey Type box. This will result in a different Available list from which you can make additional selections.
