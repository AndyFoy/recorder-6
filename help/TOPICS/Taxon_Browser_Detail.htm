Taxon Browser Detail

If the detail pane is showing (see Hiding or showing dictionary details) you will see a summary of information about the selected species:

The preferred name for the taxon is shown at the top of the panel, and any synonyms and common names are given underneath. Then, other checklists upon which this species appears are listed.

The other checklist names are hyperlinks and, if you click on one, the information available on that checklist will be shown. For example, clicking 'Wildlife and Countryside Act (Schedule 1)' in the example above would show the status of this species on that piece of legislation:

The 'Find information ... from the NBN' link will only work if you have an active connection to the Internet. In this case it will start your web-browser and query the NBN Gateway.

If any taxon facts are available, they will be shown. For example, a species account from the Insect Red Data Book and the Recorder 3 species account:

If any facts have been entered by the user (see New Taxa Facts), then they are also displayed.

Using information from the detail panel in other software

  • You can select all or part of the information displayed in the detail panel using your mouse and copy it to the Windows Clipboard by:

    • Selecting Edit - Copy from the Main menu.

    • Pressing the <Ctrl-C> short-cut key.

  • The information can then be pasted into other Windows software, such as a word-processor document.

Printing the detail panel

You can print the information shown in the detail panel by selecting File - Print from the Main menu.
