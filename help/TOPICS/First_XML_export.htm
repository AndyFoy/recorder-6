Your first NBN export

The first time you export in NBN transfer format you need to provide meta-data about your particular database. This information is REQUIRED by the NBN transfer format and forms part of the header information in every NBN transfer file. You only need to enter it once. It is stored in the database, and will be re-used for subsequent export files.

The first time you choose to export to the NBN format you will be presented with the following dialog box:

Dataset name

The name by which your dataset is identified. This should be a short piece of text suitable to appear in a summary list.

Dataset owner

The individual or organisation who owns this dataset.

Dataset restriction

A paragraph of text describing any restrictions on access to, or use of this dataset. For example "Unrestricted", "To be used only according to the terms of the following licence ..."

Dataset status

A paragraph of text indicating the status of the Dataset. For example, is it complete?, has it been published?.

Dataset title

The full title of the dataset. This can be longer than the name.

Dataset validation

A paragraph of text describing the quality control processes that have been applied to the dataset. What checks have been carried out and to what standards?

To enter the details of your dataset:

  • Select each of the six categories by clicking on it with your mouse. The symbol in the grid in the upper half of the dialog indicates which one is selected.

  • Click on the button. The "Exported information" box will be activated. If you have already entered information for the selected category it will be displayed and you can edit it. Otherwise the box will be empty - just type in the relevant details.

  • When you have completed your entry, click the button to save the information. The first few words will appear in the "Exported information" column of the grid to indicate that this entry now contains data. The button will cancel any changes and restore the entry to its state before you started editing.

  • Once you have made an entry in all six categories, the [Continue] and [Abort] buttons will become active. Choose the [Continue] button to save the details you entered, and proceed with the export operation.

  • If you close the dialog box without saving all the necessary details (by choosing [Abort] or by clicking the button at the top-right of the window), the export process will be cancelled. You will be prompted for this information again the next time you start an NBN export.

How the data is stored

These six fields are stored in a table called SPECIAL_XML_ELEMENT in the nbndata.mdb file. They have a type of "M" (for "meta-data") and keys which correspond to the six categories listed above. This dialog box will appear at the start of an NBN export if the "DATA" column for any of these six rows is empty.

Editing the metadata

You can edit this data by selecting Tools - Edit Metadata from the menus.
