Selection by Bounding Box

You will see this screen if you choose to select places or observations using spatial references.

 

NE and SW corners

Enter a pair of spatial references which define a rectangular area.

This button will open the Map Browser allowing you to specify the spatial references by drawing a rectangle on the map.

Locations are selected by testing whether the grid squares and centroid listed for the location lie within the rectangular area defined by the SW and NE corner.

For example, if a bounding box was defined as SW corner = NZ063151, NE corner = NZ283733, then a location with NZ16 (green hatched) listed as its Grid Square is entirely within the bounding box, and would always be selected. However, a location with NZ26 (red hatched) listed as its grid square is not entirely within the bounding box. In this case, the setting of the options 'Include places partially inside the bounding box' or 'Do not include places partially inside the bounding box' determines whether or not the location is selected. If the first option is selected, it will be included; if the second is selected, it will not.

NOTE: The selection of locations using a bounding box is made entirely using the grid squares and site centroid given in the location record, even if the location is associated with a site boundary shown on the map. Site boundaries are NOT used in this selection.
