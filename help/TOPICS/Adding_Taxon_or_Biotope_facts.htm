Adding Taxon or Biotope facts

These are added via Edit Taxon details- Facts or Edit Biotope details - Facts.

The text of the fact should be typed as an HTML fragment and can include a wide range of HTML tags, for example:

<I>...</I>

The enclosed words will be displayed in italics.

<B>...</B>

The enclosed words will be emboldened

<BR>

Inserts a line break

<P>

Starts a new paragraph

Other possibilities include bulleted or numbered lists and tables. In fact almost any syntax included in the HTML 3.2 specification can be utilised.

You can include special characters, including those with accents, by giving their character entity reference from the ISO 8859-1 (Latin-1) character set. This consists of either a mnemonic or a character number. For example the character é can be written as "&eacute;" or "&#233;".

Example

<I>Criorhina asilica</I> (Fall&#233;n, 1816) will be displayed in the Browser detail pane as Criorhina asilica (Fallén, 1816)

Including images

You can include a picture in the Fact simply by including an HTML IMG tag which points to the file containing the image (in GIF - including animated gifs, BMP, JPEG, PNG formats and including transparent images where these are supported by these file formats). For example "<IMG SRC="c:\temp\MyPicture.jpg">.

However, this may cause a problem if you exported the Fact to another computer. They might not keep their images in this directory. To help overcome this problem, you can use a special tag "#LOCAL_IMAGES" which stands for the path set in "Image files:" in the Tools- Options - File Locations page. The example would then become "<IMG SRC="#LOCAL_IMAGES\MyPicture.jpg">. When you export the Fact, providing you also send the image file and it is put in the path pointed at by "Image files:" in the target copy of Recorder 2002, then the image will appear correctly. This allows the target copy's owner to keep Image file where he or she chooses.

You could embed other files (a sound or a movie clip for example) in your Fact using the same principles.

Including a source for the Fact

If you put reference stored in the Documents table in Recorder 2002 in the Source prompt of the Edit Taxon details - Facts or Edit Biotope details - Facts pages, it will appear as a hyperlink n the Browser detail pane. Clicking this hyperlink will open the Document window and select this reference.
