Report Wizard Biotope Selection

You will see this screen if you have chosen the option to report on one or more biotopes.

 

Classification

Select a biotope classification from those available in the biotope dictionary. The list also includes an item 'Current Rucksack'. This allows you to select items from the biotope list in a rucksack.

If the Biotope list in the Rucksack includes items which have items nested underneath them in the classification hierarchy, you will also see an item 'Current Rucksack (Expanded)'. Choose this item to select items from the rucksack list including everything in the hierarchy below the listed items.

Tip: type "*" at the Find Taxa prompt to add all the items from the rucksack to the Available list.

Find biotope

This works like the Find dialog. Type a name, or part of a name, and all matching names are shown in the 'Available' list. If you type part of a name (e.g. 'wood') then the 'Available' list will contain all names in the selected checklist or rucksack starting with what you typed. If you prefix it with "*" (e.g. '*wood') then all names containing this string will be included.

Include All Biotopes

If this option is selected then all biotopes are included in the report. Therefore, selecting it disables the Classification, Find Biotope and Available and Selected list buttons.

Move the biotopes whose associated observations you require in the report into the 'Selected' list. See Add and Remove items from selected list.

You can add one or more names to the 'Selected' list, and then make changes in the 'Classification' and/or 'Find' part of the screen. This will result in a different 'Available' list from which you can make additional selections.

If you select a rucksack as a source of names, and the biotope names listed in the rucksack include items which have sub-categories, then the names of those sub-categories will automatically appear in the 'Available' list. You can then use the button to add all these names to the 'Selected' list in one go. This provides a quick way of making a report for whole branch of a classification hierarchy.

NOTE: You can include the names of biotopes at any level in the classification, in the 'Selected' list. This will select observations attached to the named biotope only, NOT observations of sub-categories nested within it in the classification. If you want to include observations from all its sub-categories as well, then you MUST include all their names in the 'Selected' list using the mechanisms described above.
