Printing maps

  • A map, as it is currently displayed on screen, can be printed by:

    • Selecting File – Print from the Main menu.

    • Clicking the button on the toolbar.

  • You will be prompted for a title for the map.

The printed map includes a legend containing the title, scale and a key to any distribution symbols which are currently plotted. Grid overlays, polygons and any background map details are also drawn.
