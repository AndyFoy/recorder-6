LogIn screen

The first time you log in to Recorder you will see:

  • Enter your first name and surname.

  • Enter a password, if you want to password protect your Recorder system.

  • Type your password again in the "Confirm Password" prompt. You will not be allowed to proceed unless the text matches your "Password". This protects you against minor typing errors, which could result in setting an incorrect password and finding you cannot log on to the application next time!

Note: If you leave the password boxes blank your password will be "password".

  • If you would like Recorder to create a survey for you tick "Automatically create the first survey" and enter a survey name, e.g. "Biological Records". All your observations (records) will be stored in surveys so this will save you having to enter a survey by the normal means before you can start entering observations.

Note: You will also see the window above if your system only contains the "Default User" username, which is in the database when Recorder is installed. This will happen if all the other usernames are deleted.

On subsequent log ins, if your username is the only one in the system apart from the "Default User" and your password is "password" you will go straight into Recorder without having to enter your username and password, as in the window below. Otherwise you will be asked to enter these details:

  • Use the drop-down arrow to select your username, if it is not already shown.

  • Type your password, if it isn't "password".

    • Click [OK] or press <Enter> to log on.

    • Click [Cancel] if you decide not to start the application.

Note: The password for the "Default User" is "password" unless you change it.

Note: The default password for a newly created username is always "password".

Note: When data is entered into Recorder or changed, the system will record who entered or changed it using the username of the person logged on so it is not advisable to enter or change data when logged on as the "Default User". If you want to prevent this, you can change the "Default User" to a Read Only user via Tools - User Configuration.

  • If you don't have your own username, then ask the System Manager to create one for you. The first time you use this new user name, the password will be "password". Once you have successfully logged on for the first time, go to Tools – Change password to set your own password. See Tools menu.

  • Passwords are case sensitive. If you set your password to "Fred", then "fred" or "FRED" will NOT be accepted.
