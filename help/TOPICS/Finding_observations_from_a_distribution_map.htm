Finding observations from a distribution map

Overview

Plotting a distribution map is a very good way of checking for unusual observations which are outside the normal range. These may, of course, be caused by errors in the spatial references given to Survey events and Samples making this one of the tools you might use for data cleaning. However, it is then necessary to locate the original observation(s) that gave rise to the distribution symbol in question in order to investigate it further. This facility provides a means of going straight to the observations which gave rise to a distribution symbol.

To find observations from a distribution map symbol:

  • When the Map Browser is showing a distribution map, choose the Symbol selection tool:

    • Select Map - Find Source Data from the menus
    • Click the button in the toolbar

  • The cursor will change to

  • Click on a distribution map symbol (i.e. a filled or hollow circle)

  • If the distribution map symbol is based on a single observation, the Observation window will open with the relevant observation (a Survey event, Sample, Taxon occurrence, Biotope occurrence depending on what was mapped) selected.

  • If the distribution map symbol is based on more than one observation then you will see a list of the observations:

  • Select one of these observations and click [OK]. The Observation window will open with the relevant observation selected.
