Installing on a network server

Recorder 6 is a Microsoft SQL Server system so users need a version of SQL Server to run it. It is supported when used with:

  • SQL Server 2000, or its free run-time version, MSDE, Microsoft SQL Server 2000 Desktop Engine.

  • SQL Server 2005, or its free run-time version, SQL Server 2005 Express.
  • SQL Server 2008, or its free run-time version, SQL Server 2008 Express.

If you already have SQL Server installed, you will be able to install Recorder 6 using your existing SQL Server instance. If not, you will be able to download and install SQL Server Express from a Microsoft web-site then install Recorder 6. Alternatively, you may wish to buy SQL Server so that you can take advantage of more of the functionality available. If you are in any doubt about which version to use, please request advice from the supplier of your copy of Recorder, or via the Recorder Forum, part of the NBN Forum, as to which version is most appropriate for you. Prior to version 6.14 the installation CDs had MSDE on them. The version 6.14 and 6.15 disks had SQL Server 2005 Express on them but later versions do not contain SQL Server because different versions are available for 32-bit and 64-bit computers and because Microsoft upgrades may necessitate using the latest version of SQL Server for installing Recorder 6.

There are standalone and network installation guides in the Documentation folder on all the Recorder 6 installation DVDs and in the Resources section of the Recorder web site, www.jncc.gov.uk/recorder. They have not been updated for some time but they still contain much useful information. Please refer to these for more information about SQL Server, the system requirements and installing Recorder 6, or request updated information via the NBN Forum.

Installing on a network is a much more involved process than a standalone installation and should only be attempted by someone who knows the network in question and is knowledgeable about such procedures. The network installation guide should provide the information they will require but, in summary, the steps are:

  • Creating an instance of SQL Server Express - only required if an existing instance of SQL Server is not already available for Recorder 6 to use. Note that this stage must be performed directly on the database server - not via a workstation.

  • Creating the Recorder 6 database (NBNData_Data.MDF) and attaching it to the instance of SQL Server. Note that this stage must be performed directly on the database server - not via a workstation.

  • Installing the Recorder 6 files on the file server. This stage may be performed via a workstation.

  • Performing workstation installations. Prior to installation ensure that the workstation has a drive mapping to the file server. The mappings from each of the workstations should be identical. Do not install using the DVD. Navigate to WorkstationSetup.exe in the Recorder 6 server folder and double click on it to install Recorder 6 on a workstation. This needs to be done on each workstation.

The second and third steps are performed by inserting the installation DVD, selecting 'Install Recorder 6 on your network' then selecting the appropriate option. For more details see the example network installation in the network installation guide.

If you are installing Recorder 6 on the same computer as your Recorder 2002 system, the installation process will pick up your Site ID from Recorder 2002. Otherwise, you will be asked to enter your Site ID and verification key. You will find these on the licence agreement you received with the installation DVD.

The network installation guide also contains information on:

  • Transferring data from Recorder 2002 to Recorder 6, although more detailed information is available in the Recorder 2002 Resources section of the Recorder web site, www.jncc.gov.uk/recorder

  • SQL Server permissions, security and login options
  • Comparisons of the different versions of SQL Server
  • Migrating from MSDE to SQL Server
  • Backing up the database
  • Uninstalling Recorder 6
  • The installation folders
  • The registry settings.

Note: The network installation DVD can be used to install Recorder 6 on a network or on a standalone computer, although there is a separate DVD for standalone installations.

Note: Recorder 6 version 6.17 is officially supported for use with Microsoft Vista and Windows 7 but the installation guides have not yet been updated. Until they are, for more information, please consult the Recorder Forum, part of the NBN Forum, or contact the supplier of your copy of Recorder, or any of the Recorder suppliers who are listed on the Recorder web site, www.jncc.gov.uk/recorder.
