Copy & Paste

There are two main ways in which you can use Copy & Paste:

  • When you need to add or edit a reference to an item elsewhere in the system. For example, you could copy the name of a location from the Location window, and paste it into the location prompt of a record card.

  • When you want to exchange information between Recorder and another application. For example, you may already have a site description in a word-processor file which you want to use in the 'Location description', or you may want to copy text from Recorder into a document you are writing. Most Windows software supports copying and pasting information, but consult the documentation for the other software for details.

To Copy:

  • Select the information you want to copy using your mouse or keyboard.

  • Copy the information to the Windows Clipboard. In Recorder this can be done by:
    • Selecting Edit – Copy from the Main menu.
    • Clicking the copy button in the toolbar.
    • Pressing the short-cut key <Ctrl-C> on the keyboard.

To Paste information you have copied:

  • Position the cursor at the place were you want to use it.

  • Paste the information from the Windows Clipboard by:
    • Selecting Edit – Paste from the main menu.
    • Clicking the paste button in the toolbar.
    • Pressing the short-cut key <Ctrl-V> on the keyboard.

Troubleshooting

If the information does not appear, it could be for several reasons:

  • You were not successful in copying it to the Clipboard – try again.

  • The information you are trying to paste is not appropriate at that point (for example, you cannot paste a species name into a prompt for a location).

  • The window into which you are trying to paste the information is not in Edit mode.

  • You do not have the necessary access rights to add or change the data at the place where you are trying to paste the item.
