Common names

Common names can optionally be shown on screen wherever names are displayed in preference to scientific names. If this option is selected, then the "preferred common name" is shown followed by the scientific name in brackets, e.g. "Peacock (Inachis io)".

When a species record is entered, the name of the taxon is chosen from a particular checklist. The common name that will be displayed if this option is selected is the "preferred common name" from that checklist. If the checklist does not include a common name for the taxon, then the scientific name will still be displayed.

If you enter your own common name (via Edit Taxon Names) for a taxon, then this will be used in preference to a "system supplied" common name.

To turn on the display of common names:

  • Select Tools - Options from the main menu

  • In the General Options page, tick the "Use Common Names:" box.
