Features Management Aims Tab

The page allows a detailed explanation for specific management aims for a feature.

 

Name

Required.

Free text. A label for this management aim.

Authority

Required.

The individual or organisation responsible for carrying out this management aim. This is a reference to an item in the Names and Addresses window. It can be entered by:

  • Dragging & dropping, or copying & pasting a name from the Names and Addresses window or a rucksack.

  • Typing part of a name and pressing <Enter>. This will display the Find Individuals/Organisations dialog showing matching names.

  • Choose the button to display the Names and Addresses window. Locate the desired item, and Return the data.

Description

RTF Memo. Text describing the management aim.

Next appraisal date

The date at which this management aim should be reviewed. Unlike most dates in Recorder 2002, this can be set into the future. This is a 'standard' date (i.e. NOT a vague date).

Agreement date

Required.

The date on which the Authority agreed to take responsibility for this management aim. This is a 'standard' date (i.e. NOT a vague date).

 

To add a management aim:

  • Click on the button and the detail panel will go into Edit mode with all the fields empty.

  • Type in the required information.

    • Click the button to save your changes.

    • Click the button to abandon your changes.

To delete a management aim:

  • Highlight the management aim that you want to delete.

  • Click the button.

  • You will be asked to confirm that you really want to delete this record.

To edit a management aim:

  • Highlight the management aim that you want to edit.

  • Click the button to put the detail panel into Edit mode.

  • Make the changes you require.

    • Click the button to save your changes.

    • Click the button to abandon your changes.
