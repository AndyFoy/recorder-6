Location Tenure Tab

The Location Tenure tab holds any information on the tenure of the location. This tab allows the user to add or delete any number of owners for a location.

At the top of the tab is a box which shows all of the tenure information associated with the location, and allows the user to add, delete or edit tenure information.

Name

Required.

To enter a name use one of the methods described in Find overview.

Type

Required.

Select a value from the drop-down box. The Tenure type is a ‘Term-list’, and items in this list can be added or deleted by clicking Options - Term-lists.

From

Required.

Enter a valid vague date for the start of the tenure information.

To

Optional. Enter a valid vague date for the end date of the tenure information (must be later than the 'From' date).

 

To add tenure information:

  • Click the [Add] button.

  • Enter the values for the tenure (see the table below).

  • Click the [Save] sub-record button.

To delete tenure information:

  • Highlight the item you want to delete.

  • Click the [Delete] button.

To edit tenure information:

  • Select the item you want to edit.

  • Click the [Edit] button.

  • Make the changes to the data.

  • Click the [Save] sub-record button.
