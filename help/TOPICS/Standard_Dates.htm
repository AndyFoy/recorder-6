Standard dates

  • The formats in which standard dates are entered and displayed depends upon the Regional settings in the Windows control panel, except that years are always displayed with 4 digits, irrespective of Windows settings.

  • The Windows Regional settings control the order of the elements (day, month, year or month, day, year, etc.), the way in which months are shown (digits, abbreviated name, full name), and the separator expected between elements.

  • To change these settings, edit the values in the Date tab of Regional settings in Windows control panel.

If you set the region to 'English (British)' or 'English (United Kingdom)' in the Regional Settings tab, and have not changed the defaults in the 'Date' tab, then the elements will be expected in the order 'day, month, year', and the separator will be '/'. See two digit years.
