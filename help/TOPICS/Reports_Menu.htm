Reports Menu

Run

Re-run a previously saved report. This will generate a new report which summarises the current state of the database using selection criteria, and a report template saved previously.

Wizard

Query the database to find observations for particular taxa, biotopes or geographic areas. The resulting observations can be printed in a report, mapped or exported in various formats.

Quick Report

Runs a number of reports that have been either provided as standard or saved/imported by the user as XML files

This menu changes when the Report Designer is active. See View Menu.
