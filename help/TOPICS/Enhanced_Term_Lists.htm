Enhanced Term Lists

Recorder enables users to compile a list of Keywords that can be assigned to documents in the Document list and a list of Tags that can be used to organise Surveys into folders in the Observation hierarchy. The window is accessed by selecting Tools - Enhanced Termlists from the Main menu.

Term List

Choose the Term List you wish to work on from the drop down list. The options are:

  • Keywords that can be assigned to Documents in the Document list.

  • Tags that can be used to organise Surveys into folders in the Observation hierarchy.

  • Thesaurus Fact Types.

Term

Required.

The label for the item, which will be listed under Available Terms when the list is accessed.

Language

Required.

The language for the term selected from the drop down list.

Synonyms

Synonyms for the term and their language.

To add a new item:

  • Select an existing item in Available Terms, if present.

    • Select Edit – Add from the Main menu.

    • Choose [Add] at the bottom of the left-hand list.

  • Select 'Add Term' or 'Add Child Term'. 'Add Term' adds the new item at the same level in the hierarchy as the item originally selected. 'Add Child Term' adds the new item at the next level down. A '<New Term>' indicator will show where the new term will be placed initially.

  • When you have entered the required data in the boxes on the right, choose [Save] in the right-hand panel of the window to update the database with the new item.

  • You can choose [Cancel] in the right-hand panel to abandon any changes. No changes will be made to the database, and the window will be restored to its original state.

To edit an existing item:

  • Ensure that the item you wish to edit is highlighted in the list on the left of the window.

    • Select Edit – Edit from the Main menu.

    • Choose [Edit] at the bottom of the left-hand list.

  • When you have made the required changes, choose [Save] in the right-hand panel of the window to update the database and make the changes permanent.

  • You can choose [Cancel] in the right-hand panel to abandon any changes. No changes will be made to the database, and the window will be restored to its original state.

To remove an item:

  • Ensure that the item you wish to remove is highlighted in the list in the left of the window.

    • Select Edit – Delete from the Main menu.

    • Choose [Delete] at the bottom of the left-hand list.

  • You will be prompted to confirm that you really want to delete this item.

NOTE: You cannot edit items 'belonging' to someone else, which includes 'system supplied' items (i.e. items which come with Recorder).

You cannot remove 'system supplied' items or an item which is referred to by other entries in the database. Therefore, if a term is in use, you will not be able to remove it unless you remove it from the items that use it first.
