Options Appearance

The options on this tab affect the colours used in the application and other aspects of the appearance of the screen. The [Undo] button allows you to change your mind, and restores the settings that were in force when the Options window was opened.

Mandatory fields

The [colour selection] button allows you to choose a colour which will be used as the background colour for mandatory fields. Select white if you wish to turn this feature off. The example on the right allows you to assess whether a colour is suitable. Dark colours do not work well because it is difficult to read the text.

Information transfer colours

The 'Disable Information Transfer Colours' checkbox allows you to turn off the facility to have coloured borders around controls between which you can drag and drop information.

The [Information source] and [Information destination] colour selection buttons allow you to select the colours used for the borders of such fields.

  • Information sources are fields from which information can be dragged.

  • Information destinations are fields where it can then be dropped.

Some fields can be both a source and a destination, and have a dashed border with both colours. The examples on the right allow you to assess whether your chosen colours are suitable. Some combinations may not work well for some users.

Application background

Use this option to select whether the background should be plain or should consist of a bitmap.

  • If you choose a plain background, then its colour is that set in the Windows Control panel under Display properties – Appearance.

  • If you choose to use a bitmap, then the button opens a standard Windows file selection dialog to load a suitable image file. The selected image will be repeatedly applied (like a tile) to form a continuous background.

Show tool tips

This option selects whether or not tool-tips appear. Tooltips are short help messages which appear when the mouse cursor hovers over a control. They are helpful to new users, but can eventually become irritating to experienced users.
