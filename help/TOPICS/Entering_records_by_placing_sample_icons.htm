Entering records by pointing at the map

This facility allows you to start entering records using a Recording Card by clicking at the sampling location on the map. It is only available if at least one sample type has been associated with a Recording Card - see Sample type term list. If this is the case, the button on the Map Browser's active toolbar will be enabled.

  • To add a new sample:

    • Click the button in the active toolbar.

    • Select Map - Add sample from the menus.

  • A submenu of sample types will be displayed. Select the desired sample type.

  • The map cursor will change to a cross-hair .

  • Click on the map at the location where the sample was taken. The dot at the centre of the cross hair is the point at which a spatial reference will be recorded.

  • The Recording Card associated with the selected sample type will open with the spatial reference already filled in. If you clicked within a site boundary polygon which is associated with a location, the name of the location will also be filled in.

  • Fill in and save the Recording Card as normal.

If you remove, rename or delete the recording card that is referenced here- you will receive an error every time you attempt to open the map window. This won't stop you doing anything but it can be an annoyance.
