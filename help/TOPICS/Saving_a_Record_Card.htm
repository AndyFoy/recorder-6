Saving the information entered using a Recording Card

Once you have completed a recording card, you can save the all the information you have entered by clicking on [Save].

The information on a recording card is stored as a new sample for the selected survey. Where the sample is stored depends upon the location, date and recorder for the recording card. If a survey event already exists with the same location, date and recorder as the recording card, then the information is saved as a new sample for that event. However, if no matching event exists, a new survey event is created, and the information added as a new sample for this event. A taxon occurrence will be created for each species you ticked in the list, and a single biotope occurrence will also be created if you made an entry in the 'Biotope' prompt on the card.

Once the card is saved, all of the observations are cleared from the card, leaving the other information intact (e.g. the survey, location, recorders etc.), with the exception of the sample comment. This allows the user to enter another set of observations without having to re-enter all the data in the heading of the Recording card. The comment isn't retained because it is more likely to be different for each sample.
