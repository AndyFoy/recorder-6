Report Wizard Designation Selection

You see this screen if you selected the Restrict report to one or more Designations option in the starting screen of the Wizard. It is used when you want to produce a report that only contains observations with specified designations.

Designation Set

Use the drop down arrow to select the set of designations you want to use for your report. The sets available in Recorder 6 version 6.14 are:

All Designated Species (excl Northern Ireland)
All European Designated (Birds Dir, H&S, Berne, Bonn, EC CITES)
All UK Legally Protected (W&C Act, Badgers and Cons Regs)
Wildlife & Countryside Act (Sch 1, 5 & 8)
IUCN Red Lists
Biodiversity Action Plan species (2007)
Country/UK Biodiversity Lists
NI Wildlife Order
Farmland Environment Planning
Planning Legislation
Statuses of Nationally: Rare, Notable, Scarce

Additional Designation Sets can be added to the system by those with appropriate skills. An XML Batch update could be written to do this.

Note: If Designation Sets are added, the Taxon Designation Index must be rebuilt by selecting Tools - Database Tools - Rebuild Taxon Designation Index from the Main menu.

Available

If you want observations with any of the designations in a designation set in your report use the double headed red arrow between the Available box and the Selected box on the right to send all the designations in the set to the Selected box.

If you only want observations with some of them in your report, select the designations you require and use the single headed red arrow to send them to the Selected box.

By changing your selection in the Designation Set box, you can add other designations from other sets.

Observations will only be included in your report if they meet all the selection criteria you specify as you go through the report wizard. In the case of designations, observations will only be included if they have one or more of the designations listed in the Selected box on the right.

The fields that appear in your report are controlled by your selection of attributes.
