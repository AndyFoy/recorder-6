Source page

This page occurs in most of the main data entry screens. It allows documentary sources of information to be indicated by referring to an entry in the Documents window. It also allows external files to be linked to the record. These could include word-processor files containing further details, pictures, sounds or any other type of file supported by your computer.

Internal references

A list of items from the Document window associated with the current record.

Reference

Identifies the item in the Document window

Original

Is this the original source of information (Yes/No), or is it indirect i.e. a document that cites another source as the original?

For example, A finds a rare species and publishes the observation. Later B publishes a review of the species which cites A's publication. Both publications might appear in the Sources page of a taxon occurrence entry detailing this observation: A's publication is the original source and would be flagged 'Yes' whilst B's publication would be flagged 'No'.

Type

The type of document. Automatically filled in.

To add an internal reference:

  • Ensure the detail panel is in Edit mode (The [Save] and [Cancel] buttons in the detail panel will be active).

  • Locate the desired documentary reference. You can do this by:

    • Dragging the document from the document window or Rucksack into the grid.

    • Click the button to open the Find Document dialog. Highlight the document you want in the list, and click [OK] or press <Enter>.
    • Choose the button. The Document window will be displayed. Select the document you want and return it (by selecting Edit – Return data from the menu, clicking the button in the active toolbar, or pressing the short-cut key <F9>).

  • Change the setting in the 'Original' column if necessary – it defaults to 'No'.

To edit an internal reference:

  • Ensure the detail panel is in Edit mode (The [Save] and [Cancel] buttons in the detail panel will be active).

  • Click on the cell you wish to edit and make the desired changes. Changing the document is the same as adding it. Use drag and drop; the button to open the Find Document dialog, or the button to locate an entry in the Document window, and then Return the data by pressing <F9>.

To remove an internal reference:

  • Ensure the detail panel is in Edit mode (The [Save] and [Cancel] buttons in the detail panel will be active).

  • Select the row in the grid that you wish to delete by clicking on one of its cells with your mouse, or by moving to a row using the <up arrow>,<down arrow>, <PgUp>, <PgDn>, <Home> and <End> keys.

  • Choose the button. You will be prompted to confirm that you really want to delete the selected item.

External references

These consist of a list of files available on this computer, or the network to which it is attached.

To add an external reference:

  • Ensure the detail panel is in Edit mode (The [Save] and [Cancel] buttons in the detail panel will be active).

  • Click the button to open a standard Windows file location dialog. Select the file you require, and choose [Open] in the dialog. The path and name of the selected file will appear in the list.

To delete an external reference:

  • Ensure the detail panel is in Edit mode (The [Save] and [Cancel] buttons in the detail panel will be active).

  • Select the file name you wish to delete by clicking with your mouse, or by moving to a row using the <up arrow>, <down arrow>, <PgUp>, <PgDn>,<[Home> and <End> keys.

  • Choose the button. You will be prompted to confirm that you really want to delete the selected item.

You cannot edit a file name. To make a change, delete the file, and then add the new version.

To view an external file:

  • Select the file name you wish to view by clicking with your mouse, or by moving to a row using the <up arrow>, <down arrow>, <PgUp>, <PgDn>, <Home> and <End> keys.

  • Click the button

OR

  • Double click on the name of the file you wish to view.

The application which is registered with Windows to view that type of file will be launched, and the file loaded into it. For example, if you view a .txt file, Windows will open Notepad with that file loaded into it.

NOTE: Viewing a file uses the facility provided by Windows to launch the application required to view that file. This is based on that file’s extension (.txt, .bmp, etc). It therefore relies upon the file extension being correctly registered with Windows, and the necessary application being available.
