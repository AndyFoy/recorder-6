Location of files on a network installation

For a network installation of Recorder 6, most files are located on the file server so that they can be shared between multiple Recorder 6 users across the network. However, the connection between a workstation and the file server on a network is likely to be significantly slower than the speed with which data can be written to and read from a local hard disk. Therefore on a networked installation certain folders are maintained on the local disk for performance reasons.

In summary, if the default options are used during installation with Windows XP:

  • The Recorder 6 database (NBNData_Data.MDF) and its associated log file (NBNData_log.LDF) will be stored in the Data folder in a Microsoft SQL Server folder, e.g. C:\Program Files\Microsoft SQL Server\MSSQL.1\MSSQL\Data for SQL Server 2005 Express.

  • Most of the application files will be stored in the Recorder 6 Server folder on the server. The location of some of the application files will be the folders listed in Tools – Options – File Locations.

  • The application files which need to be stored on each workstation for performance reasons will be in C:\Program Files\Recorder 6 Workstation.

See the network installation guide for more information. There are standalone and network installation guides in the Documentation folder on all the Recorder 6 installation DVDs and in the Resources section of the Recorder web site, www.jncc.gov.uk/recorder.

Note: Recorder 6 version 6.17 is officially supported for use with Microsoft Vista and Windows 7 but the installation guides have not yet been updated. Until they are, for more information, please consult the Recorder Forum, part of the NBN Forum, or contact the supplier of your copy of Recorder, or any of the Recorder suppliers who are listed on the Recorder web site, www.jncc.gov.uk/recorder.
