Editing an item

To edit a single item:

  • Ensure that the item you wish to edit is highlighted in the list on the left of the window.

    • Select Edit – Edit from the Main menu.

    • Choose [Edit] at the bottom of the left-hand list.

  • The [Add], [Edit] and [Delete] buttons will be disabled, and the detail panel on the right-hand side of the window will be activated. You can now make changes.

  • When you have made the changes you require, choose the [Save] button in the right-hand panel of the window to update the database and make the changes permanent.

  • You can choose [Cancel] in the right-hand panel to abandon any changes. No changes will be made to the database, and the window will be restored to its original state before you started to edit.

To edit multiple items:

  • When editing the dates and location data for observations you will be offered the option to cascade the change up and/or down the observation hierarchy, as specified in editing dates and location data.

  • If you need to edit multiple items in a sample and the taxon occurrences it contains you can load the sample into an appropriate recording card and make the changes in one operation - see Modifying data in samples at the end of Entering data using a Recording Card.

Note: you cannot change an item using these facilities unless you have sufficient access rights and your copy of Recorder is the custodian of it, i.e.

  • it was entered into your copy of Recorder using one of the standard data entry facilities and custody (ownership) has not been transferred to another Recorder system when exporting it, or

  • custody was transferred to you when the item was exported from another copy of Recorder.

Note: for information on the custody of an item, select it then click the Show Metadata button in the active toolbar.

Note: where adding information to an item involves creating a new item in a table in the Recorder database, e.g. adding a new determination to a taxon or biotope occurrence, you can add the new information, e.g. a determination, if you have sufficient access rights, even though you are not the custodian.
