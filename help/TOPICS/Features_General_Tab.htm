Features General Tab

This page is used to enter a general description of the feature.

 

Name

Required.

Free text. A label for the feature. This will be shown in the hierarchy in the Locations window and in Reports, to identify the feature.

Type

Used to categorise the feature. Select an item from the 'Location feature type' term list. Users can add new items to this term list if necessary.

Grading

Select an item from the 'Location feature grading' term list. These terms are linked to the feature type, so you will only see grades appropriate for the feature type.

Comments

RTF Memo. describing the feature.
