Uninstalling

To uninstall the software

  • Make sure Recorder is not running.

  • Select Start - Settings - Control panel from the Windows menus.

  • Double-click on Add/Remove Programs.

  • Ensure that the Install/Uninstall page is displayed. Select 'Recorder remove only' in the list towards the bottom of the dialog box.

  • Once the program is selected, the [Add/Remove...] should become active. Choose this button, and follow the on-screen instructions.

This process should completely remove Recorder from your machine-complete with all of the registry settings and files. You may wish to run the uninstall programme if you have a partial or broken installation and want to clear your system before installing the programme again
