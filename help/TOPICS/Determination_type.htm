Determination type

The determination type is used on taxon and biotope determinations where the required value is selected from the determiner type term list - illustrated below. If the required type is not present users can add it as specified in Adding a term to a list.

Adding, editing and deleting determination type term list items is the same as for most other term list items - see Adding a term to a list - except for the Additional Information:

  • Determination Verification. This is used to set the Verified flag on taxon and biotope occurrence - see Determination Verification for details. The default value is Not Verified.

  • Hide - tick this box if you want to block the use of this item for data entry, e.g. if you do not want it to appear on the drop down list for Type on taxon and biotope determinations.
