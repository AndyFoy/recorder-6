Names and Addresses Contact Nos Tab

This page allows contact details like telephone number or email address to be entered. One contact number can be marked as preferred – this is the number which appears in the summary on the first page.

 

Type

Required.

The type of contact number (telephone, email, etc). Free text with suggested values available from list. Commonly, types of contact numbers can be picked from the list.

Prefix

Free text. Use this if a prefix like '9' is required, for example, to get an outside line. (Provided for future expansion such as a link to an autodialer).

Number

Required

Free text. The full details of the contact number, e.g. the telephone number including dialing code or the full email address.

Constraints

RTF memo. Any comment indicating the circumstances that this contact number should, or should not, be used.

Preferred

If you only enter one contact number, this option will be greyed out, and that contact number is automatically 'preferred'. If more than one contact number is entered, only one can be 'preferred'. Click on the preferred box to mark a contact number. The tick will automatically be cleared from the contact number that was previously marked as preferred. The summary on the first page shows the preferred contact number.

To add a contact number:

  • Click on the button and the detail panel will go into Edit mode with all the fields empty.

  • Type in the required information.

  • Click the button to save your changes.

  • Click the button to abandon your changes.

To delete a contact number:

  • Highlight the contact number that you want to delete.

  • Click the button.
  • You will be asked to confirm that you really want to delete this record.

To edit a contact number:

  • Highlight the contact number that you want to edit.

  • Click the button to put the detail panel into Edit mode.
  • Make the changes you require.

  • Click the button to save your changes.

  • Click the button to abandon your changes.
