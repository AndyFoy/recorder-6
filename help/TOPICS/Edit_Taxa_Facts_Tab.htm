Edit Taxon details - Facts

The Edit Taxon details - Facts page holds any factual information for the taxon. This information could be descriptions, ecology notes, images and distribution maps. This tab allows the user to add, edit or delete factual information for the taxon.

At the top of the tab is a box which shows all of the facts associated with the taxon, and allows the user to add, delete or edit facts.

Title

Optional. Enter a title for the fact.

Type

Required.

Select a fact type from the drop-down box.

Date

Optional. Enter a valid vague date for the fact.

Facts

Required.

Enter the data for the fact. This should be in HTML format and is displayed in the Taxon Browser.

Reference

Optional. A valid internal reference for the fact.

To enter a reference, use one of the methods described in Find overview.

To add a fact:

  • Click the [Add] button.

  • Enter the values for the fact.

  • Click the [Save] sub-record button.

To delete a fact:

  • Highlight the item you want to delete.

  • Click the [Delete] button.
  • You will be asked to confirm the deletion.

NOTE: Facts that are system-supplied cannot be deleted.

To edit a fact:

  • Select the item you want to edit.

  • Click the [Edit] button.

  • Make the changes to the data.

  • Click the [Save] sub-record button.

NOTE: Facts that are system-supplied cannot be edited.
