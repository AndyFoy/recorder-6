New Taxa General Tab

The New Taxa General tab holds general information on the new taxon item. This tab allows the user to add, edit or delete information for a new taxon only. It does not allow the user to edit system-supplied data.

Name

Required.

The taxonomic name for the item.

Authority

Optional. An authority for the taxonomic name, and optionally the year for original description.

Rank

Required.

Select a value from the drop-down box. The ranks displayed in this box will depend on the rank of the parent taxon. For example, if a new taxon is being added to a genus, only taxonomic ranks that lie beneath genus will be displayed in the box.

To enter a new taxon:

  • Click on the [Add] button .

TIP: If the [Add] button is disabled, the selected checklist is not installed on the hard drive, but is being read from the CD-ROM. To enable editing of this checklist, install the checklist (see Installing Checklist).

  • Select either a sibling or a child to add.

  • The hierarchy changes to grey to indicate that an item is being added.

  • Enter the values for the new taxon (see the table below).

  • Click [Save].

To edit a taxon:

  • Select the item that you want to edit.

  • Click on the [Edit] button .
  • Make the changes to the values (see the table below).
  • Click [Save].

NOTE: With system-supplied data (see Centrally supported dictionaries and new taxa/biotopes), no information can be edited on this tab. However, with user-entered data all items can be edited.

To delete a taxon:

  • Select the item that you want to delete.

  • Click [Delete].

  • You will be asked to confirm the deletion.

NOTE: System-supplied data (see Centrally supported dictionaries and new taxa/biotopes) cannot be deleted, but user-entered data can be.

TIP: Taxa cannot be deleted until all of their children have been deleted.
