Using items from a Rucksack list

  • Open the appropriate list by clicking on the 'Taxon', 'Biotope', 'Locations', 'People' or 'References' tab.

  • Drag items from the list, or use copy and paste, to get an item into an appropriate control.

  • Drag a Taxon or Biotope from the appropriate list onto the map shown in the Map Browser to produce a distribution map.

TIP: You can only move information into controls designed to take that type of information. You cannot, for example, put a taxon name into a prompt for a location.

The taxon or biotope lists from the current (open) rucksack can be used by the Report Wizard to select the species or habitats on which to report. See Report Wizard Biotope or Taxon selection.

The taxon list from a rucksack can be used as the basis for a Recording Card. See New Recording card.
