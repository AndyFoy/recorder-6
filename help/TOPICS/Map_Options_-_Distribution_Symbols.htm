Map Options - Distribution Symbols

To add symbol sets:

  • Drag a Survey or Survey event from the Observation window to the map

  • Drag a taxon name from the Taxon Dictionary or a Rucksack to the map
  • Drag a biotope category name from the Biotope Dictionary or a Rucksack to the map
  • Send the results from the Report Wizard - Filter Results screen to the map

You can add up to 10 symbol-sets to the map. If you try to add an 11th set, you will be prompted to choose the one you wish to drop to make room for it.

To remove symbol sets:

  • To remove ALL the symbol sets in one go:

    • select Map - Clear from the menus, or click the button in the toolbar.

  • To remove a particular symbol set:

    • Open the Map Options dialog (if it is not already open) and go to the Distribution Symbols page
    • Select the symbol set you wish to remove by clicking on it
    • Press the Delete button
    • You will be prompted to confirm that you really want to remove the Symbol set.

To turn symbol sets on or off:

Click on the tick-box to the left of the symbol set name in either the list on the left-hand side of the Map Browser window, or in the Map Options dialog Distribution symbol page. Click once to turn off the display (the tick-box will be empty) and again to turn it back on (the tick-box contains a tick).

To change the colour of the symbols for a symbol set:

Double-click on the coloured dot to the left of the symbol set name in either the list on the left-hand side of the Map Browser window, or in the Map Options dialog Distribution symbol page. A colour selector dialog will be displayed, from which you can choose a new colour for the symbol set.

To change the legend for a symbol set:

Click on the name of a symbol set in either the list on the left-hand side of the Map Browser window, or in the Map Options dialog Distribution symbol page. The text will be surrounded by a focus-box and a cursor will appear at the point at which you clicked. You can now edit the text in the normal way. Click somewhere outside the focus box to save the changes.

You can also set the cut-off year for open or closed distribution symbols from here.
