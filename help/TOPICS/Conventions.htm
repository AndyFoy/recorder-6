Conventions used in this Help file

The keyboard

When you are instructed to press a key, it is shown in triangular brackets, e.g. <Enter> means 'press the Enter (or Return) key on your keyboard'. <Ctrl-V> means hold down the control key (marked 'Ctrl') and press 'V' on your keyboard at the same time.

The keys to move the cursor around the screen, which are often labelled on the keyboard with arrows pointing upwards, downwards, to the left and to the right, are referred to as <Up>, <Down>, <Left> and <Right>.

Buttons on the screen

Where buttons are just identified by an icon (picture), the icon is used in the help file (e.g. 'click '). Buttons that are identified by labels are referred to by giving the label in square brackets, e.g. 'choose [OK] to finish').

Warnings and tips

Things you should really know about and/or need to watch out for are shown like this:

NOTE: You really MUST do this and NOT this.

Useful hints and tips are shown like this:

TIP: This will make life easier.

Links

Hyperlinks are used widely in this Help file so that you can jump to another topic from the point at which it is mentioned. They appear like this. You know when you are over a link because the cursor changes to a pointing finger . You can use the button to return to the topic you came from.

You will also find glossary items (e.g. taxon) which show a definition in a pop-up window, rather than taking you to another topic. Just click outside the pop-up window to dismiss it.
