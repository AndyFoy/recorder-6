Location Administrative Areas page

The Location Administrative areas page holds information on any administrative areas that are related to the location. The page allows the user to add or remove any number of administrative areas for the location.

Name

The name of an Administrative area from the Admin area browser.

Type

This shows the type of admin. area you selected (e.g. English county, Vice-county). It is automatically filled in.

To add an Admin area:

  • Select the admin area

  • Click on the [Add] button.

  • The Admin areas browser displays.

  • Select the admin area that you want to add.

  • Press <F9>.

OR

  • Drag a location into the grid from the Admin area browser.

  • Click the button to save your changes.

  • Click the button to abandon your changes.

To remove an admin area from a location:

  • Highlight the item you want to delete.

  • Click on the [Delete] button.

  • You will be prompted to confirm that you really want to remove the Admin area.
