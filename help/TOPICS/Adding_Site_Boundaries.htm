Adding site boundary polygons and transects

You can either draw lines and polygons using the cursor (see drawing polygons), or import them (see importing polygons) from vector map files in one of the formats supported by MapServer.

Once a site boundary or transect appears on the map, it can be associated with a record in the Locations table (see associating polygons). The location record is then used to hold information about the item on the map, such as its name and designation.

To add a line or polygon:

  • Select Map – Add boundary from the Main menu. Then choose Draw or Import from the sub-menu.

  • Click on the button on the active toolbar. Then choose Draw or Import from the menu which opens below the button.
