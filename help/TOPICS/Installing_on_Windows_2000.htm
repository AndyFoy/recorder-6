Installing on Windows 2000

The additional security implemented by Windows 2000 creates some extra issues when installing Recorder 2000:

  1. To install Recorder 2000 (as with most packages) you must be logged on as a user with "Administrator" privileges.

  2. Installation creates some registry settings which need to be copied to the profile of each user-name that will use Recorder 2000 on that particular machine. It is therefore necessary to make files containing these settings:

    • Run RegEdit (select Start - Run from the Windows menu, type "regedit" and click [OK])

    • Select the HKEY_CURRENT_USER\Software\JNCC key and then select Registry - Export Registry File from the menus. It is recommended that you save the file in \Recorder 2000\Settings.

    • Select HKEY_LOCAL_MACHINE\SOFTWARE\JNCC and export that key in the same way.

    • Exit from RegEdit.

  3. Now log "Administrator" off and log on using the user-name of one of the people who will use Recorder 2000. Go to the \Recorder 2000\Settings directory and double-click on each of the two ".reg" files you saved in step 2. You should see a message saying the settings have been successfully imported into the registry. This user should now be able to start Recorder 2000 normally.

  4. Repeat step 3 for each user-name who will use Recorder 2000 on that computer.
