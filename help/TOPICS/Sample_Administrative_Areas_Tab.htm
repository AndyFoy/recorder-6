Sample Administrative Areas page

Often a collator will want to associate observations with particular vice counties or other administrative areas. In some cases the location hierarchy has been used for this purpose but this creates problems where the observations have been imported from another copy of Recorder and already have a location associated with them (if the user is not the custodian of the observations this cannot be altered) or there is a genuine need to use the location hierarchy in another way.

To deal with this users can associate observations with administrative areas using the Administrative Areas tab on the sample. This tab holds information on any administrative areas that are related to the sample. It allows the user to add or remove any number of administrative areas for the sample. Start by editing the sample for which you want to add or delete administrative areas.

Name

The name of an administrative area from the Administrative Area browser.

Type

This shows the type of administrative area you selected (e.g. English County, GB Vice-county). It is filled in automatically.

To add an admin area:

  • Select the Administrative Areas tab on the sample

  • Click on the [Add] button.

  • The Administrative Area browser opens.

  • Find the admin area that you want to add and select it.

  • Click the [Return data] button in the toolbar to return the selected admin area to this page.

OR

  • Drag an admin area into the grid from the Administrative Area browser.

OR

  • Copy an admin area into the grid from the Administrative Area browser.

  • Click the [Save] button to save your changes.

  • Click the [Cancel] button to abandon your changes.

To remove an admin area from a sample:

  • Select the item you want to delete.

  • Click on the [Delete] button.

  • You will be prompted to confirm that you really want to remove the Admin area.

  • Click the [Save] button to save your changes.

  • Click the [Cancel] button to abandon your changes.
