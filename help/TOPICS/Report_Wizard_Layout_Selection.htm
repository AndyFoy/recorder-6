Layout Selection

Each report consists of a number of different steps:

  • Selection of the observations that you want included

  • Choosing the data (or attributes) from those observations (like location name, observation comment, etc.) that you want included

  • Specifying what order you want them in

  • Designing a layout for displaying or exporting your observations (a report design or a database snapshot)

When you reach the Layout Selection page of the report wizard you will have selected the observations you want to include although you will be able to refine your selection further by setting conditions or changing the constraints. Both these can be done via the [Additional Filters] button.

If you select New Layout here and click [Next] you will move on to the page where you choose the attributes followed by the one where you specify the sort order.

If you select Choose the same attribute columns and layout template/snapshot as used in an existing report you will be able to select an existing layout template/snapshot to use for your report. Initially only two options will be available, the standard reports: Places for Occurrences and Occurrences for Places. In this case, when you click [Next] you will move on to the Summary of Selected Options page. You will skip the pages where you choose the attributes and the sort order as they are stored as part of the existing layout and do not need to be chosen again. The existing reports that you may choose from will be:

  • A simple report filter displayed as a grid, or

  • A report filter and associated report template, or

  • A report filter and associated database snapshot.

Layout templates and snapshots are files stored in the Templates or Snapshots folders in the User Files folder in your Recorder system unless the locations have been changed using Tools - Options - File Locations. To create new ones, select New Layout at this stage in the report wizard then when you reach the filter result use the [Report output] button and select New Report Template to open the report designer to design a new report layout or New Snapshot to open the snapshot tool. Alternatively, you will be able to obtain them from another user by copying the required file to the appropriate location in your system.
