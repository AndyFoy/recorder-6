Location report

The [Report] button on the Location window can be used to produce two types of report about the Location which is currently selected:

  • Summary: basic information like the name, alternative names, designations, description and location of the Location.

  • Species list: a list of the species recorded from that Location (optionally including its sub-sites and associated sites).

Theses reports can be printed or saved in a variety of formats.

To produce Location Reports

  • Ensure that the Location on which you want to report is selected and press the [Report] button.

  • Choose Summary or Species list from the menu.

  • The Summary report immediately displays a report in the print preview screen.

  • Selecting the Species list report results in the following dialog:

Include subsites

This will only be active if the selected Location has sub-sites.

Include related sites.

This will only be active if the selected Location has related-sites.

Attributes

Remove the ticks from any items you don't want in the resulting list.

Press [OK] to produce a species list. This will be displayed in the print preview screen.
