Upgrading Recorder

Upgrades for Recorder will periodically become available and will need to be installed in your system so that you can take advantage of the bug fixes and the new/improved features they contain. When an upgrade to Recorder becomes available JNCC will announce it on the Recorder Forum, part of the NBN Forum, which can also be accessed from the Recorder web site, www.jncc.gov.uk/recorder. You are advised to check this from time to time for news of upgrades and other information about Recorder.

Upgrades to Recorder are made available as ".zip" files that you download from the Recorder web site, www.jncc.gov.uk/recorder. They are "zipped" or compressed so that they will download more quickly.

For the details of the changes in an upgrade, see the accompanying release notes, which can be accessed from the same place on the Recorder web site as the upgrade itself. Please read them to check for any variations in the standard instructions below for running a particular upgrade. Provided this Help is updated for an upgrade, the Home page will list the new features and changes to functionality in it.

Note: Prior to version 6.9 of Recorder, upgrades to the software and the dictionaries were combined but in version 6.9 a Dictionary Upgrade Kit was introduced so that changes to the taxon dictionary could be done independent of changes to the software. See Dictionary Updates for details.

Note: Sometimes upgrades are cumulative, i.e. running one upgrade will apply more than one so it is not necessary to run each upgrade separately. Whether or not an upgrade is cumulative will be made clear on the web site so please check for this information before upgrading your system.

Note: Whenever an upgrade is to be installed, it should be done through an administrator user account to avoid any potential problems due to limited user account access rights. The "sa" login or a trusted connection (see below) with database owner privileges over the Recorder database is required during the upgrade.

If you are upgrading a standalone copy of Recorder on your personal computer running Windows XP, the way you normally login will probably provide sufficient rights to run an upgrade, but if you are operating on a network this may not be the case and you may need to ask a system administrator to run the upgrade for you. If you are running Vista, you will need to right click RecorderUpgrade.exe and select Run as administrator, to give yourself the necessary rights. If you are running Windows 7, you will need to double click RecorderUpgrade.exe then answer Yes to the User Account Control question 'Do you want to allow the following program (RecorderUpgrade.exe) from an unknown publisher to make changes to this computer?

To check the version of your copy of Recorder to ensure you run the correct upgrade, use Help - About Recorder 6 from the main menu.

The upgrade process replaces various files and may change your database. This will not, however, affect any information you have entered. It may take a few minutes but you will see progress bars and messages as the various tasks take place. When an upgrade completes, it stores a sequence code for the upgrade being run in the Recorder database. This can be found in the Setting table, in the DATA field where NAME = "DB Seq". If you have SQL Server tools or Microsoft Access, you will be able to check the value in this field. It is used to prevent you from skipping upgrades or running them out of sequence.

Users are advised to backup their database prior to upgrading via Tools - Database Tools - Backup Database or the SQL Server tools they use for backups.

To upgrade Recorder :

  • Download the required upgrade from the Download Latest Version section of the Recorder web site, www.jncc.gov.uk/recorder, and save it somewhere on your hard disk. It is recommended that you store this in a folder outside your Recorder 6 folder(s).

  • Unzip the file you downloaded. Clicking on the ".zip" file will usually open the extraction software on your computer. You need to select the options that will unzip the selected file to a new folder in the same location and retain the folder structure when the file is unzipped. If extraction software is not available, you will need a suitable software tool, such as WinZip, to extract the files from the ".zip" file.

Note: It is important that the folder structure is retained when the file is unzipped but this may not happen automatically unless you have the appropriate options selected. If there is a Scripts folder in the file you downloaded, there must also be one in the unzipped folder. Any scripts in the upgrade will not be run unless they are in that folder.

  • Make sure that Recorder is not running.

  • Run the upgrade by:

    • Locating the RecorderUpgrade.exe file amongst the files you unzipped using "My Computer" or "Windows Explorer", and double-clicking it, if you are running Windows XP, or by right clicking it and selecting Run as administrator, if you are running Vista or Windows 7.

    • Alternatively, select Start - Run from the Windows menu, type "\RecorderUpgrade.exe" preceded by the full file path for the folder that contains it and click [OK]. Note that for Vista and Windows 7 this option works a little differently.

Note: If you are upgrading a network installation the upgrade needs to be run from a computer that operates as a Recorder workstation as the upgrade uses the registry settings on the workstation to locate the things to be upgraded.

  • You will be prompted to select the SQL Server system administrator login you would like to use to log on to the SQL Server. The options you chose will depend on how SQL Server was installed. They are:

    • "Use the following SQL Server system administrator login", for which you need to input a Username and Password. If you are running the system under MSDE accept the default, i.e. Username "sa" and Password left blank. For other versions of SQL Server, you will need to know the password that was specified when SQL Server was installed.

    • "Use my Windows Account to login (Windows Authentication)", the trusted connection option. Use this option if Windows Authentication or Mixed Mode Authentication was specified when SQL Server was installed. If you are running a standalone installation with Windows XP, Vista or Windows 7, it will be easiest to use this option.

  • When the upgrade is complete the message "Recorder upgrade complete" will appear.

You are now in a position to use Recorder again. Open it, and use Help - About Recorder 6 to check that the system has been upgraded.
