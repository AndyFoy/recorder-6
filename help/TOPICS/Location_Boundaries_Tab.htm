Location Boundaries page

Location boundaries are physical representations of the locations on the Map browser as points, lines or polygons. Users can import these boundaries or draw them directly onto an internal Recorder map, and then associate boundaries with locations.

Once a location has been associated with a boundary, then when a user clicks on the boundary on a Recorder map, the location name will display at the bottom of the Map browser. This allows a user to clearly identify the position of a location from an internal map, and provides a useful spatial validation when entering data.

The location boundaries page holds information on the boundaries on an internal map associated with a location. It can also be used to hold external data, i.e. the GIS file from which the polygon was imported. This page is used to add, edit or delete any number of boundaries for a location. Start by editing the location for which you want to modify boundaries or external data.

At the top of the screen is a box which shows the boundaries and external data associated with the location. It allows the user to add, edit or delete these entries.

From

Optional. Enter a valid vague date for when the boundary was first recognised or used.

To

Optional. Enter a valid vague date for when the boundary was altered or no longer used (must be later than the From date).

Version

Required.

Enter an identifier for the boundary version. This value must be a number.

Map File

This displays the Recorder map and polygon layer (in brackets) on which the polygon is stored when the entry represents a link between the location and a polygon on a map in Recorder.

When the entry is used to store external data, e.g. the GIS file from which the polygon was imported, it will contain a file name including the full file path.

Object ID

This displays the Object ID of the polygon to which the location is linked by this entry when it represents a link between the location and a polygon on a map in Recorder.

When the entry is used to store external data, this contains the value of an attribute from the file selected in the Map File box. The type of attribute will be displayed above the Object ID.

To add a boundary:

  • Click the [Add] button on the Geo Info - Boundaries tab.

  • Click the [Get] button beside the Object ID box to open the Map browser.

  • Select an existing boundary, or draw a new boundary on an internal map and select it.

  • Click the [Return data] button in the toolbar to return the required information to the Map File and Object ID boxes.

  • Enter the remaining values for the boundary.

  • Click the button to save your changes.

  • Click the button to abandon your changes.

Note: To import boundaries see Importing Polygons.

Note: Boundaries can also be added from an internal Recorder map using the Associate Boundary tool. When this option is used an entry is added to the Boundaries tab automatically.

To store external data:

  • Click the [Add] button on the Geo Info - Boundaries tab.

  • Click the button beside the Map File box to open a standard Windows file locator dialog and locate the external file that contains a boundary for the selected location.

  • Use the drop down arrow beside the Boundary Attribute Field in the following dialog to select the attribute you want to store in the Object ID field:

  • Click [OK].

  • Use the drop down arrow beside the Object ID box to select a value for the specified attribute.

  • Enter the remaining values for the boundary.

  • Click the button to save your changes.

  • Click the button to abandon your changes.

To delete a boundary or external data:

  • Highlight the item you want to delete.

  • Click the [Delete] button.

  • You will be prompted to confirm that you really want to remove the link between the location and the boundary.

To edit a boundary:

  • There is no direct method of editing a boundary – the shape of the boundary is fixed once it has been drawn or imported on an internal map. The best way to edit a boundary is to delete it then redraw it (or re-import it) as a new version of the boundary.

  • The [Edit] button is used to change some of the information about a boundary, not the actual boundary itself, e.g. From date, To date and Version.

TIP: There are many sources for boundary data that can be used in Recorder. See Importing polygons for a couple of options.
