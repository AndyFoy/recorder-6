Measurements

This page occurs in the data entry windows for Locations, Samples, Taxon occurrences and Biotope occurrences. It is used to enter all types of measurements and counts.

 

Measurement

A value chosen from a Term list. The user can add terms if required.

The type of measurement or count.

The types offered depend on the context (i.e. whether this is a measurement for a Location, Sample, Taxon occurrence or Biotope occurrence).

Qualifier

A value chosen from a Term list. The user can add terms if required.

Specifies what is being measured.

The qualifiers offered for selection depend on the type of measurement. See Linked lists.

Data

Free text.

The measurement or count that was made. This could be a numerical value or a code.

Unit

A value chosen from a Term list. The user can add terms if required.

The units depend on the type of measurement. See Linked lists.

For example, if Measurement was Altitude, 'metres' or 'feet' might be available as units, whereas if the Measurement was Area, the options might be 'acres' or 'hectares'.

Accuracy

Free text. Some common values are available from the list.

Indicates how accurately the measurement or count was made.

You don't have to enter anything in the measurements window, but once you start to enter a measurement, all fields except 'Accuracy' are required. Accuracy defaults to 'Unknown'.

To enter a measurement:

  • Ensure the detail panel is in Edit mode (The [Save] and [Cancel] buttons in the detail panel will be active).

  • To enter the first measurement, just fill in the five cells in the first row of the grid.

  • To enter a second and subsequent measurement, press the button to create a new, empty row in the grid and then fill in the cells.

To edit a measurement:

  • Ensure the detail panel is in Edit mode (The [Save] and [Cancel] buttons in the detail panel will be active).

  • Click on a cell in the grid using your mouse and enter a new value.

To delete a measurement:

  • Ensure the detail panel is in Edit mode (The [Save] and [Cancel] buttons in the detail panel will be active).

  • Select the row in the grid that you wish to delete by clicking on one of it's cells with your mouse, or by moving to a row using the <up arrow>, <down arrow>, <PgUp>, <PgDn>, <Home> and <End> keys.

  • Choose the button.

  • You will be prompted to confirm that you really want to delete the selected measurement.
