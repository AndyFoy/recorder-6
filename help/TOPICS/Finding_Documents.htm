Finding Documents

This dialog allows you to search for a document by looking for matches to typed text in the citation, or amongst the keywords.

When you are prompted for a document, it can be accessed by typing some text and pressing <Enter>. The text you typed will appear in the 'Search Text' box. You can add to or change the text in this box at any time to refine your search.

When you are in the Document window, you can also access it by:

  • Selecting Edit - Find from the Main menu.

  • Pressing the short-cut key <Ctrl-F>.

  • Clicking the button in the active toolbar.

To find a document:

  • Type some text likely to appear in the citation (author(s), date, title) of the document you want to find, or amongst its keywords. If there is no tick in 'Search for Reference Keywords', the list will show documents with citations starting with the string you type. If there is a tick in this box, the list will show documents with keywords starting with the string you type.

  • If you prefix the text you type with an asterisk, '*', then the list will show documents containing the text, or documents whose keywords contain the text.

  • Select an item in the list by clicking on it with your mouse, or press <Tab> to position the cursor in the list and then use the <up arrow>, <down arrow>, <PgUp>, <PgDn>, <Home> and <End> keys to position the highlight over the correct category.

    • Choose [OK] or press <Enter> to select the highlighted item.

    • Double-click on an item in the list to select it.

    • Choose [Cancel] or press <Esc> to close the Find dialog without making a selection.
