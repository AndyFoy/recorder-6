Related site types

The 5 options have the following meanings.

Within

The current location is entirely within the one with which you are establishing a relationship.

Contiguous

The current location has the same boundary as the one with which you are establishing a relationship. This is quite common when, for example, a Nature Reserve and an SSSI have the same boundary.

They should however be maintained as separate location entries because:

  • They are managed by different organisations

  • Their boundaries may not necessarily remain contiguous in future if, for example, they are revised at different dates.

Adjoining

The current location shares a boundary with the one with which you are establishing a relationship.

Overlapping

The current location overlaps with the one with which you are establishing a relationship.

Adjacent

The current location is near the one with which you are establishing a relationship, but does not share a boundary or overlap with it.
