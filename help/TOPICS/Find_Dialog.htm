The Find dialog

When you are at a prompt which requires an item stored elsewhere in the system, you can type some text, and press the <Enter> key to find items starting with that text. For example, at the location prompt, type 'br' <Enter>:

In the Find dialog:

  • The 'Search text' prompt contains the text you typed.

  • The 'Matches' list contains matching items – in this example, all locations with names starting with 'br'.

TIP: The search is NOT case sensitive so 'Br', 'BR', 'bR' or 'br' produce the same result.

The Find dialog will start to search the database once you have typed two or more letters. You will see items starting to appear in the 'Matches' list as soon as they are found, but it may take some time for all matching items to be located (e.g. if you are searching a large checklist for a commonly occurring item, it could take up to a minute or two). You can continue typing your search text, or select one of the items that has been found so far, while the search is in progress.

In these circumstances, the cursor will be shown as when it is over the list.

You can:

  • Continue to type in the 'Search text' box. For example, if you now type 'as' (so that Search text is 'bras'), the list will be refined, and will only contain one item, 'Brasside Pond'.

  • Select an item in the 'Matches' list either by clicking with the mouse or by pressing <Tab> to move the cursor to the list, and then using the <up arrow>, <down arrow>, <PgUp>, <PgDn>, <End> and <Home> keys to move within the list.

  • Return the currently selected item by clicking [OK] or pressing <Enter>.

  • Return any item in the list by double-clicking on it.

  • Close the dialog without returning anything by clicking [Cancel] or pressing <Esc>.

Wild card searches

If the text you type starts with an asterisk, '*', then a 'containing' search is done (instead of a 'starting with' search). Therefore, in this example, if you typed '*wood <Enter>', the list would include all locations with 'wood' somewhere in their name(s).

 

The Find Dialog is used to find:

  • locations

  • taxa

  • biotopes

  • names and addresses

  • documents

  • administrative areas

  • surveys

  • survey events

  • samples

  • taxon occurrences

  • biotope occurrences
