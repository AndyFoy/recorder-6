Edit Taxon details - Statuses

The Edit Taxon details - Statuses page holds status information for the taxon. This tab allows the user to add, edit or delete status information for the taxon.

At the top of the tab is a box which shows all of the statuses associated with the taxon, and allows the user to add, delete or edit statuses.

Type

Required.

Select the designation type from the drop-down box. The Taxon Designation Type is a ‘Term-list’, and items in this list can be added or deleted by clicking Tools - Term-lists.

From

Required.

Enter a valid date when the status applied to the taxon.

To

Optional. Enter a valid date when the status no longer applied to the taxon.

Geographical area

Optional. Enter a geographic constraint on the status. For example, the status may only apply to Northern Ireland.

Constraints

Optional. Any other constraints on the status. For example, the status applied at the genus level (genus Glis) may apply to all species except Glis glis.

Details

Optional. Any other details.

Reference

Optional. A valid internal reference for the status.

To enter a reference use one of the methods described in Find overview.

To add a status:

  • Click the [Add] button.

  • Enter the values for the status.

  • Click the [Save] sub-record button.

To delete a status:

  • Highlight the item you want to delete.

  • Click the [Delete] button.

  • You will be asked to confirm the deletion.

NOTE: Statuses that are system-supplied cannot be deleted.

To edit a status:

  • Select the item you want to edit.

  • Click the [Edit] button.

  • Make the changes to the data.

  • Click the [Save] sub-record button.

NOTE: Statuses that are system-supplied cannot be edited.
