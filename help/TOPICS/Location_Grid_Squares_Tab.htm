Location Grid Squares Tab

The Location Grid Squares tab lists the grid squares which the location spans. This tab allows the user to add or delete any number of grid squares. To add or delete grid squares you must be in Edit mode.

If the location has been associated with a polygon on one of the map layers, the icon can be used to 'Automatically extract grid squares from the associated location boundaries'. See below for extracting grid squares from polygons on a Recorder map.

Square

A list of the grid squares in which the site is located

Type

The size of the square which can be:

  • Hectare (i.e. 100m square)

  • 1 km

  • Tetrad (i.e. 2km square)

  • 10km

  • 100km

It is not necessary to enter any data, but if you make an entry in a row, values are required in both columns.

To add a grid square:

  • Either

  • Click on the [Add] button.

  • Enter a valid grid square value and a grid square type.

  • or

  • Click on the button.

  • Select the required grid square type from the drop down list that appears.

  • When you have added all the required grid squares and completed any changes on the other tabs, choose [Save] in the right-hand panel of the window to update the database with the changes for the selected location.

  • You can choose [Cancel] in the right-hand panel to abandon any changes. No changes will be made to the database, and the window will be restored to its original state.

To delete a grid square:

  • Highlight the item you want to delete.

  • Click on the [Delete] button.

  • You will be prompted to confirm that you really want to delete this item.

To edit a grid square:

  • Remove it and add the required square.

NOTE: If locations have grid square information, it is used to validate data entry for survey events and samples. Any grid references entered for events or samples should lie within the grid squares for the location. Similarly, grid square information is used in the Reporting Wizard to retrieve a list of locations within a bounding box (See Reporting overview).

To extract grid squares from polygons on a Recorder map:

  • Open the map containing the required polygon by selecting Map - View Map from the Main menu then, in the sub-menu, select the required map, or by using the Display Map Window icon on the toolbar. If the map window is already open, use the Active Map box (top left) to select the required map.

  • Select the pointer tool and the cursor will change to .

  • Click on the polygon you wish to extract grid squares for. It will change colour (to the Selected Colour) to indicate that it is selected.

    • Select Map – Extract Grid Squares from the Main menu, or

    • Click the button on the active toolbar, or
    • Right-click and select Extract Grid Squares from the resulting menu.

  • You will see the Extract Grid Squares dialog:

When the necessary details have been filled in, choose [OK] to extract the grid squares for the selected polygon and add them to the location it is associated with. [Cancel] abandons the process without making any changes to the database.

Note: Selecting All boundaries in layer will extract the grid squares for all locations that are associated with polygons on the selected polygon layer. This will be the only option available if a polygon hasn't been selected.
