Adding a map layer

Each map is made up of a base map and background and polygon layers that can be turned on and off. Map layers can be added from the Map Layers list in the Map Window or from the Map menu if the map window is already open, or from the Map Layers tab in Map - Map Options from the Main menu.

Note: It is important that you add the correct type of layer. Raster files, such as OS map tiles are background layers. If you attempt to add them as polygon layers you will get an error.

Either

  • Open the map to which you want to add the new layer by selecting Map - View Map from the Main menu then, in the sub-menu, select the required map, or by using the Display Map Window icon on the toolbar. If the map window is already open, use the Active Map box (top left) to select the required map.

  • Right-click in the list of map layers to the left-hand side

  • Select Add Background Layer or Add Polygon Layer from the resulting menu.

or

  • Open the required map as for the previous option

  • Select Map - Add Background Layer, or Map - Add Polygon Layer from the Main menu.

or

  • Select Map - Map Options from the Main menu.

  • Select the map to which you want to add the new layer using the drop down arrow beside the Map box on the Map Layers tab. Note that this box is only displayed if you have more than one map loaded.

  • Right-click on the list of map layers

  • Select Add Background Layer or Add Polygon Layer from the resulting menu.

Adding a background layer:

  • You will see the Map Registration Details dialog.

Adding a polygon layer:

  • You will see the Polygon Layer Display Details dialog box.

Fill in the details in the appropriate dialog box and press [OK]. The new layer will be created.

For deleting and changing map layers, or changing their order, see Map Options - Map layers
