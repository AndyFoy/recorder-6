Filter by Admin Area

You will see this screen if you selected the option to 'Restrict report to one or more admin areas' when selecting the style of report. This screen allows you to select particular administrative areas.

 

 

  • Choose the type of administrative area (e.g. Vice-county, English Civil Parish) from the list in the box at the top. The 'Available' list will then show the administrative areas of that type.

  • Move the names of administrative areas whose associated observations you want to see in the report into the 'Selected' list. See Add and Remove items from selected list.

  • You can add one or more names to the 'Selected' list, and then select a different type. This will result in a different 'Available' list from which you can make additional selections. This allows you to build up a list of areas of different types (e.g. Vice-county,1974-1996 County and Unitary).

TIP: If a name is moved into the 'Selected' list, all of its children in the hierarchy will automatically be added as well. For example, if you add a 1974-1996 English County to the selected list, its districts will also be added. If you don’t want to include the districts for some reason, remove them from the 'Selected' list in the normal way. See Remove items.

An observation is linked to an Administrative Area via a Survey event or Sample which refers to a Location which has this information listed (Locations - Geo Info - Administrative Areas). Therefore, a report which uses this feature will only include records with such a linkage.
