User Configuration

Accessed by Tools – User Configuration from the Main menu.

Only accessible to System Managers. Allows System Managers to create new users, to change the access level of an existing user, to remove users or to restrict a user's access to data in the database.

 

Username

A name from the list of individuals in the Names & Addresses window. If a new user's name is not already in the list, it must be added.

Access Level

See User Access Levels for the options available.

Survey

Lists all the surveys in the observation hierarchy. The user will have access to the ones that are ticked. By default, all surveys are ticked.

Note: This restricted access applies to reports they produce as well as the surveys they can see but not to exports done using export filters. XML reports and batch updates will only be restricted to specified surveys if they have been written to do so.

To add a username:

  • Click the [Add] button.

  • Use one of the methods described in Finding things to select the username from the list of individuals in the Names & Addresses window.

  • Use the drop down arrow beside the Access Level box to select an access level.

  • If this user's access is to be restricted to particular surveys untick all surveys to which they are not allowed access.

  • Click the [Save] button to save your changes.

  • Click the [Cancel] button to abandon your changes.

To edit a username:

  • Click the [Edit] button.

  • Make the required changes.

  • Click the [Save] button to save your changes.

  • Click the [Cancel] button to abandon your changes.

To delete a username:

  • Highlight the item you want to delete in the list on the left.

  • Click the [Delete] button.

  • You will be prompted to confirm that you really want to remove the username.
