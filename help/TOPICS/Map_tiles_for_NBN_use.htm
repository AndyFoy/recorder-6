OS 1:50,000 Map tiles for NBN use

The NBN has negotiated an agreement with Ordnance Survey to supply 1:50,000 map tiles for use by volunteer recorders at a much reduced licence fee.

A set of CDs has been produced containing the full set of 1:50,000 tiles in compressed and encrypted form, together with the necessary software to choose, order and install them. These are available from the NBN secretariat - contact maps@nbn.org.uk.

Registered Recorder 2002 users can licence just those tiles they wish to use. The process has been set up to allow licences to be obtained either by E-mail, or by post for those who don't have access to, or don't wish to use, electronic communications (but note that there is a small additional handling charge for postal orders):

  • The Map browser software is installed from the first CD. To install the software, you must agree to the terms of the licence agreement between NBN and OS.

  • On first running the Map browser, you are prompted to set up an account with the administrator at the NBN secretariat. This involves filling in you contact details (name, e-mail address, postal address, etc.) in an on-screen form which is posted or e-mailed to the administrator (with your first order for tiles).

  • You can then choose the tiles you wish to licence. The Map browser generates an order by e-mail, or it can be printed and posted to the administrator.

  • You pay the licence fees either by ringing up and paying via a credit or debit card, or by posting a cheque with the printed order form.

  • Once payment has been received, the administrator generates "unlock keys" which are specific to your copy of Recorder 2002 and the particular tiles you requested. These keys are e-mailed or posted back to you, depending on how your order was received.

  • When you receive the unlock keys, you run the Map browser software again and it reads the e-mail, or prompts you to type in the unlock keys from the letter.

  • The software will then prompt you to insert the relevant CD and will install the chosen tiles into your copy of Recorder.

If you order tiles by e-mail and pay by credit or debit card over the phone, it should be possible to gain access to a given tile almost immediately during normal office hours.
