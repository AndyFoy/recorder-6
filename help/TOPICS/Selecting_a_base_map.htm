Selecting a base map

When you first use the mapping facilities in Recorder (i.e. the first time you use the View Map or Map Options commands), or after you have used the Reset Map command, you will be prompted to choose at least one base map to use for your maps.

Note: There is the option to have more than one base map and run them in parallel. This is most likely to be of use for UK wide organisations that need to map using both the Irish and GB grids.

Three choices of base map are offered:

  • An outline of Great Britain

  • An outline of Ireland

  • An outline of the UK

The choice of base map will depend on the maps you wish to use with.

  • If you intend to use OS-GB map tiles and data based on the British National Grid (for example, digitised site boundaries provided by British conservation bodies), then choose the GB outline.

  • If you intend to use OS-NI map tiles and data based on the Irish Grid (for example, digitised site boundaries for Northern Ireland), then choose the Irish outline.

  • If you wish to map mixed data for both Britain and Ireland, then choose either the UK outline or have two maps- one for GB and one for the whole of Ireland.

Spatial References

By default- Recorder is set up to display data using OSGB spatial references. If you wish to change to another system you must change this through the spatial references tab of the options screen
