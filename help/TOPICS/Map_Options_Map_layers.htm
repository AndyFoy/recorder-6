Map Options - Map layers

Maps in Recorder can have three different types of layers:

  • The outline base map

  • Background layers

  • Polygon layers . This icon is shown in the unselected colour and the outline is infilled with the polygon layer's pattern, so the colour and pattern will vary for different polygon layers.

These can be managed in three ways:

  • Select Map - Map Options from the main menu, then select the Map Layers tab. Select the map you want to work on using the drop down arrow beside the Map box. Note that this box is only displayed if you have more than one map loaded.

  • Use the Map Layers box in the bottom left-hand corner of the open Map window. To open the map you want to work on select Map - View Map from the Main menu then, in the sub-menu, select the required map, or use the Display Map Window icon on the toolbar. If the map window is already open, use the Active Map box (top left) to select the required map.

  • Open the required map as for the previous option, select Map from the Main menu, then the required option from the menu that appears.

These options for managing maps can be used to:

  • Add, delete or rename map layers on a given map.

  • Change the properties of map layers, e.g. change the cut in and cut out scales for background layers, or the colours and pattern for polygon layers.

  • Change the order in which the map layers are displayed by moving them up and down.

  • Turn the map layers on and off by ticking and unticking the layers.

  • Export polygon layers

Note: Not all functions are available on all the ways of managing map layers, e.g. the export functions aren't available from the Map Layers tab in Map - Map Options; the options to change the properties of a map layer or to delete it aren't available from the Map menu.
