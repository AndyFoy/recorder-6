The Map Window

The Map window can show:

  • background maps that have been loaded using the Map Options - Map Layers page;

  • distribution symbols for taxa, biotopes, samples and survey events;

  • polygons and lines representing administrative areas, site boundaries or transects.

In the Map window you can:

  • Zoom in and out to view maps at an appropriate scale and pan around the map to view particular parts of it.

  • Use the pointer tool to obtain spatial references and location names, or to select polygons by clicking on the map.

  • Draw a bounding box to define the area covered by a survey or the area to be used to retrieve data for a report.

  • Draw or import polygons and lines representing site boundaries or transects.

  • Delete polygons.

  • Add and remove distribution symbols, control their scale and find the observations which generated a given distribution point.

  • Associate polygons with locations in the location hierarchy or administrative areas.

  • Extract the grid squares for a polygon

  • Overlay a grid at various scales.

  • Print the current map.

  • Save map to a file (bitmap or JPEG format) or to the clipboard (so that it can be pasted in another program).

  • Add samples by positioning an icon on the map.

Active Map

Shows the map that is currently selected and being used. The active map can be changed using the drop-down list.

Distribution Points

Shows the details of up to 10 sets of distribution map symbols. From here you can control the colour of the symbols, turn the display of symbols on and off and remove sets of distribution points. See Map Options - Distribution Symbols.

Map Layers

Lists the base map for the map on view, and the background and polygon layers that have been added. From here you can add new layers, remove existing ones, turn the display of layers on and off and change the properties for particular layers, e.g. change the cut in and cut out scales for background layers or the colours and pattern for polygon layers. You can also export polygon layers. See Map Options - Map Layers.

Current Polygon Layer

Selects the polygon layer that will be used to store a polygon which is drawn or imported.

Spatial ref

Shows the spatial reference of the point at which you last clicked with the pointer tool. It is formatted according to the currently selected spatial reference system (see Options - Spatial References).

This spatial reference can be dragged or copied to the clipboard from here. It is the spatial reference which is returned by the Return Data facility.

Location

Shows the name of the location associated with the line or polygon on which you last clicked with the pointer tool. If you did not click on a line or polygon, 'None' will appear here. If you clicked on a line or polygon that is not associated with a location, 'Location not named' will appear here.

The location can be dragged or copied to the clipboard from here.

The scale for the current map display is shown in the status bar.
