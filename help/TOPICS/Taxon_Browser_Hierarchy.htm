Taxon Browser Hierarchy

An example of the hierarchy view is shown below.

In this example, the current checklist is ‘Ulster Museum & Marine Conservation Society Marine Directory’.

The tree view shows the recognised hierarchy for this checklist as a series of nodes, with icons representing the taxonomic level. Each node displays the taxon, and at the species level, the Latin name for the species and an Authority (if one is known) are displayed.

The hierarchy is sorted according to the sort order in the database when the browser is opened. To display the items in a different order:

  • Right click on the hierarchy, and select the preferred sort order.

  • Click on the [Sort Order] button on the tool bar, and select the preferred sort order.

The sort order only applies to the currently selected level in the hierarchy up to the parent. Only items at the same level in the tree view for the same parent will be sorted according to the preferred sort order. To sort items elsewhere in the hierarchy, select the item that needs sorting, and repeat the above procedure. To return the sorting to the order of the database, select 'default' as the preferred sort order.

TIP: To show all the names nested below the currently selected one, press the asterisk key on the numeric key pad.

TIP: Selecting an item in the hierarchy will move the selection in the list view to the same item.
