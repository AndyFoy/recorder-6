Report Wizard Print Preview

This window allows you to view a report on screen before it is printed. It has a toolbar which can be used to look through the report, and also to save it in various formats.

The active toolbar appears as follows:

 

Zoom to fit. Display the preview so that a whole page fits onto the screen.

Zoom to 100%. Show the page at the actual size it will be when printed.

Zoom to width. Show the preview so that the width of the page fits across the width of the screen.

Zoom. Every time you click this button, the size of the previewed image will be increased.

Un-Zoom. Every time you click this button, the size of the previewed image will decrease.

First page. Preview the first page of the report.

Previous page. Preview the previous page of the report.

Next page. Preview the next page of the report.

Last page. Preview the last page of the report.

Export. Save the report in one of a variety of formats.

Print the report.

This help screen.

TIP: All these commands are available from the Report menu while the print preview screen is active.

The following file formats are available for saving reports to disk:

Text (txt)

A plain text file. Does not maintain any formatting, but compatible with just about any software, on just about any computer system.

Comma separated text (CSV)

Fields are separated by commas. Text fields are enclosed in quotes. Suitable for importing into most spreadsheet or database packages. Can be passed to other computer systems (e.g. Macintosh, Unix).

Rich Text Format (rtf)

Supported by most word-processors. Maintains the formatting of the text (such as font, italics, etc).

HTML (htm)

Suitable for display in a web page. Maintains the formatting of text.

Windows Meta-File (wmf)

A 'picture' of the file. Suitable for drawing packages.

Excel spreadsheet (xls)

Microsoft Excel format spreadsheet format. Supported by most recent Windows spreadsheet and database software and many GIS packages. Can also be imported into Word documents as a table.
