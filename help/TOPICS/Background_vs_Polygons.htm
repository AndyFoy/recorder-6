Background maps and site-polygons

  • Background map layers provide a picture over which information, such as site-boundaries or distribution maps can be displayed.

    • They are loaded by adding them to the Map Layers list in the Map window - see Adding a map layer.

    • They can be turned on and off.
    • They can be deleted or renamed and their properties can be changed.
    • They can consist of vector maps or raster tiles.
    • Ordnance Survey 1:50,000 map tiles for Great Britain are available from the NBN for use as background layers in Recorder.

  • Polygons (closed shapes) and vectors (lines) indicate site boundaries or transects (e.g. Butterfly Monitoring Scheme transects) which can be linked to locations or administrative areas.

    • They are drawn using the polygon tool and stored on a polygon layer, or imported onto a polygon layer. To add a polygon layer - see Adding a map layer.

    • Polygon layers can be turned on and off.
    • Polygon layers can be deleted or renamed and their properties can be changed.
    • Polygons are always vector maps.
    • They can be used to select observations on which to report (i.e. those falling within a polygon on the map).
    • The Watsonian vice-county boundaries for Great Britain are available from the NBN for use as polygons in Recorder.
