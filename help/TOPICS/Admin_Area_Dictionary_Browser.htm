Administrative Area Dictionary Browser

This is accessed by:

  • Selecting Dictionaries - Admin Areas from the Main menu.

  • Clicking the button in the Main toolbar.

Clicking the[Add] button beside administrative areas boxes will also open this browser.

To select an administrative area:

  • Select the type of administrative area you are looking for from the list at the top of the window.

  • Select the desired item from the list by clicking on it with your mouse, or by positioning the highlight using <up arrow>, <down arrow>, <PgUp>, <PgDn>, <Home> and <End> keys.

  • You can search for a name in the selected list by clicking the button in the active toolbar, or pressing the short-cut key <Ctrl-F>. This will display the Find Admin area dialog, allowing you to search for terms starting with, or containing, the text you type.

Tip: To show all the names nested below the currently selected one, press the asterisk key on the numeric key pad.
