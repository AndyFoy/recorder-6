Changing the settings for a map layer

The settings or properties of a map layer depend on whether it is a background or polygon layer. They include such things as its name, the cut in and cut out scales for background layers, and the pattern and colours for polygon layers. Changing these settings or properties can be done as follows:

  • Select the map layer you wish to change the setttings for

Either:

    • Open the Map that the layer is attached to by selecting Map - View Map - <base map name> from the Main menu.

    • Click on the icon for the layer to be changed ( for background layers, for polygon layers or for the base map) in the Map Layers box to the left.

or

    • Open the Map that the layer is attached to by selecting Map - View Map - <base map name> from the Main menu.

    • Right-click on the name of the layer to be changed in the Map Layers box to the left.

    • Select Properties from the menu.

or

    • Open the map options screen by selecting Map - Map Options from the Main menu.

    • Select the Map Layers tab.

    • Choose the the Map that the layer is attached to from the drop-down list provided.

    • Click on the icon for the layer to be changed, or click on the name and click the [Properties] button.

  • For a background layer or a base map layer, the Map Registration Details dialog will open. For a polygon layer, the Polygon Layer Display Details dialog will open.

  • Edit the details you wish to change.

    • Choose [OK] to make the changes to the map layer.

    • Choose [Cancel] to close the dialog without making any changes to the map layer.

NOTE: If you change the south-west and/or north-east spatial references for a raster tile, you are in effect changing the spatial reference that will be obtained by clicking on the map. If you have already used this method to obtain spatial references for locations or observations that are stored in the database, you need to go back to these records and correct them. This cannot be done automatically!

Renaming a layer:

You can rename a layer without opening the Properties dialogs.

  • Select a map layer in the list.

  • Either double-click on it, or right-click on it and select Rename from the menu.

  • Edit the name.

  • Click away from the layer to save the change.

  • Press <Esc> to abandon the change (the name will return to what it was before you started editing).
