Showing a grid overlay

The grid system in use can be shown by:

  • Selecting Map – Grid lines from the Main menu, and then selecting the desired frequency from the sub-menu.

  • Clicking the button on the active toolbar, and selecting the desired frequency from the menu which appears below the button.

The options are:

  • 100km

  • 50km
  • 10km
  • 1km
  • None

If the selected frequency would result in lines being drawn too close together, or if you un-zoom with a grid overlay turned on, the program will automatically revert to a coarser grid.

NOTE: There is no facility to control the colour or appearance of the grid. It is plotted as dashed blue lines.
