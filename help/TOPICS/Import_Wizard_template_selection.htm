Import Wizard - template selection

The template selection page of the import wizard allows the user to:

  • Select a template that was saved as part of a previous import.

  • Opt to import only a selection of rows from the import file.

  • Select any formatting options that may be required.

Note: the formatting options are not shown by default as they are unlikely to be required very often. You must click on the [More] button to display them, then click on the [Less] button to hide them again.

For help with this stage of an import, see the text on the left of the window.

Templates

If you import data in the same format on a regular basis, you can save some of the selections you make as you go through the wizard to a template. This can be used in subsequent imports of similar files to speed up the process. Things like the column matching will be stored in the template so that it won't need to be done again when using a template.

Templates can be created and saved at any point in the data entry process, up to, but not in, the analysis phase. Use the [Save Template] button in the bottom left-hand corner of the screen.

To use a saved template, select the required one using the drop-down list at the top of the page.

Formatting Options

The default options are likely to be appropriate for most imports. However, you can change the:

  • Date format

The import process will interpret variations such as 2 digit years correctly. This facility will probably only need to be used if the import file contains American style dates where the month comes before the day.

  • Date delimiter

  • Decimal symbol

  • Thousand separator

When you have made your selections, click on the [Next] button to move to the next screen.
