Abbreviated species names

Recorder automatically creates a 5-letter abbreviation for each scientific name in the Taxon dictionary. This consists of the first two letters of the generic name and the first three letters of the specific name.

Examples:

Name

Abbreviation

Eristalis tenax

erten

Lychnis flos-cuculi

lyflo

Phylloscopus sibilatrix

phsib

Tip: Abbreviations are formed for every name in the Taxon dictionary which has at least two words in it separated by a space (not a hyphen), whether or not it is a scientific name. So "White wagtail" has an abbreviation "whwag". Where there are more than two words the abbreviation is formed from the first two words, e.g. "euwat" is the abbreviation for Eurasian Water Shrew in the MAMMALIA list.

Using abbreviations:

Abbreviated names are intended to provide a rapid way of finding a taxon with the minimum of typing if you already know its name. You will usually find, especially in smaller checklists, that only one or a very few names match a given abbreviation.

When you are at a prompt for a taxon name you can simply type the abbreviation and press <Enter>. If only one taxon has this abbreviation, its full name will appear in the prompt. If more than one match is found, the Find Taxon dialog will open, and will list all species with that abbreviation. Experience has shown that this is usually the quickest and most efficient way to find a taxon by name.

You can, of course, use 5-letter abbreviations in the 'Search text' prompt at any time when you are using the Find taxa dialog.

Note: for this to work in the Find Taxon dialogue box, Search by must be set to 'Abbreviation'.

Note: on some checklists the standard 5-letter abbreviation for birds has been replaced by the BTO 2 letter code, e.g. RW for Reed Warbler.

Partial name search for taxa

If 'Enable partial name search for taxa' is ticked on the General tab in Tools - Options, as well as searching using 5-letter abbreviations, you can search for taxa using the following parts of the name, separated by a space:

  • a variable number of letters from the start of the first word or genus.

  • a variable number of letters from the start of the second word or species, or, a variable number of letters from the end of the second word preceded by '.' - optional.

  • a variable number of letters from the start of the third word or species, or, a variable number of letters from the end of the third word preceded by '.' - optional.

Note: for this to work in the Find Taxon dialogue box, Search by must be set to 'Name'.

Examples:

"camp rotun" finds Campanula rotundifolia, Harebell, in the VASCULAR PLANTS list

"c betu" finds Carpinus betulus, Hornbeam, in the VASCULAR PLANTS list

"erin .eus" finds Erinaceus europaeus, Western European Hedgehog, in the MAMMALIA list

"eur wa shr", "eur wa .w", "eur wat" and "eur shr" find Eurasian Water Shrew in the MAMMALIA list

 

Troubleshooting:

If you cannot find a taxon for the text you type, check that:

  • The correct Search by option is selected in the Find Taxon dialog.

The Find Taxon dialog can search either the names or abbreviations of taxa for the text you type. This is controlled by a pair of buttons in that dialog. Make sure that the correct one is selected. See Find Taxon for more details.

  • The correct checklist is selected.

If you have chosen to restrict searches to the Current Checklist, only the currently selected checklist is searched. Open the Taxon dictionary browser to check which checklist is currently selected and to change your selection, if necessary. Alternatively, change to an Unrestricted search or to a Preferred Checklists search using the the drop down arrow beside Restrict Searches to, or by changing 'Restrict Taxon Searches to' on the General tab in Tools - Options.
