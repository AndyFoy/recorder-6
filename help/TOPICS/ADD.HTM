Add

To add a new record in the window that is currently active:

  • Choose [Add].

  • Select Edit – Add from the Main menu.

Depending on where you are, you may see a menu asking what type of data you wish to add. For example, in the Locations window, you could add a new site, sub-site or feature.

  • A '<new item>' indicator will appear in the left hand hierarchy, and the detail panel on the right will contain an empty form for you to enter data.

  • Required fields will have a coloured background if you have selected this option. See Options – Appearance.

  • When you have finished entering data, choose [Save] to save the new record in the database. This performs any outstanding validation checks and, if the data are valid, makes permanent changes to the database.

  • Choose [Cancel] to abort this process at any time. No changes are made to the database, and the window is returned to its original state before you started to add the new record.

  • If you attempt to close the window, or the application, before completing the addition, you will be prompted to indicate whether you want to finish the addition first, or abandon the process and close the window or the application.
