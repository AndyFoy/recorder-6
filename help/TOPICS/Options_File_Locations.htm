Options File Locations

This page of the Options window allows you to specify the folders in which the following types of files created by Recorder will be stored. Select Tools - Options from the main menu to access it.

  • Rucksack files.

  • Report templates.

  • Snapshot files.

  • Queries this is where a query produced by the Report Wizard will be stored.

  • Recording cards.

  • Images for Taxon and Biotope facts.

  • Polygon filters.

The initial settings depend on the folder name chosen for Recorder during installation, but will be as follows for a 'standard installation'.

Rucksack files

\Recorder 6\User Files\Rucksacks\

Report templates

\Recorder 6\User Files\Templates\

Snapshot files

\Recorder 6\User Files\Snapshots\

Queries

\Recorder 6\User Files\Reports\

Recording cards

\Recorder 6\User Files\Recording Cards\

Image files

\Recorder 6\User Files\User Dictionary Images\

Polygon filters

\Recorder 6\User Files\Polygon Filter\

These settings are not affected by the [Default settings] button.

Network installation

The Rucksack, Report, Query and Recording card folders will usually be located on the server in a networked installation. If this is the case, then any user on the network has access to any of these files, irrespective of the workstation from which they were saved.

To change a setting:

  • Typing or editing the path shown in the box.

  • Choosing the button beside each of the four paths to open a standard Windows folder selection dialog, which allows a folder to be chosen.

The folder path you enter must exist and be accessible.

NOTE: Each path MUST end with '\'. This will be added when you click [OK] should you fail to do so.
