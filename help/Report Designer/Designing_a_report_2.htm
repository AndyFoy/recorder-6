Report designer

The Report Designer in Recorder 2002 is a "banded report designer" meaning that reports are built up of sections called "bands" which contain your information.

You can design a report from scratch by adding bands to the report layout screen and then positioning data fields on the bands (see step by step instructions below).

You can open a previously saved report layout (File - Open from the Main menu) and modify it. If you do this it is important that you have selected all the attributes (in the Report Wizard - Select Attributes screen) that the report layout needs. You can modify and existing report by adding more data fields, but a layout won't work as expected if you try and use it on a set of results that does not supply all the data fields it requires.
