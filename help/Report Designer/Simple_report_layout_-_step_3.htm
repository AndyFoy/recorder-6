Simple report layout - step 3

Setting up the Page header band:

  • Add your logo to the page header. Click on the "add new image" button and then click in the Page header band. The following dialogue box will appear which allows you to find and format the picture:

  • The ellipses button opens a standard Windows file dialog to help you locate the file. Once you have the file you want, click [OK].

  • You will probably need to change the height of the band so that the image fits. Click in the page header band so that grab handles appear and then drag the one at the middle of the bottom edge until the band is the correct width.
