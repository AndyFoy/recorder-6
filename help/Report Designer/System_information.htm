System information

Click the "Add new system field" button and then in the band where you want the information to appear.

The Systemfield dialog opens allowing you to select the type of information you want. The options are:

  • Time prints the time at which the report was printed

  • Date prints the date on which the report was printed

  • Date/Time prints the date and time at which the report was printed

  • Detail count prints the total number of rows of data that will be printed in the Detail bands. Use this to show the number of observations included in the report.

  • Detail number prints a running number which is incremented each time a new detail band is printed.

  • Page number prints the current page number - useful in Page headers or footers.

  • Report title prints the title specified in the Report Option dialog - again useful in Page headers and footers.
