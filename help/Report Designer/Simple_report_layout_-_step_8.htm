Simple report layout - step 8

Saving and printing

The completed layout looks like this:

You can:

  • Save it for future use by selecting File - Save as... from the menu. You will be prompted to give the report template template a name. By default templates are stored in the \Recorder 6\User Files\Templates directory and have a ".tpl" extension.

TIP: Recorder uses long file names, and this is the name you will use to recognise this report in future. It is worth typing a suitably descriptive name so that you, and others, will recognise the report in future.

NOTE: You MUST document the attributes and sort order selections which should be made to re-use this report in future. If you do not select the correct fields, then the report may not work at all, or missing fields will appear blank. If you have used group-bands, then the order in which observations are sorted is critical for the group bands to work correctly. If the report is for your own use, you probably will not remember these details in a few months time. If it is for others to use, you must let them know which selections to make.

  • Print the report by selecting File - Print... from the menu.

  • Preview the printed appearance by selecting File - Print Preview... from the menu. Once you are in the Print Preview screen you can see what the report would look like, send it to the printer or save it as a file in a variety of other formats.
