Simple report layout - step 6

Setting up headers for the columns

Now that you know where the columns of data are going to be, you can add titles for them by positioning labels in the Column header band.

  • The process is the same as for adding the title in step 4. Remember to change the font of each label, as you add it, if you would like them to be bold and/or in a larger type face.

  • Use the same methods to align the titles nicely within the band as with the data fields in the previous step.

Unfortunately, you cannot select multiple elements in different bands, so you cannot use the alignment tools to line up the headers and the data fields.

  • You may want to add a line at the bottom of this band to separate the column headers from your data.

    • Select the Column header band by clicking within it, but not within any of the labels you have positioned on it. (So that band has grab handles, but not any of the elements it contains).

    • Click the Frame tool and tick the "Bottom" option. Adjust the thickness and colour of the line if you wish, then click [OK].
