Simple report layout - step 1

  • The first thing to do is set the overall options for the report. Right click somewhere on the layout screen and the Report Options dialog box will appear:

Items you might want to change:

  • Paper size, Layout, Margins and columns. These are self explanatory.

  • Font. This is the default font for the whole document. By default it is Arial / Regular/ 10 pt / black. Setting something different here doesn't stop you changing the font of individual elements, but it is more convenient to set your preferred font here than having to set it for each element individually as you lay them out.

  • Frame. Use this if you would like a line around the whole report. It will appear in the same place as the dotted blue line on the design screen. You use it like the Frame tool.

Once you are happy with your selections click [OK] You can now begin to design the inside of your report.
