Simple report layout - step 2

The Report Designer in Recorder creates reports made up of sections or "bands". Each type of band has specific characteristics that control when it appears in your final report and how many times. Within each band you can position the data that you want along with images and dates etc.

Adding the bands

We want our report to contain the following:

  • A title and the author's name printed on the first page only.

  • A header on every page except the first containing a logo.

  • Some items of data from the observations selected in the Report Wizard. These will be laid out in columns with column headings at the top of each page.

  • A summary section on the last page showing how many observations the report includes and a copyright/usage notice.

  • A footer on each page showing the date and page number.

To add the bands:

  • Click on the "add new band" button and then click inside the blue dotted rectangle. A band will be added at the top of the layout area entitled "BAND 1" and the following dialog box will appear:

  • Select the correct band type and format your band using the elements in the dialog box. For example, when you select the "Page header" band, tick the "Not on first page" option if you don't want a header on the title page. Once you are happy with your selection, click [OK].

  • When the band is added to the design, you will see it on the layout like this:

    • The bands are named "BAND 1", "BAND 2", etc in the order in which you add them

    • The band type is shown in small letters at the bottom left-hand corner of the band
    • The band has "grab handles". You can use these to change the height of the band.

Add the following bands to your report:

Title

for the title and author's name

Page header

for the logo. Remember to choose the "Not on first page" option.

Column header

to provide headings for the data columns on each page

Detail

to contain columns of data from your observations

Summary

for the total number of observations

Page footer

for the date and page number

You are now ready to put information on the bands.
