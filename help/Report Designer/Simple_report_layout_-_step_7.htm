Simple report layout - step 7

The Summary and Page footers.

We wanted the Summary band, which prints just once at the end of the report, to show how many observations the report includes and to contain a paragraph of text giving information about the copyright and use of the report.

Displaying the number of observations:

The number of times the detailed band is printed in the report is one of the items available via a system field, so:

  • Click "add new system field" button and then click in the Summary band. You will see the following dialog:

    • Select the Type of system information you want (in this case "Detail count")

    • Type the Text you want to appear with this information. Don't forget to leave a space on the end of your text so that the system supplied data (the number) is separated from your text.

    • You may also want to set the font, perhaps to show this comment in italics. When you are happy, click [OK].

Copyright and usage statement:

If you want to add a paragraph of text, add a memo or RTF memo field. The difference is that all the text in a memo has the same font whilst an RTF memo can contain text in a variety of fonts (e.g. if you wanted to have some words italicised within your paragraph, you would choose an RTF memo).

  • Click the "add new memo" button and then click in the Summary band. The Memo dialog will open:

  • Type your text and set any other options you want (e.g. set the font) and then click [OK].

Setting up the Page footer

  • The page footer will contain the report date and the page number. These are both available via system fields so add two system fields of Type "Date" and "Page number" as above.

  • You might want this band to have a line at its top edge to separate it from the rest of the page. Use the Frame tool as in step 6.
