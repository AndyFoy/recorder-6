Graphics buttons

These two buttons allow graphics to be placed on the report. Click the button and then click on the band in the report where you want the element to appear. Graphics can also be placed on the background of the report (click on the design surface outside of the bands).

  • Add a new image allows an image from a file to be placed in the report. You will be prompted for a file name. Images files in Windows bitmap (BMP), Windows meta-file (WMF, EMF) or icon (ICO) format can be used.

  • Add a new shape allows a shape (Rectangle, Circle various sorts of lines) to be drawn. You can control colour, thickness, drawing style, etc.
