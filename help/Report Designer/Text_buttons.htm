Text buttons

Click one of these three buttons to add fixed text to your report and then click the band where you would like the text to appear. A dialog will open allowing you to type your text, set its font etc.

  • Add new label adds a simple text label - useful for title, column headers, etc.

  • Add new memo allows you to type longer paragraphs of text, but all the text will appear in the same font.

  • Add new rich-text add a Rich Text Format memo. This also allows you to type paragraphs of text, but the font can be changed within this text
