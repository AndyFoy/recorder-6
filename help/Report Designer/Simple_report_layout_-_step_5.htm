Simple report layout - step 5

Adding the data fields to the Detail band:

  • Drag the items which you want to appear on the report from the Data Fields list to the Detail band.

  • You can get them to line up nicely using the alignment tools.

    • First position one of the fields you added to the top margin of the band by clicking in it and then dragging it into position with your mouse.

    • Select all the other fields by holding down the [Shift] key and clicking each in turn.

    • Click the "align top edges button" and all the fields will line up along the top of the band.

    • Finally you will probably want to reduce the height of this band so that the fields just fit. Remember that the bulk of the resulting, printed report will consist of repetitions of the Detail band, one for each row of data.
