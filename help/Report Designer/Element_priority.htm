Element priority

It is possible to place elements on the report so that they overlap. In such cases the order in which they are displayed matters, the element "on top" may hide the one "underneath".

  • Select an element and then click the "send element to the back" button so that it is "behind" all other elements.

  • Select an element and click the "bring element to the front" button so that it is "in front of" all other elements.
